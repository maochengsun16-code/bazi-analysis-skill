# 实战断事速查

> **用途**：快速查询各实战专题的判断规则，配合第10步使用。

---

## 一、婚姻感情速查

### 夫妻星定位
| 方法 | 规则 |
|------|------|
| **夫妻宫** | 日支固定为夫妻宫 |
| **财官候选** | 财星和官杀星都可作配偶星；男命财星优先，女命官杀优先 |
| **成立条件** | 候选夫妻星必须与日柱发生作用：天干合冲日干，或地支合、六合、半合、冲、害、会、半会、破、生夫妻宫 |
| **无作用** | 财官不与日柱发生作用时，不定夫妻星，只按桃花、短缘或露水情缘看 |
| **星宫同位** | 日支本气为财/官杀可直接定；日支藏干财/官杀可参考，优先级较低 |
| **降级规则** | 财官都不成立时，再看与日柱发生作用的其他十神，尤其六合到夫妻宫者 |

### 婚姻稳定性速查

| 星宫关系 | 婚姻结果 | 置信度 |
|---------|---------|--------|
| 合进来（天合地合） | ✅ 最稳定，一婚到头 | 高 |
| 天干五合（地支无冲克） | ✅ 比较稳定 | 高 |
| 半合/半会 | 可成关系，但稳定性不足 | 中 |
| 冲 | ❌ 二婚命 | 高 |
| 害（穿） | ❌ 也会离婚 | 高 |
| 刑进夫妻宫 | ❌ 痛苦、不稳定 | 高 |
| 破到夫妻宫 | ❌ 关系破损，易闹离婚 | 高 |
| 生入夫妻宫 | ❌ 不稳定 | 中 |
| 绝入夫妻宫 | ❌ 一生婚姻极不稳定 | 高 |

### 出轨判断速查

| 判断条件 | 结论 | 置信度 |
|---------|------|--------|
| 夫妻星与比劫同宫同柱 | 另一半出轨（必断） | 高 |
| 夫妻星被比劫合到 | 出轨 | 高 |
| 比劫合到夫妻宫 | 百分百出轨 | 高 |
| 暗合出现在夫妻宫 | 不安分 | 高 |
| 一宫合两星 | 同时拥有多段关系 | 高 |

### 离婚时间速查

| 情况 | 判断 |
|------|------|
| 男 | 劫财运/年，劫财坏夫妻宫之年 |
| 女 | 伤官同时坏夫妻宫+夫妻星之年 |
| 天合地穿 | 一婚不到头（表面好实际恶劣） |

## 二、财富层次速查

### 财富层级

| 层次 | 条件 | 示例 |
|------|------|------|
| **过亿级** | 年上财官被用神制到家里 | 年柱天地一气为用神 |
| **千万级** | 月上财官被制到家里 | 月令做功 |
| **百万级** | 食伤生财/日时财被制到 | 一般级别 |
| **较差** | 无制化或制化无力 | 无财 |

### 做功方式排序（从高到低）

| 优先级 | 方式 | 说明 |
|--------|------|------|
| 1 | **库做功** | 大格局，家里比劫库制年月财库=发财 |
| 2 | **印化**（印来生） | 别人给钱（父母、贵人），最轻松 |
| 3 | **食伤制官杀** | 自己干，最累 |
| 4 | 其他 | 一般 |

### 特殊发财机制

| 机制 | 说明 |
|------|------|
| **真神得用** | 原局当前专题用神大运再见 → 对应专题高峰 |
| **化忌为用** | 忌神生用神且用神得力 → 大富之命 |
| **穿的形式取财** | 劫财穿财 → 偏门/黑灰产 |
| **用神在年上天地一气** | 比劫除外 → 亿级 |

> 不一定走喜用神大运才能发财——用制财官的那个字（即使它是忌神）的大运，同样能发财。

## 三、子女判断速查

### 子女星定位

| 顺序 | 女命 | 男命 |
|------|------|------|
| 标准 | 伤官→儿，食神→女 | 七杀→儿，正官→女 |
| 无食伤时 | 七杀→女，正官→儿 | 伤官→女，食神→儿 |

### 克子女结构

| 情况 | 判断 |
|------|------|
| 女命印旺克食伤（尤其偏印在月令） | 克子女（严重可能无子） |
| 食伤被克泄太过 | 克子女 |
| 男命食伤旺克官杀 | 子女不如自己 |

## 四、父母判断速查

### 父母星定位

| 关系 | 男命 | 女命 |
|------|------|------|
| 母亲 | 正印（无则偏印） | 偏印（无则正印） |
| 父亲 | 偏财（无则正财） | 正财（无则偏财） |

### 六亲健康判断

> ⚠️ **核心原则**：不看喜忌，只看状态——该十神在原局是否被刑冲克害泄

| 状态 | 健康结论 |
|------|---------|
| 没有被刑冲克害 | 身体好 |
| 被穿（六害） | 身体不好（穿最严重） |
| 被冲 | 突然发病 |
| 被刑 | 长期慢性病 |
| 入墓 | 功能减弱 |
| 遇绝 | 极度危险 |

## 五、灾祸牢狱速查

### 四种牢狱结构

| 类型 | 特征 | 风险 |
|------|------|------|
| **官杀旺无制化** | 官杀当令旺且为忌 | 走官杀运就进去 |
| **伤官见官** | 官为用，伤官克官 | 单纯倒霉 |
| **伤官治官/比劫扛官** | 拿好处后被反制 | 曾经辉煌后败落 |
| **三刑** | 丑未戌三刑为官杀 | 大运流年凑齐触发 |

> **核心警句**：富贵靠官杀，倒霉也靠官杀。

## 六、职业取向速查

| 类型 | 判断依据 |
|------|---------|
| **打工命** | 印旺 + 用神无力 + 食神被克 |
| **老板命** | 食神制杀（食伤透月上制七杀）+ 外食伤生家里财 |
| **体制内掌权** | 杀印相生（外七杀生家里印）+ 财生杀 = 实权 |
| **体制内上班** | 财生杀→杀生印的链条 |
| **自由职业** | 食伤旺 + 印不旺 |

> **先看食伤**（想法）——有没有想法决定会不会出去做事。

## 七、大运流年速查

### 大运流年分专题核心口诀

```
忌神生用神 → 当前专题应吉
用神生忌神 → 当前专题应凶
用神交战 → 当前专题应凶
忌神交战 → 当前专题应吉
```

先锁定专题，再用口诀。财运只看财星、财库、财路、做功链、经营投资、现金流、债务和回款；婚姻、健康、父母等不利信息另列，不得否定财运结论。

### 大运作用于命局的三条路径

| 路径 | 表现 |
|------|------|
| **生助忌神** | 大运生旺当前专题忌神 → 该专题不利 |
| **合出忌神** | 大运与原局合化出当前专题忌神 → 该专题不利 |
| **用神生忌神** | 大运用神却生了当前专题忌神 → 该专题好事打折 |

### 流年判断规律

| 场景 | 规律 |
|------|------|
| 当前专题好大运 + 当前专题差流年 | 该专题减量或波动，不直接翻车 |
| 当前专题坏大运 + 当前专题好流年 | 该专题相对好一点 |
| 当前专题坏大运 + 当前专题差流年 | 该专题容易爆雷 |
| 流年作用顺序 | 先与大运作用 → 再作用于原局 |

### 墓库运特殊看法

| 情况 | 判断 |
|------|------|
| 走辰戌丑未运 | 看天干透出的字决定该库属性 |
| 透出实 | 吉凶明显 |
| 虚透 | 力量有限 |

## 八、健康/疾病速查

### 地支穿（六害）对应疾病

| 穿组合 | 疾病 |
|--------|------|
| 寅巳穿 | 头疼、四肢筋骨 |
| 子未穿（女） | 子宫肌瘤/卵巢囊肿、近视 |
| 卯辰穿 | 肝胆、皮肤 |
| 酉戌穿 | 牙齿、心脏 |
| 申亥穿 | 暗疾 |
| 丑午穿 | 近视、牙齿 |

### 五行受伤对应

| 五行 | 对应器官 |
|------|---------|
| 金被克 | 肺、呼吸系统、骨骼 |
| 木被克 | 肝、胆、四肢 |
| 水被克 | 肾、泌尿、耳 |
| 火被克 | 心脏、眼睛、神志 |
| 土被克 | 脾胃、消化系统 |

### 抑郁症速查

| 条件 | 判断 |
|------|------|
| 食伤受刑（三刑或自刑） | 极易抑郁 |
| 食伤自刑（原文提及） | 抑郁 |
| 偏印过旺克制食神 | 情绪低落 |
