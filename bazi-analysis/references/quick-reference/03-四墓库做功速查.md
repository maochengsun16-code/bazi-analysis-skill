## 三、开库方式（满足任一即开）

| 方式 | 说明 | 示例 |
|------|------|------|
| **天干透出** | 库中藏干在天干出现 | 辰库→天干透癸水 |
| **地支出现** | 库中五行同类在地支出现 | 辰库→地支见亥子 |
| **冲库** | 辰戌冲、丑未冲（两库全开） | 辰戌同时出现 |
| **刑库** | 丑戌刑、未戌刑、三刑（两库全开） | 丑戌同时出现 |
| **拱开** | 相邻地支拱合 | 申辰拱子、寅戌拱午 |

### 开库程度判断

| 条件 | 开库程度 | 说明 |
|------|---------|------|
| 天干透出 + 地支有根 | 彻底打开 | 能量完全释放 |
| 天干透出 + 地支无根 | 打开但弱 | 力量有限 |
| 冲库/刑库 | 强烈打开 | 力量突然爆发 |
| 拱开 | 部分打开 | 力量缓慢释放 |

## 四、闭库方式（六合闭库）

| 六合 | 闭合的库 | 说明 |
|------|---------|------|
| 辰酉合 | 闭辰库（水库） | 辰中乙木也被酉中辛金破坏 |
| 卯戌合 | 闭戌库（火库） | 卯中乙木被戌中辛金破坏 |
| 子丑合 | 闭丑库（金库） | 子水闭丑中金库 |
| 午未合 | 闭未库（木库） | 午火闭未中乙木库 |

### 半开半闭

```
天干透出 + 地支有六合 → 门开了条缝，依然按开库论
因为天干透出的力量 > 六合闭库的力量
```

## 五、库做功：发财/破财判断

### 空间定位

| 位置 | 含义 |
|------|------|
| 年月（年柱+月柱） | 家外/社会——别人的、公共的 |
| 日时（日柱+时柱） | 家里——自己的、私有的 |

### 发财原理

```
家里的比劫库制年月的财库 → 发财 ✅
```
- 例：日时有未（比劫库），年月有丑（财库）
- 丑未冲 → 家里的比劫库冲开外面的财库 → 发财
- 层次：年上→过亿级；月上→千万级

### 破财原理

```
年月的比劫库制家里的财库 → 破财 ❌
```
- 例：年月有丑（比劫库），日时有未（财库）
- 丑未冲 → 外面的比劫库冲开家里的财库 → 破财

### 财富层次

| 做功位置 | 财富层次 | 示例 |
|---------|---------|------|
| 年上库做功 | 过亿级 | 年上的财官被制到家里 |
| 月上库做功 | 千万级 | 月上的财官被制到家里 |
| 日时库做功 | 百万级 | 日时的财被制到 |

## 六、四墓库月取用

### 原则
**两层做功**——土本身一层做功 + 土中透出之物另一层做功

### 判断流程
```
1. 看土本身（比劫/食伤属性）
    ├── 土旺 → 喜官杀制/喜财耗/喜食伤泄
    └── 综合全局

2. 看透出之物
    ├── 天干透出什么？
    ├── 该透出物的十神是什么？
    └── 透出物是否为用？
         ├── 是 → 此物为该八字第一用神
         └── 否 → 土本身做功优先
```
