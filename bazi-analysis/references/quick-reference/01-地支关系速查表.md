# 地支关系速查表

> **用途**：快速查询地支之间的关系，配合第7步分析使用。
> **原则**：谁在月令谁力量大，强者才有主动权。

---

## 一、六合（合作/暗合/闭库）

| 组合 | 性质 | 效果 | 特殊说明 |
|------|------|------|---------|
| 子丑合 | 闭库之合 | 闭丑金库 | 合化为土 |
| 寅亥合 | 破合 | 表面合作内里算计 | 合中带破 |
| 卯戌合 | 闭库之合 | 闭戌火库 | 卯中乙木被戌中辛金坏 |
| 辰酉合 | 闭库之合 | 闭辰水库 | 辰中乙木被酉金坏 |
| 巳申合 | 破合 | 表面合作内里算计 | 合中带破 |
| 午未合 | 闭库之合 | 闭未木库 | 合化为火 |

### 六合重点应用
- **闭库**：辰酉→闭水库；卯戌→闭火库；子丑→闭金库；午未→闭木库
- **暗合**（私下行为）：寅丑暗合、午亥暗合 → 出轨、偷情、暗中求财
- **破合**：寅亥合、巳申合 → 合作中隐藏算计

## 五、三刑（痛苦/纠缠）

| 类型 | 条件 | 应事 |
|------|------|------|
| **丑未戌三刑**（最凶） | 原局自有才算 | 官杀三刑→牢狱；财星三刑→克父克妻；食伤三刑→抑郁 |
| **寅巳申三刑** | 原局自有才算 | 夏/春→申金受伤；秋→寅木受伤 |
| **自刑** | 亥亥/辰辰/酉酉/午午 | 自我纠结、内耗、食伤自刑→抑郁 |

> 大运流年补齐三刑不算真三刑（原局要有两个才能被运年补齐激活）

## 六、破（破坏）

| 组合 | 说明 |
|------|------|
| 子酉破 | 原局子水当令才破酉金 |
| 寅亥破 | 与六合共存（寅亥合中带破） |
| 卯午破 | 原局卯木当令才破午火 |
| 辰丑破 | 不看季节，但原局需相邻 |
| 巳申破 | 与六合共存（巳申合中带破） |
| 未戌破 | 不看季节，但原局需相邻 |

- 运年破原局：不再按原局月令和位置筛选，重点看运年之字破到哪个十神或宫位。
- 破到财星：主破财、损耗。
- 破到夫妻宫：主关系破损、争吵；星宫同破或运年引动时离婚风险高。

## 八、关系权重使用边界

地支关系不能用一个全局“最强/最弱”排序套所有问题。先看专题：

- 财运：看制财官、库做功、破到财星。
- 婚姻：看夫妻星与夫妻宫的合、冲、穿、刑、破、绝。
- 六亲健康：看六亲星是否被穿、刑、绝、破、冲等伤到，不先看喜忌。
- 合作：破、破合、冲、穿常比普通合更重要。

在夫妻、六亲、合作等关系伤害语境下，绝通常比破重；但这是破绝内部比较，不是所有专题的全局优先级。

## 九、核心口诀（所有关系通用）

```
用神制忌神 → 应吉（多富贵）
忌神制用神 → 应凶（易负债、招灾）
用神与用神交战 → 应凶
忌神与忌神交战 → 应吉（日主得利）
用神化忌神 → 应吉
忌神化用神 → 应凶
```
