# 分析校验清单

> **用途**：分析完成后逐项检查，确保没有遗漏和错误。
> **执行时机**：完成10步分析和分项断事后，输出报告前。

---

## ✅ 完整性检查

### 基本信息
- [ ] 四柱已完整确认（年/月/日/时）
- [ ] 日主五行和阴阳已正确
- [ ] 性别已确认
- [ ] 大运信息已确认
- [ ] 程序预处理字段已确认（十神矩阵/月令十神/根气/寒燥/七杀/四墓库）

### 10步流程完整性
- [ ] 第0步：程序预处理完成
  - [ ] 四柱、大运、流年已生成
  - [ ] 天干十神已列出
  - [ ] 地支藏干十神已列出
  - [ ] 月令本气十神已标出
  - [ ] 根气摘要已生成
  - [ ] 寒燥标记已生成
  - [ ] 七杀标记已生成
  - [ ] 四墓库标记已生成
- [ ] 第1步：定日主、明五行完成
- [ ] 第2步：定月令十神、判环境完成
- [ ] 第3步：特殊优先级扫描完成
  - [ ] 四墓库做功是否优先已判断
  - [ ] 强七杀是否优先已判断
  - [ ] 寒燥调候是否优先已判断
- [ ] 第4步：定格局完成
  - [ ] 特殊组合已检查（官杀混杂/枭神夺食/伤官见官等）
- [ ] 第5步：判旺衰看根气完成
  - [ ] 坐支根气已检查
  - [ ] 他支根气已检查
  - [ ] 印星生扶已检查
- [ ] 第6步：取用神定喜忌完成（核心）
  - [ ] 最旺五行已确定
  - [ ] 月令十神已纳入取用
  - [ ] 特殊优先级是否覆盖普通月令格局已说明
  - [ ] 用神已确定
  - [ ] 忌神已确定
  - [ ] 有杀先论杀已处理
  - [ ] 真神/假神已区分
  - [ ] 调候需求已考虑（冬火/夏水是否成立）
- [ ] 第7步：地支关系分析完成
  - [ ] 六合已检查
  - [ ] 六冲已检查
  - [ ] 三刑已检查
  - [ ] 六害（穿）已检查
  - [ ] 破/绝已检查
- [ ] 第8步：四墓库分析完成（如有）
  - [ ] 开库/闭库已判断
  - [ ] 做功方向已判断
  - [ ] 财富影响已评估
- [ ] 第9步：神煞已辅助参考
  - [ ] 重要神煞已识别
  - [ ] 未喧宾夺主
- [ ] 第10步：分项断事完成
  - [ ] 婚姻（如有需要）
  - [ ] 财运（如有需要）
  - [ ] 子女/父母（如有需要）
  - [ ] 事业/职业（如有需要）
  - [ ] 大运/流年（如有需要）

## ✅ 准确性检查

### 取用神准确性
- [ ] 是否先完成程序预处理，而不是边排边断？
- [ ] 是否先扫描库做功、强七杀、寒燥调候？
- [ ] 若有库做功，是否先于普通月令格局处理？
- [ ] 若月令在亥子丑，是否检查火调候？
- [ ] 若月令在巳午未，是否检查水调候？
- [ ] 用神是否制化了全局最旺五行？（取用神核心原则）
- [ ] "有杀先论杀"是否已遵守？
  - [ ] 原局七杀（包括藏干中的）是否已全部检查？
- [ ] 是否区分了"真神"和"假神"？
- [ ] 用神/忌神虚透是否已判断？
- [ ] 四墓库月的两层做功是否已考虑？

### 婚姻判断准确性
- [ ] 夫妻星是否按最新优先级定位？（男命财星优先、女命官杀优先，不成立再看另一类）
- [ ] 候选夫妻星是否与日柱发生作用？（天干合冲日干，或地支作用日支夫妻宫；无作用只按桃花短缘）
- [ ] 夫妻宫（日支）是否已正确识别？
- [ ] 星宫关系是否完整考虑？（合/六合/半合/冲/害/会/半会/破/生/刑/绝）
- [ ] 出轨判断中的比劫因素是否已检查？

### 六亲判断准确性
- [ ] 六亲健康是否用了"不看喜忌看状态"原则？
  - ⚠️ 这是最容易错的地方！不要看喜忌！
- [ ] 六亲星被穿/冲/刑克的情况是否已检查？

### 地支关系判断准确性
- [ ] 力量对比是否以月令为基准？
- [ ] 六害(穿)的伤害程度是否被低估？（穿比冲更严重）
- [ ] 三刑是否原局自有？（运年凑齐不算真）

### 大运判断准确性
- [ ] 大运分专题吉凶是否先看与月令的作用关系？
- [ ] 大运是否按整十年看地支（不分前五年后五年）？
- [ ] 流年是否先与大运作用，再作用于原局？
- [ ] 好大运中的差年份是否按"减量"而非"翻车"处理？
- [ ] 坏大运中的好年份是否按"相对好"而非"大好"处理？
- [ ] 大运流年是否已按专题分别判断，而不是输出单一"此运好/坏"？
- [ ] 若财运好但婚姻、健康、父母等专题不利，是否已并列说明，而不是用其他专题否定财运？

### 核心口诀一致性
- [ ] 所有结论是否与核心口诀一致？
  - 用神制忌神 = 应吉
  - 忌神制用神 = 应凶
  - 用神与用神交战 = 应凶
  - 忌神与忌神交战 = 应吉
  - 用神化忌神 = 应吉
  - 忌神化用神 = 应凶

## ✅ 一致性检查

### 结论自洽性
- [ ] 不同分项之间结论是否自洽？
  - （如：财运判断为"千万级"但职业判断为"打工命"→ 需要检查是否矛盾）
- [ ] 是否避免了为追求"自洽"而强行合并不同专题？
  - （同一大运可以财运为吉、婚姻动荡、父母健康不利；必须分专题并列，不得混成一个总吉凶）
- [ ] 是否存在未解释的矛盾指标？
  - （如有若干个指标指向相反方向但未说明为什么选这个方向）
- [ ] 喜忌判断在所有分项中是否保持一致？
  - （不能婚姻分析用一套喜忌，财运分析用另一套）

### 专题隔离检查
- [ ] 用户只问财运时，是否只分析事业、财富、工作、经营、投资、现金流、债务、回款？
- [ ] 财运段落是否没有夹带婚姻、健康、父母、子女、学业结论？
- [ ] 婚姻段落是否只围绕夫妻宫、夫妻星、星宫关系、比劫竞争、伤官见官和外缘？
- [ ] 健康/灾祸段落是否只围绕受伤五行、刑冲穿害、官杀灾祸结构和身体取象？
- [ ] 父母/子女段落是否只围绕六亲星、六亲宫和六亲星状态？

### 高置信度结论验证
- [ ] 每个标注为"高"的结论是否有铁律支持？
- [ ] 高置信度结论是否做了反向验证？
  - （如取用神高 → 反向验证走用神大运是否真的好）

## ✅ 置信度检查

- [ ] 每个必须标注置信度的结论都已标注
  - [ ] 根气判断
  - [ ] 用神（喜）
  - [ ] 忌神
  - [ ] 婚姻稳定性
  - [ ] 财富层次
  - [ ] 大运分专题吉凶
  - [ ] 流年吉凶（如有给出）
- [ ] 标注为"高"的结论没有过度自信的表述
- [ ] 标注为"低"的结论明确了不确定因素
- [ ] 没有结论未标注置信度
- [ ] 置信度标注与推理过程一致

---

## ✅ 表述规范检查

- [ ] 无"一定""绝对""100%"等过于绝对的表述
- [ ] 具体年份/数字判断标注了低置信度（或不给）
- [ ] 建议语气与置信度匹配
  - 高 → 明确建议
  - 中 → 温和建议
  - 低 → 仅供参考
- [ ] 每个结论都有依据（非凭空判断）
- [ ] 使用了标准术语（非自创概念）

---

## ⚠️ 常见错误自查

```
❌ 常见错误         ✅ 正确做法
──────────────────────────────────────────
身强身弱二分法      → 看根气
得令＝身强           → 得令是环境，不是日主状态
神煞定吉凶           → 五行喜忌优先
夫妻星固定男才女官  → 性别优先级 + 日柱作用
六亲健康看喜忌      → 不看喜忌看状态
大运前五年后五年    → 整十年看地支
穿小于冲/刑         → 穿比冲刑更直接伤人
合化必成             → 合不等于化（需条件）
```

## 执行流程

```
完成10步分析
    │
    ▼
逐项检查完整性清单 → 如有遗漏，补充分析
    │
    ▼
逐项检查准确性清单 → 如有错误，修正分析
    │
    ▼
检查一致性 → 如有矛盾，按冲突消解规则处理
    │
    ▼
检查置信度 → 调整不合理的置信度标注
    │
    ▼
检查表述规范 → 修正过于绝对或模糊的表述
    │
    ▼
校验通过
    │
    ▼
输出报告
```
