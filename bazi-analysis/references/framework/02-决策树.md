# 八字分析决策树

> **用途**：当分析过程中出现需要"选择"或"判断"的时刻，按此决策树执行。
> **原则**：沿着分支一步步走，不跳跃，不预判。

---

## 决策树0：总控流程

> **触发条件**：排盘和程序预处理完成后，进入任何命理判断之前。
> **前提**：已取得 `analysis_prep` 或等价字段：十神矩阵、月令十神、根气、寒燥、七杀、四墓库。

```
输入：四柱 + 十神矩阵 + 月令十神 + 根气摘要 + 寒燥标记 + 七杀标记 + 四墓库标记

0. 程序固定层先完成：
    ├── 四柱、大运、流年
    ├── 四天干十神、四地支藏干十神
    ├── 月令本气十神
    ├── 根气摘要（明现比劫/明现印/对应库/无根）
    ├── 寒燥标记（亥子丑寒、巳午未燥）
    ├── 七杀位置和强弱标记
    └── 四墓库位置标记

1. 先扫特殊优先级：
    ├── 有四墓库且形成做功？
    │    ├── 是 → 先走“决策树4：四墓库做功判断”
    │    └── 否 → 进入下一项
    ├── 七杀强（在月令/透干有力/多处出现）？
    │    ├── 是 → 先按“有杀先论杀”处理
    │    └── 否 → 进入下一项
    ├── 寒燥偏枯？
    │    ├── 亥子丑月寒重 → 火为第一调候用神
    │    ├── 巳午未月燥重 → 水为第一调候用神
    │    └── 非寒燥极端 → 进入普通月令格局
    └── 无特殊覆盖 → 按月令十神定格局

2. 定格局：
    ├── 月令为财 → 财格/财旺
    ├── 月令为官杀 → 官杀格/杀印/食伤制杀
    ├── 月令为印 → 印格/印旺用财
    ├── 月令为食伤 → 食伤格/伤官配印
    ├── 月令为比劫 → 禄刃/建禄/月劫
    └── 月令为辰戌丑未 → 四墓库月，两层做功

3. 定用忌：
    ├── 特殊优先级已成立 → 以特殊分支为主，月令为辅
    └── 特殊优先级未成立 → 以月令十神制化为主，根气为承载能力

4. 原局定完后：
    ├── 先分析大运（以地支为重，先作用月令/原局）
    └── 再分析流年（流年先作用大运，再作用原局）
```

## 决策树1：取用神

> **触发条件**：第6步确定用神时
> **前提**：已完成程序预处理、日主、月令十神、特殊优先级扫描、格局和根气判断

```
输入：月令类型 + 八字符 + 根气情况

1. 先承接总控流程的特殊分支：
    ├── 有库做功成立 → 用神先看做功所需的开库/制库/护库之字
    ├── 寒重成立 → 火为第一调候用神，再看火是否有根、有护卫
    ├── 燥重成立 → 水为第一调候用神，再看水是否有根、有护卫
    └── 特殊分支处理后，仍要回到月令取用检查是否同向；无特殊分支则直接进入第2步

2. 检查：是否有强七杀（藏干也算，但以月令/透干/多见为重）？
    ├── 有强七杀 → 先处理七杀（有杀先论杀）
    │    ├── 有印 → 印化杀（最高层次）
    │    ├── 无印有食伤 → 食伤制杀（中等）
    │    └── 无印无食伤有比劫 → 比劫扛杀（最低）
    │
    └── 无七杀 → 按月令分类取用

3. 按月令十神分类取用神：
    ├── 月令为财星（财月）
    │    ├── 用印星生身（首选）
    │    ├── 用比劫制财（次选）
    │    └── ⚠️ 官杀不可用（财生官杀更耗身）
    │
    ├── 月令为官杀（官杀月）
    │    ├── 有印 → 印化官杀（最高）
    │    ├── 无印有食伤 → 食伤制官杀（中等）
    │    ├── 无印无食伤有比劫 → 比劫扛（最低）
    │    └── 皆无 → 贫苦命
    │
    ├── 月令为印星（印月）
    │    ├── 有财 → 财制印（首选）
    │    ├── 无财有比劫 → 比劫化印（次选）
    │    └── 注意季节：夏天未戌火不可用，辰丑水可用；冬天反之
    │
    ├── 月令为食伤（食伤月）
    │    ├── 有印 → 印制食伤（首选）
    │    ├── 无印有官杀 → 官杀生印制食伤（次选）
    │    └── ❌ 不可用：财星（更泄身）、比劫（再生食伤）
    │
    ├── 月令为比劫（禄刃月）
    │    ├── 有官杀 → 官杀制比劫（首选）
    │    ├── 有财 → 财耗比劫（次选）
    │    ├── 有食伤 → 食伤泄比劫（第三）
    │    └── ⚠️ 注意季节：冬天癸水子月，辰丑不可用
    │
    └── 月令为土（四墓库月：辰戌丑未月份）
         ├── 两层做功优先处理
         ├── 土本身一层做功
         └── 土中透出之物另一层做功

4. 处理虚透：
    ├── 用神虚透（天干有地支无根）→ 力量有限，需大运补根
    └── 忌神虚透（天干有地支无根）→ 危害有限，不足惧

5. 确认真神/假神：
    ├── 原局已有的字取为用神 → 真神（大运再见→对应专题高峰）
    └── 原局没有的字取为用神 → 假神（大运来了也作用有限）

6. 合并同向用神：
    ├── 特殊分支与月令取用同向 → 合并为同一喜用神证据链，提升该用神优先级
    ├── 例如：巳午未燥重取水，同时禄刃月比劫旺也取官杀水制比劫 → 水为最优喜用神
    ├── 其他能制化旺神的字仍为喜用神，只是作用层级不同
    └── 例如：同为禄刃月，财星能耗比劫并生官杀 → 财星也是喜用神，但弱于直接调候制比劫的官杀

7. 输出：
    ├── 用神（喜）：X、X — 置信度：[高/中/低]
    ├── 喜用神内部排序：X最优，Y次之 — 依据：[一字多用/直接制化/间接生扶]
    ├── 忌神：X、X — 置信度：[高/中/低]
    └── 依据：[引用具体规则和推理过程]
```

## 决策树2：根气强弱判断

> **触发条件**：第5步判断日主能否担财官用食伤时
> **前提**：已完成日主确认（第2步）

```
输入：日主五行 + 四柱地支

1. 检查日主的地支（坐支）
    ├── 坐支明现比劫 → 有根气（强）
    ├── 坐支明现印 → 有根气（强）
    ├── 坐支为日主对应库 → 有根气（库根）
    ├── 只有中气/余气 → 不算根气，进入第2步
    └── 坐支无根 → 进入第2步

2. 检查其他地支（年支、月支、时支）
    ├── 有其他支明现比劫 → 有根气
    ├── 有其他支明现印 → 有根气
    ├── 有对应库根 → 有根气（库根）
    ├── 只有中气/余气 → 不算根气
    └── 完全无根 → 进入第3步

3. 戊己土特殊校验
    ├── 日主为戊己土，见辰戌丑未 → 比劫根/土库
    ├── 日主为戊己土，见巳午 → 印根，也可兼看比劫助力
    └── 仍无明现比劫、印、库 → 进入第4步

4. 综合判断
    ├── 有印+有根+得令 → 极旺
    ├── 有根+得令（或得生）→ 偏旺
    ├── 有根（明现比劫/明现印/对应库）→ 有做事能力 ✅
    ├── 只有中气/余气 → 按无根或根气不足处理
    ├── 无根+有生扶 → 弱但可救，需大运补根
    ├── 无根+无生扶+失令 → 从格倾向
    └── 无根+无生扶+不得令 → 极弱

5. 输出结论
    ├── [有根气 → 可担财官用食伤] — 置信度：[高/中/低]
    ├── [只有中气/余气 → 不算有根气] — 不可直接断能担
    └── 根气不足 → 需大运补根或从格论
```

## 决策树3：婚姻稳定性判断

> **触发条件**：第10.1步分析婚姻时
> **前提**：已完成第1-6步（已确定夫妻宫和用神喜忌）

```
输入：日柱（日干+日支夫妻宫）+ 四柱财官 + 地支关系

1. 确定夫妻星
    ├── 性别只定优先级，不定唯一性
    │    ├── 男命：先看财星；财星不成立，再看官杀
    │    └── 女命：先看官杀；官杀不成立，再看财星
    ├── 成立条件：候选夫妻星必须与日柱发生作用
    │    ├── 天干与日干：五合可直接定位，相冲也可作作用线索
    │    ├── 地支与日支夫妻宫：合、六合、半合、冲、害、会、半会、破、生
    │    └── 无作用的财/官杀 → 不定位夫妻星，只按桃花、短缘、露水情缘看
    ├── 星宫同位
    │    ├── 日支本气为财/官杀 → 直接定位夫妻星
    │    └── 日支藏干有财/官杀 → 可参考，优先级低于本气明现和明确作用关系
    ├── 财官都不成立
    │    └── 再看与日柱发生作用的其他十神，尤其六合到夫妻宫者
    └── ⚠️ 如果天干合但地支绝 → 会结婚也会离婚

2. 判断星宫关系 → 决定婚姻稳定性
    ├── 合进来（天合地合）→ ✅ 最稳定，一婚到头
    ├── 天干五合（地支没有冲克）→ ✅ 比较稳定
    ├── 半合/半会 → 可成关系，但稳定性不足
    ├── 夫妻星冲夫妻宫 → ❌ 二婚命
    ├── 穿（害）→ ❌ 也会离婚
    ├── 刑进夫妻宫 → ❌ 痛苦、不稳定
    ├── 破到夫妻宫 → ❌ 关系破损，易闹离婚
    ├── 生入夫妻宫 → ❌ 不稳定
    └── 绝入夫妻宫 → ❌ 一生婚姻极不稳定（多段）

3. 判断出轨可能
    ├── 夫妻星与比劫同宫同柱 → ⚠️ 另一半会出轨（必断）
    ├── 夫妻星被比劫合到 → ⚠️ 出轨
    ├── 比劫合到夫妻宫 → ⚠️ 百分百出轨
    ├── 夫妻宫有暗合 → ⚠️ 不安分/偷情
    └── 一宫合两星（夫妻宫同时合两个异性）→ ⚠️ 同时拥有多段关系

4. 判断离婚时间
    ├── 男：劫财运/年 → 劫财坏夫妻宫之年
    │    ├── 劫财合夫妻宫 → 必离
    │    └── 劫财冲夫妻宫 → 大概率离
    └── 女：伤官年 → 伤官同时坏夫妻宫+夫妻星
         ├── 仅伤官出现 → 婚姻危机（不一定离）
         └── 伤官同时冲/穿夫妻宫+夫妻星 → 离婚年

5. 特殊情况标记
    ├── 正偏财混杂（男）→ 多妻星，桃花多
    ├── 官杀混杂（女）→ 多夫星，感情复杂
    ├── 天合地穿 → 表面好但实际恶劣，一婚不到头
    └── 桃花+空亡同时出现 → 桃花多但无结果

6. 输出
    ├── 婚姻稳定性：[稳定/一般/二婚/多段] — 置信度：X
    ├── 出轨可能：[有/无] — 置信度：X
    └── 关键应期年份：[XX年注意事项] — 置信度：X
```

## 决策树4：四墓库做功判断

> **触发条件**：第8步分析四墓库时
> **前提**：八字中有辰戌丑未

```
输入：四墓库地支位置 + 天干透出 + 地支关系

1. 识别四墓库及属性
    ├── 辰（本气戊土）：水库+木库
    │    └── 十神看日主：甲→印库+比劫库；丙→官杀库+印库
    ├── 戌（本气戊土）：火库+金库
    │    └── 十神看日主：甲→官杀库+食伤库；丙→比劫库+财库
    ├── 丑（本气己土）：金库+水库
    │    └── 十神看日主：甲→官杀库+印库；丙→财库+官杀库
    └── 未（本气己土）：木库+火库
         └── 十神看日主：甲→比劫库+食伤库；丙→印库+比劫库

2. 判断开库/闭库
    ├── 开库（满足任一）：
    │    ├── 天干透出库中藏干
    │    ├── 地支出现库中五行同类
    │    ├── 冲库（辰戌冲/丑未冲）→ 两库全开
    │    ├── 刑库（丑戌刑/未戌刑/三刑）→ 两库全开
    │    └── 拱开（相邻地支拱合）
    └── 闭库：
         ├── 辰酉合 → 闭辰库（水库）
         ├── 卯戌合 → 闭戌库（火库）
         ├── 子丑合 → 闭丑库（金库）
         └── 午未合 → 闭未库（木库）

3. 判断做功方向 → 发财/破财
    ├── 空间定位：年月=家外/社会；日时=家里
    ├── 家里的（日时）比劫库制年月的财库 → ✅ 发财
    └── 年月的比劫库制家里的（日时）财库 → ❌ 破财

4. 四墓库月取用
    ├── 土本身做功（一层）
    └── 土中透出之物做功（二层）
         ├── 透出比劫 → 土旺
         ├── 透出财星 → 土中藏财
         └── 透出官杀 → 土中藏官

5. 财富层级别断（如果有库做功）
    ├── 年上库做功 → 过亿级
    ├── 月上库做功 → 千万级
    └── 日时库做功 → 百万级

6. 输出
    ├── 库：[开/闭]
    ├── 做功方式：[库做功/无]
    ├── 发财/破财：[结论]
    └── 置信度：[高/中/低]
```

## 决策树5：财富层次判断

> **触发条件**：第10.2步分析财运时
> **前提**：已取用神、看地支关系、看四墓库

```
输入：用神情况 + 做功方式 + 宫位关系

1. 检查做功方式（从高到低）
    ├── 有库做功 → 最高优先级
    │    ├── 家里的比劫库制年月的财库 → 发财（大格局）
    │    ├── 年上的库被制到家里 → 过亿级
    │    └── 月上的库被制到家里 → 千万级
    │
    ├── 无库做功 → 检查印化
    │    ├── 印星生身生财（印化）→ 别人给钱
    │    └── 印化级别低于库做功
    │
    └── 无库无印化 → 检查食伤制杀
         ├── 食伤制官杀 → 自己干（最累）
         └── 级别低于印化

2. 检查"制"的方位 → 定层次
    ├── 用神在年上天地一气（比劫除外）→ 亿级
    │    └── 年上的财官被制到家里 → 过亿
    ├── 月上的财官被制到家里 → 千万级
    ├── 日时的财被制到 → 百万级
    └── 无制化或制化无力 → 一般/无财

3. 检查特殊发财机制
    ├── 真神得用（原局用神大运再见）→ 对应专题高峰
    ├── 化忌为用（忌神生用神且用神得力）→ 大富之命
    ├── 穿的形式取财（劫财穿财）→ 黑灰产/偏门
    └── 忌神生用神之运 → 表面是忌神但结果好

4. 输出
    ├── 财富层次：[过亿/千万/百万/一般/较差] — 置信度：X
    ├── 做功方式：[库做功/印化/食伤制杀/其他]
    └── 发财大运：[XX运]
```

## 决策树6：地支关系综合判断

> **触发条件**：第7步分析地支关系时
> **前提**：已完成第1-6步

```
输入：四柱地支 + 月令 + 用神喜忌

1. 列出所有地支关系
    ├── 六合：[列表]
    ├── 三合：[列表]
    ├── 六冲：[列表]
    ├── 二刑：[列表]
    ├── 六害：[列表]
    ├── 破：[列表]
    └── 绝：[列表]

2. 确定力量对比（看月令）
    ├── 谁在月令谁力量大
    ├── 月令的地支在冲中占主动
    ├── 月令的地支在穿中占主动
    └── 月令的地支在刑中占主动

3. 先按专题目标确定关系权重，不机械套统一排序
    ├── 看财运 → 先看库做功、制财官、破到财星是否破财
    ├── 看婚姻 → 先看夫妻星与夫妻宫，重点查冲/穿/刑/破/绝/合
    ├── 看六亲健康 → 先看六亲星受伤状态，穿/刑/绝/破/冲都要回到状态
    ├── 看合作关系 → 重点看破、破合、冲、穿是否破坏合作
    └── 看整体结构 → 再综合月令力量、用神忌神和宫位

4. 应用核心口诀
    ├── 用神制忌神 → 应吉
    ├── 忌神制用神 → 应凶
    ├── 用神与用神交战 → 应凶
    ├── 忌神与忌神交战 → 应吉
    ├── 用神化忌神 → 应吉
    └── 忌神化用神 → 应凶

5. 特殊情况处理
    ├── 多组关系叠加 → 先分析与当前专题直接相关的关系，再考虑其他调整
    ├── 既有合又有冲/破/穿/刑/绝 → 先看伤害关系是否破坏合，再看合能否缓解
    ├── 破绝比较 → 只在夫妻、六亲、合作等关系伤害语境中参考，不能当全局优先级
    ├── 运年破原局 → 不再按原局月令和位置筛选，只看破到哪个十神或宫位
    └── 一组关系同时涉及多个十神 → 按每个十神分别分析

6. 输出
    ├── 主要关系：[XX] → 影响：[吉/凶] — 置信度：X
    ├── 次要关系：[XX] → 影响：[吉/凶] — 置信度：X
    └── 健康提示：[XX方面注意]
```

## 决策树7：大运流年吉凶判断

> **触发条件**：第10.6步分析大运流年时
> **前提**：已完成第1-6步（已有用神喜忌）

```
输入：大运干支 + 流年干支 + 月令 + 原局结构 + 用户指定专题

0. 先锁定专题
    ├── 用户只问财运/事业 → 只输出事业财运结论
    ├── 用户只问婚姻 → 只输出婚姻感情结论
    ├── 用户只问健康/灾祸 → 只输出健康灾祸结论
    ├── 用户只问父母/子女 → 只输出父母子女结论
    └── 用户问整体 → 分专题并列输出，不合并成单一吉凶

1. 大运与月令作用（当前专题的核心第一步）
    ├── 大运地支与月令地支的关系
    │    ├── 合 → 合作/改变
    │    ├── 冲 → 改变/冲突
    │    ├── 刑 → 痛苦/变动
    │    └── 穿 → 伤害/有事
    ├── 大运天干与月令天干的关系
    └── 结论：大运对此专题的定调

2. 大运与当前专题用神/忌神的关系
    ├── 大运为当前专题用神 → 该专题吉
    │    └── 真神得用（原局有此用神）→ 该专题高峰
    ├── 大运为当前专题忌神 → 该专题凶
    │    └── 忌神虚透（天干有地支无根）→ 危害有限
    ├── 忌神生用神 → 应吉（化忌为用）
    └── 用神生忌神 → 应凶（好事变坏事）

3. 流年作用顺序
    ├── 流年先与大运发生作用
    │    ├── 流年与大运合 → 改变大运方向
    │    ├── 流年与大运冲 → 打破大运稳定
    │    └── 流年生扶大运 → 加强大运能量
    └── 流年再作用于原局
         ├── 流年与原局冲刑合害 → 应事
         └── 流年引动原局忌神/用神 → 应吉/应凶

4. 吉凶程度判断
    ├── 当前专题好大运 + 当前专题好流年 → 该专题明显顺
    ├── 当前专题好大运 + 当前专题差流年 → 该专题减量/小波折（不直接翻车）
    ├── 当前专题坏大运 + 当前专题好流年 → 该专题相对好（但底色没变）
    ├── 当前专题坏大运 + 当前专题差流年 → 该专题容易爆雷
    └── 墓库运：看天干透出的字决定运的性质

5. 输出
    ├── 当前大运：[大运干支]（X-X岁，XXXX-XXXX年）
    ├── 专题结论：[事业财运/婚姻感情/健康灾祸/父母子女] → [吉/凶/平] — 置信度：X
    ├── 当前流年：[干支] → 当前专题[吉/凶/平] — 置信度：X
    ├── 专题外信息：[如婚姻动荡/父母健康/健康伤灾]，仅在用户询问整体或特别需要提醒时另列
    └── 建议：[只针对当前专题的应对方向]

禁止项：
    ├── 财运分析中，不得因夫妻宫受冲穿刑害而否定财运好坏。
    ├── 财运分析中，不得因健康三刑、父母星受伤、子女信息而改断为坏财运。
    ├── 婚姻分析中，不得因财运好而覆盖夫妻宫、夫妻星的不利。
    └── 整体分析时，不输出单一“大运好/坏”，只输出分专题并列结论。
```

## 决策树8：六亲健康判断

> **触发条件**：第10.3-10.4步分析六亲健康时
> **前提**：已定位六亲（父母星/子女星/兄弟姐妹星）

```
输入：六亲星位置 + 该字在地支的状态 + 地支关系

1. ⚠️ 核心原则：不看喜忌，只看状态
    └── 这是实战派体系和传统子平最大的区别之一

2. 检查六亲星的受伤情况
    ├── 该字被穿（六害）→ 最严重，必有健康问题
    │    └── 被穿的位置对应身体部位
    ├── 该字被冲 → 严重，突然发病
    ├── 该字被刑（三刑）→ 长期慢性病
    ├── 该字被克（天干克/地支藏干克）→ 一般健康问题
    ├── 该字入墓 → 能量被收藏，功能减弱
    └── 该字遇绝 → 极度危险

3. 看月令状态
    ├── 在月令当令（能量最强）→ 身体好
    ├── 在月令失令（弱）→ 身体弱
    └── 在月令被月令刑冲 → 身体不好

4. 判断健康问题部位
    ├── 金被克 → 肺/呼吸系统/骨骼
    ├── 木被克 → 肝/胆/四肢
    ├── 水被克 → 肾/泌尿系统/耳
    ├── 火被克 → 心脏/眼睛/神志
    └── 土被克 → 脾胃/消化系统

5. 特殊健康判断
    ├── 寅巳穿 → 头疼/四肢筋骨
    ├── 子未穿（女）→ 子宫肌瘤/卵巢囊肿
    ├── 卯辰穿 → 肝胆/皮肤
    ├── 酉戌穿 → 牙齿/心脏
    ├── 丑未戌刑 → 胃病（土硬）
    └── 食伤受刑 → 抑郁症风险高

6. 输出
    ├── 六亲：[母/父/子女/配偶] — 健康：[好/注意XX/差] — 置信度：X
    └── 判断依据：[引用的规则和推理]
```
