# 排盘工具集成指南

> **用途**：指导 AI 如何调用八字排盘工具并正确解析输出结果。

---

## 一、工具概况

| 项目 | 内容 |
|------|------|
| **工具位置** | `scripts/bazi_engine.py` |
| **核心函数** | `generate_chart()` — 一个输入出一份完整排盘 JSON |
| **依赖库** | `lunar-python` v1.4.8（已 vendor 到 `scripts/vendor/` 目录） |
| **输出格式** | JSON + 纯文本 |
| **命令入口** | `scripts/chart_cli.py` |

---

## 二、调用方式

### 方式1：Python 直接调用（推荐）

```python
import sys
sys.path.insert(0, "scripts")
from bazi_engine import generate_chart, json_dumps

# 输入格式（阳历）
chart = generate_chart({
    "calendar_type": "solar",       # "solar" 或 "lunar"
    "year": 1990,
    "month": 5,
    "day": 12,
    "hour": 14,                     # 0-23
    "minute": 30,
    "gender": "male",               # "male" 或 "female"
    "sect": 2,                      # 2=晚子时日柱算当天（推荐）
    "yun_sect": 1,                  # 1=按时辰数起运（推荐）
    "da_yun_count": 10,             # 大运数量
})

# 获取纯文本格式（适合直接使用）
plain_text = chart["plain_text"]
print(plain_text)

# 获取完整 JSON（包含全部详细字段）
full_json = json_dumps(chart)
```

### 方式2：通过 Bash 调用

```bash
python3 scripts/chart_cli.py --calendar solar --year 1990 --month 5 --day 12 --hour 14 --minute 30 --gender male
```

### 方式3：手动录入（工具不可用时）

直接接收用户提供的排盘结果，验证后再使用。

## 三、输入参数说明

| 参数 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| `year` | ✅ | 出生年（如 1990） | — |
| `month` | ✅ | 出生月（1-12） | — |
| `day` | ✅ | 出生日（1-31） | — |
| `hour` | ✅ | 出生时（0-23） | 0 |
| `minute` | ❌ | 出生分（0-59） | 0 |
| `gender` | ✅ | "male" 或 "female" | "male" |
| `calendar_type` | ❌ | "solar"(阳历) 或 "lunar"(农历) | "solar" |
| `is_leap_month` | ❌ | 是否农历闰月 | false |
| `sect` | ❌ | 2=晚子日柱算当天；1=算明天 | 2 |
| `yun_sect` | ❌ | 1=按时辰；2=按分钟 | 1 |
| `da_yun_count` | ❌ | 生成大运步数 | 10 |

### sect 字段说明

| sect | 规则 | 适用场景 |
|------|------|---------|
| 1 | 晚子时（23:00-00:59）日柱算明天 | 传统子平法 |
| **2（推荐）** | **晚子时日柱算当天** | **实战派体系常用** |

### yun_sect 字段说明

| yun_sect | 规则 | 说明 |
|---------|------|------|
| **1（推荐）** | 按天数+时辰数起运 | 传统算法 |
| 2 | 按分钟数起运 | 更精确但与传统稍有偏差 |

## 五、注意事项

### 5.1 真太阳时校正

排盘工具**未做真太阳时校正**（标注在 `rules.time_standard` 字段中）。

- 出生在东部沿海（北京、上海、广州等）：北京时间 ≈ 真太阳时，影响很小
- 出生在中国西部（四川、新疆、西藏等）：北京时间 - 真太阳时可差 **0.5-1.5小时**
- 影响：可能导致**时柱错误**

**建议流程**：
```
用户提供出生地信息
    │
    ├── 不提供出生地 → 按北京时间排盘，提示"未校正真太阳时"
    └── 提供出生地
         ├── 东部 → 按北京时间排盘基本准确
         └── 西部 → 手动计算真太阳时校正
```

### 5.2 子时（23:00-00:59）处理

子时跨两天，需注意：
- `sect=2`（推荐）：23:00-00:59 的日柱算**当天**
- `sect=1`：23:00-00:59 的日柱算**第二天**
- 建议使用 `sect=2`

### 5.3 时间缺失处理

如果用户没有出生时辰：
- 只能排**三柱**（年/月/日）
- 无法计算**大运**
- 所有涉及时柱和大运的判断**置信度降为低**
- 部分判断（如时柱代表的事业/子女）**不做或标注极低置信度**

### 5.4 性别缺失处理

- 必须提供性别才能计算大运
- 如果不确定性别：可以做四柱分析，但不做**大运流年**分析
