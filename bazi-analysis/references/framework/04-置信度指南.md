# 置信度指南

> **用途**：统一规范分析结论中的置信度标注，让使用者明确知道每个判断的可信程度。
> **原则**：宁可低估不可高估，不确定就标"低"。

---

## 一、置信度分级标准

| 置信度 | 含义 | 表述规范 | 适用条件 |
|--------|------|---------|---------|
| **高 (≥90%)** | 结构明确，可靠性很高 | "明确倾向..." "结构上很明显..." "大概率..." | 有明确核心规则或多个独立证据支持 + 无未消解矛盾 |
| **中 (70-89%)** | 有规则支持但需综合考虑 | "倾向于..." "通常认为..." "需要结合..." | 有规则支持但存在权衡因素、样本不足或已消解冲突 |
| **低 (50-69%)** | 多因素影响，需灵活判断 | "可能有..." "不排除..." "从XX角度来看..." | 依赖多因素综合且无明确规则 |
| **极低 (<50%)** | 高度不确定 | "不确定..." "仅供参考..." "需结合更多信息..." | 缺乏明确依据或矛盾因素太多 |

---

## 二、各分析项的默认置信度

### 第2步：定日主

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 任何情况 | **高** | 日主是确定的，没有歧义 |

### 第5步：判旺衰看根气

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 有明确明现比劫或明现印（坐支或他支） | **高** | 根气判断标准明确 |
| 有对应库根 | **中-高** | 算有根气，但力量需结合位置 |
| 只有中气或余气 | **高（按无根处理）** | 中气根、余气根不算有根气 |
| 完全无根（无明现比劫、无印、无库） | **高** | 明确无根 |

### 第6步：取用神

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 月令明显 + 制化途径清晰（无七杀干扰） | **高** | 规则明确 |
| 月令明确但制化有多个选项 | **中** | 需权衡选择最优方案 |
| 四墓库月 | **低→中** | 两层做功较复杂 |
| 有七杀干扰（有杀先论杀） | **高（杀的处理）/ 中（整体用神）** | 杀的处理规则明确，但整体协调需权衡 |
| 特殊格局（从格等） | **低** | 特殊格局规则边界较模糊 |

### 第7步：地支关系

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 关系明确 + 月令力量清晰 | **高** | 规则明确 |
| 关系存在但力量比较接近 | **中** | 需额外判断力量对比 |
| 多组关系叠加交互 | **低→中** | 复杂交互需综合判断 |
| 天干合+地支绝（天合地穿） | **高** | 明确规则 |

### 第8步：四墓库

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 开库/闭库条件明确 | **高** | 开库5法/闭库4法明确 |
| 家里的比劫库制年月的财库 = 发财 | **高** | 明确规则 |
| 年月的比劫库制家里的财库 = 破财 | **高** | 明确规则 |
| 四墓库月综合判断 | **低→中** | 两层做功较复杂 |

### 第10步分项：婚姻

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 星宫关系明确（合/冲/穿/刑/绝） | **高** | 规则明确 |
| 比劫合夫妻宫 = 出轨 | **高** | 明确规则 |
| 夫妻星与比劫同柱 = 出轨 | **高** | 明确规则 |
| 关系存在但有其他合解 | **中** | 需权衡综合因素 |
| 无明确星宫关系 | **低** | 缺乏明确依据 |
| 具体应事年份 | **中→低** | 多动态因素 |

### 第10步分项：财运

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 库做功方向明确 | **高** | 规则明确 |
| 用神在年上天地一气 | **高** | 明确规则 |
| 食伤制官杀等中等方式 | **中** | 需综合判断 |
| 多种做功方式混杂 | **低** | 需要权衡 |
| 具体财富数字 | **极低** | 只能判层级不能判精确数字（不给） |

### 第10步分项：大运流年

| 情况 | 置信度 | 理由 |
|------|--------|------|
| 大运与月令作用关系明确 | **高** | 规则明确 |
| 大运与用神喜忌明确 | **高** | 规则明确 |
| 大运分专题吉凶判断 | **中→高** | 需考虑当前专题的星、宫、做功链和运年作用；不作单一总吉凶 |
| 流年具体应事 | **低→中** | 多动态因素综合 |
| "那年一定会怎样" | **极低** | 不给出确定性判断 |

## 三、置信度调整规则

### 上调因素（可升半级或一级）

| 因素 | 调整幅度 | 适用场景 |
|------|---------|---------|
| 有明确铁律支持 + 无任何矛盾因素 | +1级 | 所有场景 |
| 反向验证通过 | +1级 | 所有有反向验证通道的场景 |
| 多个独立指标指向同一结论 | +0.5级 | 取用神/婚姻/财运等 |
| 3条及以上规则一致 | 可增强到高 | 作为增强因素，不作为高置信度的硬门槛 |

### 下调因素（可降半级到两级）

| 因素 | 调整幅度 | 适用场景 |
|------|---------|---------|
| 有矛盾指标且无法通过规则消解 | -1级到低/极低 | 按冲突严重度和证据薄弱程度决定 |
| 反向验证不通过 | -1到-2级 | 所有有反向验证通道的场景 |
| 信息不全（如缺出生时辰） | -1到-2级 | 时柱缺失时 |
| 多因素冲突但优先级规则不能覆盖 | -1级 | 综合判断场景 |
| 本分析领域属于已知的"低准确率领域" | -1级 | 具体应事年份/具体数字 |

## 四、特殊标注规则

### 必须标注置信度的位置

以下每个结论后必须标注置信度（不可省略）：
- [ ] 日主五行 → 自动高，可省略
- [ ] 根气判断 → 必须标注
- [ ] 取用神（用神和忌神）→ 必须标注
- [ ] 婚姻稳定性 → 必须标注
- [ ] 财富层次 → 必须标注
- [ ] 大运分专题吉凶 → 必须标注
- [ ] 流年分专题吉凶 → 如果给出必须标注

### 应避免的使用场景

```
❌ "此八字一定会离婚" → 即使高置信度，也要加"大概率"
❌ "今年一定会发财" → 流年应事最多给中置信度
❌ "财产有XXXX万" → 具体数字不给，只给层级
❌ "XX岁结婚/离婚" → 具体年龄不给，给年份段
❌ "此八字肯定..." → 高置信度也不要当作事实断言
✅ "婚姻不稳定（置信度：高），二婚可能性大"
✅ "今年财运较好（置信度：中），建议把握XX"
✅ "财富层次在千万级（置信度：高）"
```

### 置信度与建议的关系

| 置信度 | 应如何建议 |
|--------|-----------|
| 高 | 可以明确建议，"建议你不要做XX，因为XX" |
| 中 | 温和建议，"建议考虑XX方向，但需谨慎评估" |
| 低 | 仅做参考，"不排除XX可能，建议多方面考量" |
| 极低 | 不做建议，"信息不足以做出判断" |

## 五、反向验证流程

对所有"高置信度"的结论，增加反向验证：

```
正向推理结论：[例如：此八字喜火]
    │
    ▼
反向验证问题：
├── 如果是喜火 → 他走巳午大运应该好
├── 如果是喜火 → 他在火旺年（巳/午年）应发财
└── 如果是喜火 → 火为用神应该得到生扶
    │
    ├── 全部符合 → 维持"高"置信度
    ├── 部分符合 → 降为"中"置信度
    └── 都不符合 → 降为"低"置信度，复核取用神
```
