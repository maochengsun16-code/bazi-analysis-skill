# 冲突消解规则

> **用途**：当八字分析中出现多个指标指向不同结论时，按本文件的优先级规则裁决。
> **原则**：排在前的规则优先于排在后的规则。铁律不可违背。

---

## 规则0：特殊优先级 > 普通月令格局

| 项目 | 说明 |
|------|------|
| **冲突场景** | 月令可以按普通财官印食伤取用，但原局同时出现库做功、强七杀或寒燥偏枯 |
| **裁决** | **先处理特殊优先级**：四墓库做功 / 强七杀 / 寒燥调候 > 普通月令格局 |
| **推理** | 普通月令格局是常规主线，但库做功直接决定财富层级和做功方向，强七杀是必须先处理的压力源，寒燥偏枯是原局气候病。三者成立时，不能只按月令十神机械取用 |
| **执行** | 先判断特殊分支是否成立；若成立，以该分支定第一用神或核心忌神，再回到月令十神检查是否同向。同向时合并为同一喜用神证据链；不同向时按特殊分支优先，月令格局提供次级权重和断事背景 |
| **示例** | 亥子丑月即使月令十神为印/财，也要先检查寒重是否需要火调候；巳午未月先检查水调候；七杀在月令先论杀；辰戌丑未形成家内外冲库做功则先看库 |

### 规则0.1：特殊分支与月令取用同向时合并判断

| 项目 | 说明 |
|------|------|
| **适用场景** | 寒燥调候、强七杀、库做功等特殊分支取出的用神，与月令格局取出的用神方向一致 |
| **裁决** | 不视为冲突，也不把后者降为“辅助喜神”；两者合并为喜用神证据链，并明确喜用神内部优先级 |
| **判断方法** | 一字能同时解决多个问题时，列为最优喜用神；其他能制化旺神、耗泄忌神、或生助最优用神的字，也列为喜用神，但说明作用层级弱于最优用神 |
| **示例** | 丙火日主生巳午未月，燥重取水；若月令为禄刃/比劫旺，官杀水又能制比劫，则水同时承担调候和制比劫，为最优喜用神。金财能耗比劫并生水，也为喜用神，但弱于水官杀 |

## 规则1：五行喜忌 > 神煞

| 项目 | 说明 |
|------|------|
| **冲突场景** | 日主喜火忌水，但命带"天乙贵人"和"桃花"（吉神） |
| **裁决** | **以五行喜忌为准**，神煞仅为辅助 |
| **推理** | 神煞是锦上添花，不是雪中送炭。命主即使有贵人星，如果走忌神大运，贵人也不会出现  |
| **依据** | E1-神煞定位与使用原则.md → "旺衰 > 喜忌 > ... > 神煞" |
| **示例** | 八字喜火，但命带空亡。空亡在五行忌神位置则凶；空亡在五行喜神位置则吉 |

## 规则2：根气判断 > 得令判断

| 项目 | 说明 |
|------|------|
| **冲突场景** | 日主生于失令月（如甲木生于秋天申月），但地支有明现比劫、明现印或对应库 |
| **裁决** | **以根气为准**——有根就能担财官用食伤，不在乎是否得令 |
| **推理** | 根气是实战派的核心认知。得令≠能做事，失令≠不能做事。只要地支有正式根气，日主就有做事的能力。正式根气只看明现比劫、明现印、对应库；中气根和余气根不算有根气 |
| **依据** | G-八字全流程.md 第5步 → "关键在于根气，不在乎是否得令"；D2-透干根气与强弱判断.md |
| **示例** | 乙木生于申月（绝地），但日支见卯木明现比劫 → 有根，能担财官 |

## 规则3：月令十神格局 > 普通单一十神

| 项目 | 说明 |
|------|------|
| **冲突场景** | 月令本气为某十神，但原局另有一个零散十神或组合看起来也很明显 |
| **裁决** | **以月令十神格局为主**——月令是原局最强十神，普通单一十神不能越过月令定调 |
| **推理** | 月令决定了原局最强环境和主要矛盾。普通十神组合可以补充取象，但不能替代月令十神。只有规则0中的库做功、强七杀、寒燥调候等特殊分支成立时，才可优先于普通月令格局 |
| **依据** | G-八字全流程.md 第3步 + D4-取用神方法论.md |
| **示例** | 月令为财，原局另有印星。除非寒燥、库做功、强杀等特殊分支成立，否则仍先按财月/财格处理，再看印是否能用 |

## 规则4：有杀先论杀 > 其他取用规则

| 项目 | 说明 |
|------|------|
| **冲突场景** | 月令为财月（按规则应用印比），但原局有七杀透出 |
| **裁决** | **优先处理七杀**——有杀先论杀是最高优先级原则 |
| **推理** | 七杀是十神中最凶之神，如同一个人身上有致命伤时，要先止血再处理其他问题。先看七杀怎么处理（印化/食伤制/比劫扛），再考虑月令的大方向 |
| **依据** | D4-取用神方法论.md → "有杀先论杀" |
| **注意** | 地中藏干中的七杀也算，不要遗漏 |

## 规则5：夫妻星星宫关系 > 夫妻星多寡

| 项目 | 说明 |
|------|------|
| **冲突场景** | 男命有三个正财（多妻星），但夫妻宫被合进来（星宫关系稳定） |
| **裁决** | **以星宫关系为准**——合进来的是能稳定，财星多只是心花但不会离 |
| **推理** | 实战派体系中，婚姻稳定性由夫妻宫（日支）和夫妻星之间关系的性质决定，而不是由夫妻星的数量决定。正偏财混杂/官杀混杂代表桃花多、机会多，但最终能不能稳定看星宫关系。合进来=绑住了，冲出去=要离 |`
| **依据** | F2-婚姻感情判断.md |
| **示例** | 男命三个正财（多妻星），但夫妻宫被合进来 → 桃花多但不会离 |

## 规则6：六亲健康看状态 > 看喜忌

| 项目 | 说明 |
|------|------|
| **冲突场景** | 印星是日主忌神（按理说不好），但在原局没有被刑冲克害（状态好） |
| **裁决** | **不看喜忌只看状态**——印星虽为忌但没有受伤，母亲的身体就是好的 |
| **推理** | 这是实战派体系最反直觉但最重要的原则之一。六亲的健康状态不由"这个六亲星是喜是忌"决定，而是由"这个六亲星在原局是否受伤"决定。受伤=身体不好；没受伤=身体好（即使它是忌神）。**喜忌管的是关系好坏，状态管的是身体好坏** |
| **依据** | F4-子女与父母判断.md → "不看喜忌，只看状态" |
| **示例** | 偏财是忌神（和父亲关系不好），但偏财在原局没有被刑冲 → 父亲身体好（只是关系差） |

## 规则7：大运先与月令作用 > 大运与日主作用

| 项目 | 说明 |
|------|------|
| **冲突场景** | 大运来了火，日主喜火应好（对日主有利），但月令已经是火且过旺（对月令不利） |
| **裁决** | **先看大运与月令的作用关系**——月令是全局最旺，大运先影响月令 |
| **推理** | 月令是八字中能量最强的字。大运的气运来了，第一个冲击的就是月令这个最大能量源。月令被影响后，再传导到全局。所以"大运对日主好不好"是第二位的，"大运对月令好不好"才是第一位 |
| **依据** | F1-大运流年判断.md |
| **示例** | 月令巳火过旺为忌，大运午火虽为用神（日主喜火），但午火加强巳火忌神 → 此运凶（因为大运先与月令作用，加强了忌神） |

## 规则8：原局结构 > 大运流年

| 项目 | 说明 |
|------|------|
| **冲突场景** | 原局官杀有印化（有结构优势），但大运走了财运（财生官杀克印，似乎坏了结构） |
| **裁决** | **原局有印化杀的结构依然存在**——大运只是临时改变了动态，不能从根本上改变原局结构 |
| **推理** | 原局是"命"，大运流年是"运"。命不会因为运而改变。大运可以引动原局的某个部分（使其变强或变弱），但不能摧毁原局已有的结构。原局的印化杀结构是一个"底牌"，大运来了财官只是让这张底牌暂时被压制，不是消失。走完运还会恢复 |
| **依据** | F1-大运流年判断.md |
| **示例** | 原局有印化杀结构（好），走财运（财生官杀克制印）→ 此运有财来找麻烦，但底牌还在 |

## 规则9：冲刑害 > 合

| 项目 | 说明 |
|------|------|
| **冲突场景** | 地支既有六合（合作）又有六冲（交战） |
| **裁决** | **冲的力量大于合**——先看交战的后果，再考虑合的解救作用 |
| **推理** | 不能用一个全局力度排序机械覆盖所有专题；但当同一组关系里既有合又有冲、破、穿、刑、绝等伤害关系时，应先看伤害关系是否破坏合作，再看合能否缓解。冲是直接的、剧烈的撞击；合是温和的、渐进的合作。在冲合并见时，冲的破坏力通常大于合的修复力。破绝等关系要回到夫妻、六亲、合作、财运等具体专题取象 |
| **依据** | B6-地支作用关系全览.md |
| **示例** | 地支有子午冲（水火交战）+ 子丑合（合作解决）→ 先有子午冲，子丑合只能部分缓解 |

## 规则10：本气 > 余气

| 项目 | 说明 |
|------|------|
| **冲突场景** | 本气（如寅中甲木）没有被天干透出，但余气（如辰中乙木）透出 |
| **裁决** | **本气即使不透露，力量也大于透出的余气** |
| **推理** | 本气是一个地支最主要的气，占比约60%以上。余气占比不超过10-20%。透出使能量外显，但透出不能改变力量的根本大小。一个不透露的本气，力量仍然大于一个透出的余气 |
| **依据** | B1-地支本质与藏干.md + D2-透干根气与强弱判断.md |
| **示例** | 寅中甲木（本气）不透，辰中乙木（余气）透出 → 寅中的甲木力量仍然大于辰中透出的乙木 |

## 多因素冲突处理流程

当有3个以上的指标互相冲突时，按以下流程逐步消解：

```
输入：多个冲突指标
    │
    ▼
步骤1：识别哪些是"铁律"
    ├── 是否有库做功、强七杀、寒燥调候？
    ├── 有杀先论杀？
    ├── 六亲健康不看喜忌看状态？
    └── 铁律优先处理，不考虑其他因素
    │
    ▼
步骤2：铁律处理后，看剩余的硬规则
    ├── 五行喜忌 vs 神煞 → 五行喜忌赢
    ├── 月令十神格局 vs 普通单一十神 → 月令赢
    └── 大运 vs 原局 → 原局结构赢
    │
    ▼
步骤3：剩下的软规则综合权衡
    ├── 纳音、十二长生等作为辅助参考
    └── 如果仍有矛盾，标记为"低置信度"
    │
    ▼
步骤4：同级别规则的冲突处理
    ├── 同级别冲突 → 看具体的"力量对比"（谁在月令谁强）
    ├── 力量相近 → 看数量对比（哪个因素更多）
    └── 数量也相近 → 标记为"低置信度"，给出多种可能
    │
    ▼
输出：消解后的结论 + 置信度
```
