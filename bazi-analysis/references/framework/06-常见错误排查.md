# 常见错误排查指南

> **用途**：收集 AI 分析八字时最容易犯的错误，反向排查用。
> **原则**：先对照这个清单检查，再输出报告。

---

## 一、致命错误（必须避免）

### 错误1：用神判断基于身强身弱

```
❌ 错误用法：日主得令身强，所以用克泄耗
✅ 正确用法：制化八字中最旺五行的字即为用神
            月令为印旺，就用财制印/比劫化印
            不看日主身强身弱，看全局哪个五行最强

为什么错：身强身弱是传统子平法的思路，实战派体系的核心是"制化最旺"
影响：用神取反，后面所有判断全错（连锁反应）
```

### 错误2：月令用来判断日主强弱

```
❌ 错误用法：日主生于XX月，得令/失令，所以身强/身弱
✅ 正确用法：月令是全局环境，决定哪个五行最强
            日主有没有能力看根气（第5步）

为什么错：这是最大的认知偏差——月令决定的是"环境"不是"日主"
影响：高估或低估日主能力，影响担财官用食伤的判断
```

### 错误3：神煞作为主要判断依据

```
❌ 错误用法：带天乙贵人所以大富大贵 / 逢桃花所以乱
✅ 正确用法：神煞在五行喜忌和地支关系之后使用（优先级最低）
            桃花为喜=有魅力/桃花为忌=烂桃花

为什么错：神煞只是锦上添花，不能替代五行分析
影响：判断不准，容易误判
```

### 错误4：夫妻星用传统"男才女官"定义

```
❌ 错误用法：男命正财/偏财就是老婆，女命正官/七杀就是老公
✅ 正确用法：财星和官杀都可代表配偶，性别只决定优先级
            男命财星优先，女命官杀优先；第一优先级不成立再看另一类
            候选夫妻星必须与日柱发生作用：天干合冲日干，或地支作用夫妻宫
            无作用的财官只按桃花、短缘或露水情缘看

为什么错：实战派体系的夫妻星定义与传统不同
影响：找不到真正的夫妻星，婚姻判断全错
```

### 错误5：六亲健康看喜忌

```
❌ 错误用法：印星为忌，所以母亲身体不好
✅ 正确用法：印星虽为忌，但没有被刑冲克害，母亲身体就是好的
            喜忌决定"关系好不好"，状态决定"身体好不好"

为什么错：这是最反直觉但最重要的原则
影响：六亲健康判断全错
```

### 错误6：忽略四墓库的两层做功

```
❌ 错误用法：四墓库月也按月令取用，只看一层
✅ 正确用法：土本身一层做功 + 土中透出之物另一层做功
            两层都要看

为什么错：四墓库月的八字比其他月复杂一倍
影响：财富判断丢失重要信息
```

### 错误7：大运分前五年后五年

```
❌ 错误用法：大运前五年看天干，后五年看地支
✅ 正确用法：大运重地支，整十年都看地支，不分前后

为什么错：实战派体系与传统大运分法不同
影响：对运的判断在高发期内分割，导致错误应期判断
```

## 二、常见遗漏

### 遗漏1：忘记检查藏干中的七杀

```
情况：透干中没有七杀，但藏干中有
后果：违反"有杀先论杀"原则，取用神错误
检查点：四柱地支的藏干是否都检查了七杀？
        特别要注意：寅中有甲木（无七杀）
                    申中有庚金（藏干中的七杀）
```

### 遗漏2：忘记看旬空是否出空

```
情况：某十神在空亡位置，但大运/流年已出空
后果：漏判重要应事
检查点：当前大运/流年是否填实了旬空？
```

### 遗漏3：忘记看六害（穿）对健康的影响

```
情况：只分析冲和刑，忽略了穿
后果：漏判严重健康问题（穿比冲刑更伤人）
检查点：六组六害是否都检查了？
        寅巳穿/子未穿/卯辰穿/酉戌穿/申亥穿/丑午穿
```

### 遗漏4：只看单一地支关系，忽略多关系叠加

```
情况：看到子丑合就认为好，没看到子午冲同时存在
后果：判断过于片面
检查点：同一个地支有几种关系？有没有"合中带破""合中带冲"？
```

### 遗漏5：把"合"直接等同于好

```
情况：看到三合六合就认为"合作好"
后果：忽略了合中带破（寅亥合/巳申合）、合出忌神等负面情况
检查点：合出来的五行是喜神还是忌神？合中是否带破？
```

### 遗漏6：不区分"合"和"化"

```
情况：看到天干五合就认为合化了
后果：实际上合不等于化，合化有四个条件
检查点：合化的四个条件是否都满足？
        位置邻近？化神有根？月令支持？地支配合？
```

## 三、过度判断

### 过度1：没有根气仍断"能担财官"

```
情况：日主完全无根（无明现比劫、无明现印、无对应库；只有中气/余气也按无根处理）
后果：过度乐观判断日主能力
纠正：无根 → 需谨慎，需大运补根
      无根+无生扶+失令 → 从格倾向
```

### 过度2：判断具体年份的具体事件

```
情况："2028年一定发财" / "2030年一定离婚"
后果：用户信以为真，但实际上没有八字能确定到具体年份的100%
纠正：→ "2028年财运较好（置信度：中），建议把握机会"
      → "2030年婚姻有风险（置信度：中），建议注意沟通"
```

### 过度3：判断具体财产数字

```
情况："此人财产有500万"
后果：八字只能判层次，不能判精确数字
纠正：→ "财富层次在千万级（置信度：中）"
      → 可给范围，不给精确数字
```

### 过度4：使用"一定会""绝对"等绝对表述

```
情况："此人一定会发财" / "绝对会离婚"
后果：用户产生不切实际的期待
纠正：→ "此八字财运好（置信度：高）"
      → "婚姻有离婚风险（置信度：高）"
      保留不确定性的言语空间
```

## 四、特殊场景排查

### 场景1：子时出生（23:00-00:59）

```
问题：子时跨越两天，日柱可能不同
后果：日柱算错了，所有十神和用神都错
纠正：检查 sect 设置是否与用户确认
      sect=1 → 日柱算明天
      sect=2 → 日柱算当天
```

### 场景2：出生地远离东八区（北京时间）

```
问题：中国西部（新疆/四川西部等）用北京时间与真太阳时可差1小时以上
后果：时柱可能算错
纠正：提示用户"未做真太阳时校正"
      如果用户提供出生地经纬度，手动计算校正
```

### 场景3：农历闰月出生

```
问题：闰月出生的人，月柱计算容易出错
后果：月柱错误 → 月令错 → 取用神错
纠正：排盘工具已处理闰月（is_leap_month参数）
      确认排盘时是否正确设置了闰月参数
```

### 场景4：时间缺失（没有出生时辰）

```
问题：没有时辰只能排三柱，时柱、大运都缺失
后果：信息不全，多个判断只能给"低"置信度
纠正：明确告知用户"没有时辰，分析不完整"
      所有涉及时柱的判断标注"低"置信度
```

## 五、排查流程

```
分析完成 → 输出报告前
    │
    ▼
快速扫描本文件中的"致命错误"：
[ ] 错误1：用神基于身强身弱？（不是制化最旺？）
[ ] 错误2：月令判断日主强弱？（不是看根气？）
[ ] 错误3：神煞做主要判断？（喧宾夺主？）
[ ] 错误4：夫妻星用传统定义？（不是星宫关系？）
[ ] 错误5：六亲健康看喜忌？（不看状态？）
[ ] 错误6：忘记四墓库两层做功？
[ ] 错误7：大运分前五年后五年？
    │
    ▼
扫描"常见遗漏"：
[ ] 遗漏了藏干中的七杀？
[ ] 忘记了六害（穿）？
[ ] 只看单一关系，没看多关系叠加？
[ ] 合＝好的思维定势？
[ ] 没区分合和化？
    │
    ▼
扫描"过度判断"：
[ ] 有"一定/绝对/100%"的表述？
[ ] 判断了具体年份的具体事件？
[ ] 给出了具体数字？
    │
    ▼
全部通过 → 输出报告
未通过 → 修正后再输出
```
