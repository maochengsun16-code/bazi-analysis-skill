## RULE-20260630-01 正印不生、偏印生与幼年抚养关系特殊

- 规则名称：正印不生、偏印生，幼年亲生父母直接抚养关系不稳
- 分类：整体分析/父母分析
- 规则层级：专题应用规则
- 所属基础原理：印星、正偏印分工、月令本气十神
- 确认状态：已确认
- 置信度：中

### 适用条件

1. 不分男女，男女命均可使用。
2. 原局偏印当令，或偏印在月柱/父母宫强旺。
3. 原局正印不明现，或正印虽藏但不形成稳定生身关系。
4. 命局由偏印承担主要生身、养育、资源供给信息，即“正印不生、偏印生”。
5. 若要断为具体事件，最好有反馈或大运流年引动父母宫、印星作为验证。

### 判断逻辑

正印与偏印都属于印星，都能代表养育、庇护、资源和长辈缘分。正印不明现或不参与生身，而偏印当令、偏印强旺并承担生身时，养育信息偏向非典型、非直接、非标准的亲生父母抚养模式。此时可倾向判断幼年不在亲生父母或亲生母亲身边长大，或者亲生父母直接照顾不足。

这种应象的现实表现可以不同：祖辈或亲属带大、母亲外出做生意、父母离异、寄养、长期分离等。具体原因不得只凭此规则强断。

### 输出断语

```text
原局偏印当令或偏印很旺，正印不明现、不生身，属于“正印不生、偏印生”。不分男女，都容易有幼年亲生父母直接抚养关系不稳的信息，可能表现为不在亲生父母或亲生母亲身边长大、由祖辈亲属带大、父母分离或母亲长期外出等。具体原因需要结合反馈确认。
```

### 不适用条件

1. 正印明透有根，并与日主形成稳定生身关系。
2. 偏印虽出现但不当令、不成势，也不承担主要生身。
3. 父母宫或印星被其他稳定合生关系修复，且现实反馈显示亲生父母抚养稳定。
4. 不能仅凭“有偏印”就断不在父母身边，必须满足正印不生与偏印成势。

### 模块边界

| 对比模块 | 区别 | 不能套用的地方 |
|---|---|---|
| 父母健康 | 本规则看幼年抚养关系和亲生父母缘分，不看父母身体状态 | 不能因为偏印生身就断父母身体好坏 |
| 学业资源 | 印也可代表学习资源，但本规则强调抚养与养育结构 | 不能直接断学历高低 |
| 财运 | 印旺可能影响行动力，但本规则不直接判断财运吉凶 | 不能用幼年抚养规则否定财运 |

### 冲突处理

若正印明透有根且稳定生身，优先考虑正印的标准养育信息；若偏印当令但被强财制住，偏印的抚养异常信息下降。该规则低于明确现实反馈，若反馈亲生父母稳定抚养，应标为反例或降置信度。

## RULE-20260630-02 寒重局辰湿土水库引动财务压力

- 规则名称：寒重忌水湿土，辰运辰年引动子辰半合与辰辰自刑，财务压力增强
- 分类：事业财运
- 规则层级：专题应用规则
- 所属基础原理：寒燥调候、辰为湿土水库、地支半合、自刑、财库
- 确认状态：已确认
- 置信度：中

### 适用条件

1. 原局生于亥子丑冬令，寒重，火为调候第一用神。
2. 原局喜火、燥土，忌水、湿土。
3. 原局有辰，且辰参与财星、财库、财路或事业财运相关位置。
4. 大运或流年再见辰，形成辰辰自刑，或辰与原局子水形成子辰半合水。
5. 财运判断必须回到财星、财库、财路、现金流、债务、回款等事项，不可跨模块套断。

### 判断逻辑

辰为湿土、水库。寒重命局本已忌水湿，运年再见辰时，辰不一定按普通土财看，必须先看它是否增强水气、湿气。若原局已有子水，辰来形成子辰半合水，寒湿更重；若原局已有辰，流年或大运再见辰，辰辰自刑，财库或财星位置被刑动，容易表现为为钱难受、亏损、债务压力、项目资金卡住、回款不顺。

### 输出断语

```text
这个命局寒重，喜火燥土，不喜水湿土。辰是湿土、水库，辰运或辰年再来，容易引动子辰半合水、辰辰自刑，财星财库受刑，财务上多为钱发愁，亏损、债务、回款不顺的压力会增强。
```

### 不适用条件

1. 原局不寒，或不忌水湿土。
2. 辰不参与财星、财库、财路或事业财运链条。
3. 运年辰被强火燥土有效转化，寒湿不成问题。
4. 原局形成更高优先级的四墓库做功发财机制，且辰被有效开库、制库、护库。

### 模块边界

| 对比模块 | 区别 | 不能套用的地方 |
|---|---|---|
| 婚姻 | 财运中看辰是否伤财星、财库、财路 | 不可见辰辰自刑就直接断婚姻危机 |
| 健康 | 健康中看受伤五行和身体取象 | 不可把财库自刑直接断疾病 |
| 父母子女 | 六亲中看对应六亲星状态 | 不可因财运不利直接断六亲不利 |

### 冲突处理

若流年同时见火旺，需分开判断：财运维度可因火调候得到局部改善，但辰湿土引动的资金压力仍需提示。若火能有效制湿、暖局并生财，则财务压力下降，规则置信度降低。

## RULE-20260630-03 伤官正官同宫遇夫妻宫填实，婚姻不满增强

- 规则名称：女命伤官与正官同宫，运年填实夫妻宫，婚姻不满和外缘风险增强
- 分类：婚姻感情
- 规则层级：专题应用规则
- 所属基础原理：伤官见官、夫妻宫、夫妻星定位、大运流年填实
- 确认状态：已确认
- 置信度：中-高

### 适用条件

1. 女命分析婚姻，以官杀为优先夫妻星。
2. 日支夫妻宫藏伤官与正官，或伤官与正官同柱同宫。
3. 大运或流年填实夫妻宫，或伏吟夫妻宫。
4. 当前运年同时引动伤官与官星，形成婚姻评价下降、挑剔伴侣、想离婚或换人的信息。
5. 若要断婚外情，需要另有比劫、外缘星、夫妻宫被合冲穿刑、或现实反馈支持。

### 判断逻辑

女命以官杀为配偶星。伤官与正官同宫，本身就有伤官见官的信息，容易对配偶能力、做事方式、责任感产生不满。若大运或流年再填实夫妻宫，伤官和官星同时被引动，婚姻不满会明显增强，容易出现想离婚、嫌弃伴侣、外缘诱惑或关系动荡。

### 输出断语

```text
夫妻宫中伤官与正官同宫，运年又填实夫妻宫，女命婚姻不满会明显增强，容易嫌弃伴侣、想离婚或有换人的念头。若同时见比劫、外缘星或现实反馈，再进一步看婚外情风险。
```

### 不适用条件

1. 伤官与官星不在夫妻宫，也没有被运年引动。
2. 官星强而有印化，伤官被制，婚姻不满被缓解。
3. 夫妻宫有稳定六合、天合地合并能化解伤官见官。
4. 不可单凭伤官正官同宫就断必然出轨或必然离婚。

### 模块边界

| 对比模块 | 区别 | 不能套用的地方 |
|---|---|---|
| 财运 | 伤官可泄身生财，在财运中未必坏 | 不能因为伤官为财源就忽略婚姻模块伤官见官 |
| 健康 | 健康看五行受伤和刑冲穿害 | 不可直接把婚姻不满套成健康问题 |
| 外缘借贷 | 外缘涉及比劫、财星、现实反馈 | 不可单凭婚姻不满断外缘骗财 |

### 冲突处理

若同一流年火为命局喜神，需专题隔离：财运维度可能因火调候而改善，婚姻维度仍因夫妻宫被填实、伤官见官而动荡。不能把火为喜神直接等同婚姻吉。

## RULE-20260630-04 忌神流年劫财临财库的外人耗财风险

- 规则名称：忌神流年劫财临财星财库，易有借贷、夺财、外人耗财风险
- 分类：事业财运/其他分析
- 规则层级：专题应用规则
- 所属基础原理：劫财、财星、财库、用神忌神、大运流年
- 确认状态：已确认
- 置信度：中

### 适用条件

1. 流年天干为劫财，或大运流年明显引动劫财。
2. 流年地支临财星、财库、财路，或与原局财库发生刑冲合害。
3. 该流年地支在当前命局中为忌神，或会坏财星、坏财库、增强忌神。
4. 现实事项涉及借钱、合伙、外缘对象、朋友同辈、竞争者、回款、债务。
5. 若要断“骗钱”，必须有现实反馈或更多命盘证据；规则本身只断夺财、耗财、回款不顺风险。

### 判断逻辑

劫财代表同辈、竞争、争夺、分财、夺财。若劫财流年同时临财星或财库，且流年地支为忌神或引动财库受刑，财务上容易出现被外人消耗、借贷不顺、钱被占用、合作分财、回款困难等。若此人同时表现为外缘对象，则感情外缘与钱财风险容易叠加。

### 输出断语

```text
此年劫财临财星财库，又是忌神流年，财务上要防外人、朋友、同辈或外缘对象耗财、借钱不还、合作分财和回款不顺。若现实中已经有人借钱或承诺还钱，应谨慎处理，但不能仅凭命理直接断法律结果。
```

### 不适用条件

1. 劫财为喜用，并且生助用神、帮助命主得财。
2. 流年地支没有触发财星、财库、财路。
3. 原局有强食伤、官杀或合化结构能制劫护财。
4. 没有现实借贷、合伙或外人钱财往来时，不宜强断具体事件。

### 模块边界

| 对比模块 | 区别 | 不能套用的地方 |
|---|---|---|
| 婚姻 | 劫财可作外缘竞争者辅助信息 | 不可单凭劫财断出轨 |
| 法律 | 命理只能提示回款和纠纷风险 | 不可断借条无效或诉讼必败 |
| 财运 | 本规则看借贷、夺财、回款风险 | 不等于全年没有赚钱机会 |

### 冲突处理

若劫财流年同时有喜用神通关或形成合作得财，应区分“朋友/同辈带来机会”和“分财、回款风险”。若现实反馈显示对方确已借钱或拖欠，本规则置信度上调；若无现实钱财往来，只作风险提醒。
