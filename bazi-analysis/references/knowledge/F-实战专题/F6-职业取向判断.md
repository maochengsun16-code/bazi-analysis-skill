# 职业取向判断

## 一、职业判断总纲：先看食伤

看一个人的职业，**先看食伤**。食伤是想法，只有想法到了才会去做。

---

## 二、打工命

### 2.1 判断标准

以下特征指向打工命：
- **印旺**
- **用神无力**
- **食神被克坏**
- **没有明显用神**

以上情况满足越多，打工倾向越强。

---

## 三、老板命与做生意

### 3.1 食神制杀

食伤透在月上制七杀，是做生意的，带团队。这种八字的人能管人、能带团队，靠自己的技术和想法吃饭。

### 3.2 外面的食伤生到家里的财

外面的食伤生到我家里来的财，一定是个做生意的老板。这是典型的做生意八字，靠外部资源转化为自己的财富。

### 3.3 制官杀到家里

制到我家里官杀的，也是老板。

### 3.4 合伙做生意

外面的伤官制到我家里的官的，当老板的，合伙做生意。

### 3.5 食伤生年上财

食伤生年上的财，又把财合进家里来的，也是一起步的八字。格局大小：
- 年上的财 -> 亿级
- 月上的财 -> 千万级

---

## 四、体制内与掌权

### 4.1 杀印相生

外面的七杀生我家里的印，被我拿到了，是**吃公家饭**的标志。杀印相生是体制内的标志组合。

### 4.2 财生杀 + 杀生印

如果还有财来生杀，代表**掌权**。组合链条：财生杀 -> 杀生印，不仅吃公家饭，还是掌权的。

---

## 五、职业判断要点综合

| 组合 | 职业取向 |
|------|----------|
| 印旺、用神无力、食神被克坏 | 打工命 |
| 食神制杀（食伤透月上制七杀） | 做生意、带团队 |
| 外面的食伤生到家里的财 | 做生意的老板 |
| 杀印相生（外七杀生家里印） | 体制内 |
| 财生杀 + 杀生印 | 体制内掌权 |
| 制官杀到家里 | 老板 |
| 外面伤官制到家里官 | 合伙做生意 |
| 食伤生年上财且合进家里 | 起家八字（亿级/千万级） |

> 核心逻辑：第一步先用食伤定位"有没有想法"，再根据印、杀、财的组合判断具体方向。
