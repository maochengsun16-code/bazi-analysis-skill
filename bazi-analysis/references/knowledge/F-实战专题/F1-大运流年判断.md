# 大运流年判断

## 一、大运看法总纲

### 1.1 核心原则：重地支

- 大运以地支为主，整个十年都看地支，不**分前五年看天干、后五年看地支**。
- 判断一个人从哪一步运开始起家，直接看地支是否为喜用即可。
- 大运流年必须按专题分别判断。事业财运、婚姻感情、健康灾祸、父母子女可以在同一步运中呈现不同吉凶，不可用某一专题的不利覆盖另一专题的有利。

### 1.2 大运与月令作用最大

大运地支作用于原局时，力量最大的是月令。

- 大运地支来了以后，首先要看它和月令产生什么作用（冲、合、刑、害等）。
- 由此可定此运在当前专题中的大体吉凶；如果切换专题，必须重新回到该专题的星、宫、做功链和适用条件。
- 月令可改变大运属性：某专题本以火为用，但月令是辰土（该专题最大忌神）时，巳火午火反去生辰土，用神生忌神，该专题好事变坏事。

### 1.3 专题隔离原则

分析某一专题时，只使用与该专题直接相关的十神、宫位、做功链和运年作用。

| 专题 | 主要判断对象 | 不可混入 |
|------|--------------|----------|
| 事业财运 | 财星、财库、财路、制财官做功链、工作平台、经营投资、现金流、债务、回款 | 婚姻稳定性、夫妻宫动荡、父母健康、子女、学业 |
| 婚姻感情 | 夫妻宫、夫妻星、星宫关系、比劫竞争、伤官见官、外缘 | 财运层次、财富金额、父母健康、子女健康 |
| 健康灾祸 | 受伤五行、身体取象、刑冲穿害、官杀灾祸结构 | 财运好坏、婚姻是否离合 |
| 父母子女 | 六亲星、六亲宫、六亲星状态、对应大运流年引动 | 命主本人财运、婚姻吉凶 |

同一大运或流年如果出现跨专题信息，必须并列输出：“事业财运维度为吉，但婚姻感情维度有动荡”。不能因为婚姻、健康、父母等专题有不利，就把事业财运的好大运改断为坏运。

### 1.4 辰戌丑未四种墓库运的特殊看法

走到辰、戌、丑、未四种地支时，要看天干透出什么，以此决定运势属性。

| 地支 | 天干透出 | 属性 | 说明 |
|------|----------|------|------|
| 戌 | 透丙 | 巳火属性 | 作火库看 |
| 丑 | 透己 | 直接当土看 | 湿土变土 |
| 辰 | 透壬 | 水库透水 | 水旺之地 |
| 未 | 透癸 | 燥土透水 | 虚透无力 |

判断准则：看透出的天干是实透还是虚透，以此定吉凶。

## 二、大运流年分专题核心口诀

### 2.1 四句核心口诀

| 组合 | 结果 |
|------|------|
| 忌神 **生** 用神 | 应吉 |
| 用神 **生** 忌神 | 应凶 |
| 用神 **交战** | 应凶 |
| 忌神 **交战** | 应吉 |

这四句话贯穿大运流年判断的始终，实战中反复使用。

### 2.2 忌神虚透与用神虚透

| 类型 | 判断标准 | 结果 |
|------|----------|------|
| 忌神虚透 | 天干透出忌神，在地支无根、无力 | 应吉 |
| 用神虚透 | 天干透出用神，在地支无根、无力 | 应凶 |

**虚实判断：**
- 坐实 = 实透：天干在地支有强根，力量强。
- 无根无力 = 虚透：天干在地支无强根，力量弱。
- 举例：庚金透出但坐寅木，庚金力量弱为虚透，此时忌神透出反吉。

### 2.3 大运作用于命局的三种路径

**路径一：生助当前专题忌神 -> 该专题不利**

大运地支直接生旺了原局的忌神。

**路径二：合出当前专题忌神 -> 该专题不利**

大运地支与原局合化，合出的元素是忌神。
- 例：月令是丑，走巳火运，巳丑合金把忌神合旺。
- 这就是对应专题看似走到用神却不应吉的原因。

**路径三：用神生忌神 -> 好事变坏事**

大运地支本身是用神，但它去生了原局的忌神。
- 例：月令辰是最大忌神，走了火运用神，但火去生辰土，用神生忌神。
- 这也是对应专题看似走到用神却不应吉的原因。

## 三、流年看法

### 3.1 流年先作用大运

流年先和大运发生作用，再作用于原局。
- 流年地支与大运地支发生冲、合、刑、害等关系。
- 再以此影响原局的用神和忌神。

### 3.2 当前专题好大运中的流年

在某一专题的好大运里，并非所有年份都好，但该专题的差年份通常只是减量或波动，不直接翻转为该专题的大凶。其他专题若有不利，另列专题说明。

| 类型 | 特征 |
|------|------|
| 好年份 | 流年助旺大运的用神；流年地支与大运地支相生相合，且合出元素为喜用 |
| 差年份 | 流年冲克大运地支（忌神制用神），如庚寅运中申年冲寅，该年不好 |

### 3.3 当前专题坏大运中的流年

在某一专题的坏大运里，某些流年可能触发该专题的大问题。其他专题若有用神到位或做功成立，仍可另断为相对有利。

- 流年与大运一起将忌神力量生旺，且没有化解。
- 流年合走用神或用神被合走。
- 例：命主喜丁火，入壬辰大运（水旺），到亥子丑水年，忌神旺而无化，最倒霉。

## 四、特殊技法

### 4.1 真神得用

原局有这个用神，大运再次出现相同的用神（同种五行且地支相同），称为真神得用。

| 情况 | 效果 |
|------|------|
| 地支完全一样（原局有壬水，大运见亥水） | 效果最好 |
| 同一五行但不同地支（原局有壬水，大运见子水） | 效果次之 |

真神得用往往是对应专题的高峰期。若真神服务于财运，则主要断事业财运高峰；若真神服务于婚姻、六亲或健康专题，则回到对应专题判断，不作全局混断。

### 4.2 化忌为用（忌神生用神应吉）

即使流年来了忌神，但如果忌神生的结果是用神，反而应吉。
- 例：命主忌水，但在庚寅运中，亥水年反而好——亥水生了寅木用神，忌神生用神应吉。
- 但如果换到壬辰大运（没有寅卯帮化），亥水年变成单纯的忌神旺，反而倒霉。

### 4.3 六亲与财运分开看

- 大运流年看财富和权力上的吉凶，与六亲的吉凶无关。
- 走财运好大运也可能有六亲应凶。
- 六亲吉凶有独立的看法体系，不可混淆。

### 4.4 运年坏原局用神

大运流年不能只看地支喜忌。若原局或当前大运已有明确用神，流年天干把这个用神冲走、合走或克坏，即使流年地支带有喜神，也要先按“用神被坏”处理。

| 情况 | 判断 |
|------|------|
| 财星为当前财运用神，被流年印星冲坏 | 赚钱机会受损，投资、房产、实体、固定资产类事项容易破财 |
| 财星为当前财运用神，被流年天干合走 | 财被别人、项目或外部条件牵走，投资扩张不利 |
| 流年地支为喜，但天干坏关键用神 | 不能按纯吉断，最多看成有局部补救 |

示例：丙火巳月比劫旺，庚子运以庚金财星、子水官杀为喜。甲辰年甲庚冲，乙巳年乙庚合，均先看大运财星被坏；辰虽为湿土，不能覆盖财用神受损。

### 4.5 财运与三刑的模块边界

流年补成三刑时，健康、灾祸、六亲模块要提高权重；但财运模块仍先看财星、财库、用神、做功链和大运作用。若申酉为财星喜用，不能只因申年补寅巳申三刑就否定财年，除非三刑同时伤到财星、财库、财路或关键用神。

## 五、总结要诀

1. 大运看地支，不分前五年后五年。
2. 大运与月令的作用关系最大。
3. 辰戌丑未运重点看透出天干的虚实。
4. 四句口诀贯穿始终：忌神生用神应吉，用神生忌神应凶，用神交战应凶，忌神交战应吉。
5. 流年先和大运作用，再看原局。
6. 好大运、坏大运必须带专题前缀；财运好大运中的财运差年份只是减量，婚姻或健康不利需另列专题。
7. 化忌为用：忌神生用神应吉。
8. 真神得用往往是对应专题的高峰，不作全局混断。
9. 运年冲合坏关键用神时，即使地支有喜，也要先处理用神受损。
