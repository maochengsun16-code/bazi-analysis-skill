# 婚姻感情判断

## 一、婚姻判断要讲清的几件事

看婚姻需要讲清以下方面：

1. 结婚的时间
2. 是否有二婚，以及二婚的时间
3. 一生几段婚姻
4. 另一半是否老实，是否会出轨
5. 另一半是否帮扶自己，还是会拖累自己

---

## 二、夫妻宫与夫妻星

### 2.1 定位规则

- **夫妻宫**：日支，是判断婚姻的核心位置。
- **夫妻星候选**：财星和官杀星都可以代表另一半，无论男女；性别只决定优先级，不决定唯一性。
- **性别优先级**：男命先看财星，财星不成立再看官杀；女命先看官杀，官杀不成立再看财星。
- **成立条件**：夫妻星必须与日柱发生作用，才可定位为长期配偶。日柱包含日干和日支，既要看天干与日干的合冲，也要看地支与日支夫妻宫的作用。
- **地支作用类型**：合、六合、半合、冲、害（穿）、会、半会、破、生，都可以作为定位线索；六合力量最强、稳定性最好，半合和半会可以定位但稳定性不足，冲、害、破、生多主关系不稳或关系质量差。
- **天干作用类型**：天干五合可以直接定夫妻星，天干相冲也可作为作用线索。天干有作用后，还要回看地支关系来判断能不能一婚到头。
- **无作用不定位**：如果财星或官杀星出现在年柱、月柱、时柱，但完全不与日柱发生合、六合、半合、冲、害（穿）、会、半会、破、生等作用，则不能定位为夫妻星；最多按桃花、短缘或露水情缘看，不按长期配偶看。
- **星宫同位**：夫妻星坐在日支，或日支本气就是财星/官杀星，可以直接定位为夫妻星。日支藏干中有财星/官杀星也可参考，但优先级低于明现、本气和明确作用关系。
- **财官都不成立时**：如果原局无财官，或财官虽有但都不与日柱发生作用，才降级看其他十神。比劫、食伤、印只要与日柱发生作用，尤其六合到夫妻宫，也可以低优先级定为夫妻星。
- **定位不等于稳定**：定到夫妻星，只说明有长期关系、领证关系或长期同居相处的可能，不代表一婚到头。婚姻稳定性必须另看星宫关系质量和喜忌。

### 2.2 夫妻星定位优先级

1. 男命：先找与日柱发生作用的财星；女命：先找与日柱发生作用的官杀星。
2. 第一优先级不成立时，男命再找与日柱发生作用的官杀星；女命再找与日柱发生作用的财星。
3. 检查星宫同位：日支本气为财/官杀时，可直接定；日支藏干中的财/官杀可作为较低优先级证据。
4. 财官都不成立时，再找与日柱发生作用的其他十神，尤其六合到夫妻宫者。
5. 多个候选同时成立时，按优先级、作用强度、是否入夫妻宫、是否本气明现来排序，不直接按数量断婚姻多段。

### 2.3 天干合但地支绝

天干五合虽然是夫妻星，但结婚后能不能一婚到头，要看地支的关系。天干合了但地支是绝的关系（如子巳相绝），就会结婚也会离婚。

## 三、星宫关系决定婚姻稳定性

### 3.1 稳定性的唯一条件

婚姻要稳定，只有一种情况最好：**合进来的**。

| 关系 | 结果 |
|------|------|
| 天合地合 | 最稳定 |
| 地支六合 | 稳定 |
| 半合、半会 | 可以定位关系，但不稳定 |
| 冲 | 不稳定，主离婚 |
| 害（穿） | 不稳定，主离婚 |
| 破 | 关系破损，容易闹离婚 |
| 生入夫妻宫 | 不稳定，主离婚 |
| 刑入夫妻宫 | 不稳定，主离婚 |
| 绝 | 极不稳定，多段婚姻 |

### 3.2 星宫交战

星宫交战，即夫妻星和夫妻宫相冲，就是二婚命。相穿（害）也是星宫交战的一种，同样会离婚。

### 3.3 二婚配偶的帮扶判断

判断离婚之后，还要看另一半对自己有无帮助，需取喜用神。

- **喜用的夫妻星合到夫妻宫**：能帮到自己，起家靠配偶。
- **忌神的夫妻星合到夫妻宫**：是拖累，二婚反而更差。
- **夫妻宫为忌且夫妻星也为忌**：婚姻受拖累。

## 四、配偶出轨判断

### 4.1 夫妻宫和夫妻星都有比劫

- 夫妻宫里藏比劫，不一定出轨。
- 但如果夫妻星里也藏比劫（夫妻星和比劫同柱或同宫），另一半必出轨。
- 这种人找谁都没用，随便跟谁过另一半都会背叛他。

### 4.2 暗合

- 夫妻星被暗合（如寅和丑暗合），另一半稳定性差。
- 如果是比劫暗合到夫妻宫，百分之一百出轨。
- 比劫六合、三合、半合到夫妻宫，也是另一半百分百出轨。

### 4.3 劫财合夫妻宫

- 主离婚。
- 如果劫财为忌，另一半可能直接卷钱跑路。

> 注意：出轨不等于一定会离婚。有些情况出轨了也不一定离。

### 4.4 从自己八字看配偶背叛

不需要看另一半的八字，从自己八字就能看出来：

- 夫妻星和夫妻宫与比劫同宫同柱 -> 另一半会出轨。
- 夫妻星被比劫刑冲克害到 -> 另一半会出轨。
- 夫妻星被比劫合到 -> 另一半会出轨。
- 夫妻宫有比劫、夫妻星又被比劫合到 -> 出轨。

---

## 五、结婚时间判断

结婚时间的核心：在适婚年龄段，夫妻星到位（或合到夫妻宫的地支出现）的年份。

- 如果夫妻宫出现六合，结婚时间最容易看。例如夫妻宫子水，有子丑合，牛年（丑来的时候）就容易结婚。
- 但这类八字有个特点：结婚时间好看，但一到合夫妻宫的那个地支来的年份就犯桃花。比如丑年结了婚，下一个丑年又犯桃花，至少精神出轨。

## 六、离婚时间判断

### 6.1 核心口诀

**男怕劫财，女怕伤官。**

- 男命：劫财合到夫妻宫的必离婚。劫财合几次就离几次。
- 女命：伤官同时坏掉夫妻宫和夫妻星的年份，就是离婚的年份。伤官冲了夫妻宫，又合了夫妻星，这一年就要离婚。刑冲合害生都算发生关系，克也算。

### 6.2 多宫多星

夫妻宫伏吟（夫妻宫的字在月上又出现一次），再加多宫多星，是二婚的八字。

### 6.3 多段婚姻常见因素

| 因素 | 说明 |
|------|------|
| 男命正偏财混杂 | 婚姻竞争、多段关系 |
| 女命官杀混杂 | 婚恋不稳定 |
| 配偶宫被冲、穿、刑、害 | 婚姻根基不稳 |
| 配偶星多而杂 | 感情选择多 |
| 配偶星入墓又被多次打开 | 婚姻反复 |
| 日支与外界多合 | 易有外缘 |

> 不能只凭一个点断二婚，要多个信息叠加。

## 七、被拖累的婚姻

夫妻宫有三刑（寅巳申三刑或丑未戌三刑）进了夫妻宫的，婚姻不好，结了婚很痛苦。

三刑克妻或克夫，无论哪里刑了夫妻宫都克另一半。

### 7.1 六害（穿）在婚姻模块中的用法

地支穿害用于婚姻时，必须回到夫妻宫、夫妻星和星宫关系，不直接套用健康或财运断法。

| 情况 | 婚姻含义 | 适用条件 |
|------|----------|----------|
| 穿到日支夫妻宫 | 婚姻根基受伤，关系不舒服，分居、争吵、离婚风险增加 | 日支参与六害，且无稳定合化解 |
| 夫妻星所在宫穿夫妻宫 | 星宫交战，配偶关系不稳，多主婚姻反复 | 财星/官杀所在支与日支穿害 |
| 夫妻星与比劫同宫，再穿夫妻宫 | 婚姻竞争、外缘、配偶不稳定倾向增强 | 夫妻星已按定位规则成立，且同宫见比劫 |
| 流年大运再引动原局穿 | 对应年份婚姻压力明显 | 原局已有穿，运年再见穿害一方 |

例如寅巳穿用于婚姻时，若日支为寅，巳为夫妻星所在宫，巳穿寅就是夫妻星所在宫穿夫妻宫；巳年再来，婚姻冲突和离婚风险会增强。此规则只用于婚姻模块，不自动推出破财或身体伤灾。

### 7.2 多个夫妻星所在宫穿夫妻宫

男命财星所在的多个地支都与日支夫妻宫发生穿害，且财星与比劫同宫同柱时，婚姻不稳和多段关系倾向会明显增强。

适用条件：

1. 已按夫妻星定位规则确认财星可作为配偶星。
2. 多个夫妻星所在宫与日支夫妻宫发生穿害、冲刑或破。
3. 夫妻星同宫见比劫，形成婚姻竞争、外缘或伴侣不稳定信息。
4. 大运流年再见穿害一方时，离婚、分居、争吵或关系危机应期增强。

可用断语：

```text
夫妻星所在宫多处穿到夫妻宫，且夫妻星与比劫同宫，婚姻根基不稳，容易有多段重要关系；若流年再引动穿害，婚姻冲突和离婚风险会明显增强。
```

边界：配偶是否曾有婚史、年龄是否偏大、是否存在第三对象，属于进一步取象；证据不足时只作低置信度提示，不直接升为稳定规则。

## 八、特殊八字（不以财官看配偶）

有些特殊八字不以财官看配偶。只有在财星和官杀星都不存在，或虽存在但都不与日柱发生作用时，才降级看其他十神。例如女命八字没有官杀星，财星也没有和日柱发生关系，这时若劫财六合到夫妻宫，可把劫财看作她的另一半。

此类八字的问题：劫财合到夫妻宫，要离婚，百分百离。而且另一半不是东西。如果找一个离过婚的男人反而能化解，婚姻还能稳定一点。

## 九、配偶特征看配偶宫十神

日支配偶宫中的藏干十神，可看配偶气质和婚姻状态。

| 配偶宫十神 | 特征 |
|-----------|------|
| 坐财（男命） | 对婚姻现实感强 |
| 坐官（女命） | 婚姻规矩感强 |
| 坐伤官（女命） | 容易挑剔伴侣 |
| 坐比劫 | 婚姻中竞争、自我、同辈因素重 |

### 9.1 配偶星五行、喜忌与配偶状态

确定配偶星成立后，可以进一步看配偶星五行、旺衰和喜忌取象。

| 条件 | 配偶取象 | 适用边界 |
|------|----------|----------|
| 配偶星五行属火，且在原局旺 | 对象性情偏急、偏烈，脾气不太好，表达直接，情绪来得快 | 必须先确认该火为已成立的配偶星；火弱或受制严重时降级 |
| 配偶星为忌 | 对方能力一般般，或能力难以真正帮到命主 | 不以配偶星强弱为唯一标准；配偶星为喜用时不按此断 |
| 配偶星旺而为忌 | 配偶存在感强、较强势，对命主形成压力 | 若同时合入日主或夫妻宫稳定，先以对方强势、命主受配偶影响较重取象，不上升为婚姻破裂结论 |

适用条件：

1. 已按夫妻星定位规则确认财星或官杀可作为长期配偶星。
2. 配偶星与日柱发生作用，或坐入夫妻宫。
3. “脾气不太好”主要由配偶星五行属火且旺推出。
4. “能力一般般”由配偶星为忌推出，不以旺弱单独裁决。

不适用条件：

1. 配偶星未成立时，不用此规则断正式配偶。
2. 配偶星为喜用时，不按“能力一般般”断。
3. 配偶星属火但原局火弱、被强力制化或无力作用日柱时，脾气烈的判断降级。

## 十、补充规则

- 男命八字劫财合到夫妻宫的必离婚，也代表会找婚外对象。
- 财官优先级都不成立时，任何十神只要六合到夫妻宫来，都可以看成另一半——不限于财官。
- 比劫六合到夫妻宫的特殊组合，狗年（戌）会结婚，但同时狗年也容易离婚。
