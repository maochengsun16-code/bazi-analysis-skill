# 子女与父母判断

---

## 第一部分：子女判断

### 一、子女星定位

| 命主 | 子女星规则 | 儿子 | 女儿 |
|------|-----------|------|------|
| 男命 | 正官看女儿，七杀看儿子 | 七杀 | 正官 |
| 女命 | 伤官看儿子，食神看女儿 | 伤官 | 食神 |
| 男命（无官杀） | 伤官看女儿，食神看儿子 | 食神 | 伤官 |
| 女命（无食伤） | 七杀看女儿，正官看儿子 | 正官 | 七杀 |

**优先顺序**：月上的子女星 > 坐下 > 年上。

> 注意：伤官被穿可能会导致变性——本来该生儿子的变成女儿，或反过来。

### 二、子女宫（时柱）

时柱是子女宫和晚年宫，与以下方面相关：
- 子女
- 晚辈
- 晚年
- 事业结果

### 三、子女能力与帮扶

| 条件 | 结果 |
|------|------|
| 子女星为喜用神 + 天地一气有力 | 孩子有能力，能跨越父母当前阶层 |
| 子女星为忌 + 天地一气 | 孩子有能力，但关系不咋地（能力与关系分开论） |
| 子女星和自己是天干五合 | 即使为忌，表面关系尚可；地支若相战则只是表面好 |

### 四、克子女的结构

#### 女命克子女

| 结构 | 影响 |
|------|------|
| 印旺（偏印在月令），食伤一柱是偏印，且原局无食伤 | 最严重，可能无子 |
| 印旺（正印或偏印） | 容易掉孩子 |
| 正印在月上很旺，食伤透偏印或正印 | 克子女 |
| 食伤被克泄太过，无救应 | 可能白发人送黑发人 |

#### 穿的作用（子女身体健康）

| 情况 | 结果 |
|------|------|
| 食伤星被穿 | 子女身体不好 |
| 穿伤官 | 对儿子不利 |
| 穿食神 | 对女儿不利 |
| 被穿的子女 | 体质差，或携带终身性疾病 |

子女模块使用六害（穿）时，必须先确认被穿的是子女星、子女宫，或子女星对应的藏干。若穿害只发生在夫妻宫或财星，不直接断子女身体。

#### 男命克子女

食伤旺的男命克子女（食伤克官杀，官杀是男命的子女）。体现为：子女不如自己优秀。

### 五、子女身体健康判断

判断六亲健康**不看喜忌，只看状态**。

状态指这个十神在原局有没有被刑冲克害泄太过。

| 状态 | 判断 |
|------|------|
| 有力、得月令、天地一气 | 身体好 |
| 被刑冲克害泄太过 | 身体有问题 |

#### 伤者入墓

某六亲在原局中受伤，走到对应的墓库大运时，就容易出问题。

### 六、生育应期

| 命主 | 应期判断 |
|------|----------|
| 女命 | 生育年龄段，流年有食伤出现且食伤与子女宫发生关系（刑冲合害生） |
| 男命 | 看官杀星和子女宫同时被引动 |
| 通用 | 流年填实了时柱（子女宫）则更直接 |

#### 男命子女星不显

男命以官杀为子女星。原局官杀不显，不可直接断无子，应优先断为子女来得晚、子女缘分弱、与子女聚少离多，或需等大运流年官杀到位、子女宫被引动时才应子女事。

适用条件：

1. 男命原局官杀不透不显，或只藏而不成作用。
2. 子女宫未形成稳定子女星关系。
3. 大运流年尚未明显引动官杀与子女宫。

边界：是否无子必须结合夫妻双方八字、实际年龄、生育反馈和大运流年，不可仅凭原局无官杀下结论。

### 七、子女个数与性别（夫妻合看）

谁的八字层次更高，就会先生到谁命里的孩子。

- 男命层次更高 -> 先生男命八字中的孩子。
- 女命层次更高 -> 先生女命八字中的孩子。
- 离婚同理：前面孩子的性别生完了，会接着生现任配偶八字中的孩子。

### 八、看子女信息的注意事项

- 看父母信息最准确的是看长子的八字（家里的男性老大），长子与父母关系最紧密。
- 长子夭折则以次子来看。
- 现代少子化，重点抓**月上**的那个孩子——月上的一般先出生。

## 第二部分：父母判断

### 一、父母星定位

#### 男命

| 六亲 | 主要看 | 无则用 |
|------|--------|--------|
| 母亲 | 正印 | 偏印 |
| 父亲 | 偏财 | 正财 |

#### 女命

| 六亲 | 主要看 | 无则用 |
|------|--------|--------|
| 母亲 | 偏印 | 正印 |
| 父亲 | 正财 | 偏财 |

#### 正偏混杂区分

优先看谁在父母宫（**月柱**）。谁在月上谁就是对应的父母。如果年月都不在，再看日时。年柱上的也算父母。

### 二、父母身体健康

判断身体健康**不分喜忌，只看状态**。

#### 状态判断

| 状态 | 身体情况 |
|------|----------|
| 有力、得月令、天地一气 | 身体好 |
| 被刑冲克害泄太过 | 身体有问题 |

#### 用穿定疾病

被穿的那个字代表的身体部位出问题。

| 被穿的字 | 身体部位 |
|----------|----------|
| 子水被穿 | 泌尿系统、肾脏、膀胱 |
| 午火被穿 | 心脏、血液、心脑血管 |

### 三、父母关系与离婚判断

| 情况 | 结果 |
|------|------|
| 正偏财混杂，且印星同时和正财、偏财都发生关系 | 母亲不止一个男人，父母要离婚 |
| 正偏印混杂，且财星同时和正印、偏印都发生关系 | 父亲会找后妈，父母一婚不到头 |

### 四、父母谁先走

| 组合 | 结果 |
|------|------|
| 喜财不见财（财星为父） | 父亲能力一般；父亲容易走在母亲前面；或父母容易离婚 |
| 喜印不见印（印星为母） | 母亲容易走到父亲前面去 |

### 五、父母能力与家境判断

| 情况 | 家境 |
|------|------|
| 父母为喜用 | 出生不差 |
| 父母为忌、没有作用 | 普通家庭 |
| 月令是比劫当令 | 一般家里不好，贫苦家庭出身，靠自己做出东西来 |
| 月上父母宫做了比劫 | 暗克父 |
| 月令是财星很旺 | 暗克母（财克印） |

### 六、劫财为忌与六亲习性

劫财为忌（一定是劫财且一定为忌），和某个六亲同宫同柱的，此六亲喜欢吃喝嫖赌抽。如果是食伤星和劫财同宫同柱，孩子也有这个问题。

### 七、克六亲的多种形式

比劫旺克妻克夫、财旺克母只是基础。还有更多形式：
- **三刑**：也是克父母——刑了什么十神就克哪个六亲。
- **穿**：也是克——穿了哪个六亲，那个六亲身体就不好。

父母模块使用六害（穿）时，必须先定位父母星和父母宫。男命母亲优先看正印，无正印或正印不成立时参考偏印；父亲优先看偏财。若印星或财星被穿害，才按父母身体或父母缘分受伤处理。六亲健康只看状态，不因父母星为喜用就直接断身体好。

### 八、寅巳穿印与母亲健康

男命看母亲优先取正印，无正印或正印不成立时参考偏印。若原局寅巳穿，且寅中甲木或相关印星被巳中庚金所伤，可判断母亲星状态受损，母亲身体易有长期不佳或反复就医倾向。

适用条件：

1. 印星已经定位为母亲星。
2. 印星所在支或印星藏干参与寅巳穿。
3. 流年或大运再见寅、巳时，穿害被引动，应期增强。

可用断语：

```text
母亲星在原局被寅巳穿伤，说明母亲身体状态长期不算稳；逢寅年、巳年或运年再引动寅巳穿时，更容易应医院、手术、筋骨四肢伤病或旧病反复。
```

边界：父母健康只看六亲星状态，不与命主财运吉凶混用；健康应象必须用倾向表达。
