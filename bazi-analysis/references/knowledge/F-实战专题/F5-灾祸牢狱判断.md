# 灾祸牢狱判断

## 一、牢狱判断总纲

什么样的八字会坐牢？主要从四种情况入手分析。

---

## 二、官杀旺无制化

### 2.1 判断标准

官杀当令且很旺，官杀混杂（地支有正官有七杀，或天干同时有正官七杀），且官杀为忌的，容易有牢狱。

### 2.2 最怕走官杀运

这种八字最怕走**官杀大运**和**官杀流年**。一旦官杀力量过旺且无制，就容易进去。

### 2.3 官杀为用不论混杂

如果官杀为用，不论混杂——官杀为用代表一个人干两份事业，反而是好事。

### 2.4 有印化解的情况

有印的话能化解一部分，但要看画干净没有，还要看大运流年会不会把印搞坏——如果大运来把印制掉，也是出问题的时候。

### 2.5 官杀的双刃剑属性

富贵靠官杀，倒霉也靠官杀。上一秒还是企业家、实权干部，下一秒就成阶下囚。

---

## 三、三刑与牢狱

### 3.1 丑未戌三刑（官杀三刑）

丑未戌三刑中的元素如果是官杀，容易坐牢。

### 3.2 寅巳申三刑

寅巳申三刑中如果某个十神是官杀，也容易坐牢。

### 3.3 刑期长短

| 情况 | 结果 |
|------|------|
| 原局有三刑，大运再走三刑运、流年再到三刑年 | 进去时间较长 |
| 原局没三刑，大运补齐三刑 | 也进去，但时间短（十天半个月） |

---

## 四、比劫犯官

### 4.1 判断标准

比劫去冲官星（例如乙辛冲，劫财冲官），容易有牢狱。

### 4.2 群体性特征

比劫犯官往往不是一个人惹事，是一群人惹事。

### 4.3 扛官的逻辑

- 用比劫扛官时，比劫力量要强于官才能扛住。
- 一旦官杀反制回来，就是牢狱。

---

## 五、伤官见官 vs 伤官治官

### 5.1 伤官治官（官为忌，伤官制官）

| 特征 | 说明 |
|------|------|
| 能拿到官的好处 | 钱财地位 |
| 有风险 | 和比劫扛官一样，力量对比决定成败 |
| 前提 | 伤官力量必须比官强，能按住官 |
| 翻车点 | 一旦官力量变强，伤官按不住，被反制则成牢狱 |
| 典型人生轨迹 | 曾经辉煌过，后来败了 |

### 5.2 伤官见官（官为用，伤官去克官）

| 特征 | 说明 |
|------|------|
| 结果 | 啥好处都没捞到，单纯倒霉 |
| 工作表现 | 工作不顺利，去哪上班领导都不待见 |
| 连带问题 | 容易有口舌是非 |

## 六、取财手段与风险

用**穿**的形式取财（十神穿杀、丑戌相刑冲库等），挣钱手段有问题，做不太合法的业务。常在河边走，哪有不湿鞋，遇到不好的年份就容易进去。

相关参考：详见《F3-财运与财富层次》"偏门取财风险"章节。

## 七、身体健康与伤灾的模块边界

本专题涉及灾祸、牢狱和健康风险。使用地支穿、三刑、自刑时，必须区分健康伤灾与财运、婚姻的用法。

| 地支作用 | 健康/伤灾用法 | 不应混用 |
|----------|---------------|----------|
| 六害（穿） | 看被穿五行、藏干和宫位对应身体部位；原局有穿，运年再引动，应期更明显 | 不直接推出破财或离婚，除非同时作用到财星/夫妻宫 |
| 寅巳穿 | 巳中庚金制寅中甲木，重点看头部、四肢、筋骨、肝胆等木象受伤 | 不因寅巳穿自动断财运差 |
| 寅巳申三刑 | 更适合身体伤灾、六亲关系、官非牢狱风险的判断 | 财运分析中不宜只因流年补三刑就否定财年 |
| 火旺偏枯 | 注意心脑血管、血压、炎症、上火等火象问题 | 健康倾向不能替代医疗判断 |

寅巳穿在健康模块中，以巳中庚金伤寅中甲木为核心。甲木受伤时，可取头部、四肢筋骨、胯骨、筋膜、肝胆等木象；流年再见巳，或原局巳多穿寅，应期增强。若同时火旺燥重，还可提示血压、血脂、心脑血管、炎症上火等火象倾向，但必须用“倾向”“注意”表述，不作医疗结论。

健康类结论必须使用倾向表达，不作确定性医疗结论。

## 八、四种牢狱情况总结

| 序号 | 类型 | 核心机制 |
|------|------|----------|
| 1 | **官杀旺无制化** | 最怕走官杀大运和流年，官杀混杂且为忌 |
| 2 | **伤官见官** | 官为用时，伤官去克官 |
| 3 | **伤官治官、比劫扛官** | 官为忌时能拿到好处，但被反制时就是牢狱 |
| 4 | **三刑** | 官杀三刑、寅巳申三刑，大运流年凑齐则触发 |

> 核心警句：富贵靠官杀，倒霉也靠官杀。
