# C6-比劫专篇

## 一、比劫定义

比劫是**同我**的力量，即与日主同类的五行。代表自我意识、同辈关系、竞争与合作。

- **比肩**：阴阳同性同类，独立、刚毅。
- **劫财**：阴阳异性同类，竞争、争夺。

### 以甲木日主为例

| 天干 | 关系 | 十神 |
| :--- | :--- | :--- |
| 甲木（阳） | 甲见甲，同阳 | 比肩 |
| 乙木（阴） | 甲见乙，甲阳乙阴 | 劫财 |

## 二、比肩 vs 劫财区别

| 对比维度 | 比肩 | 劫财 |
| :--- | :--- | :--- |
| 关系 | 同性同类 | 异性同类 |
| 性质 | 独立、自尊、刚毅 | 争夺、冲动、胆大 |
| 朋友类型 | 同性朋友、兄弟 | 异性朋友、姐妹 |
| 对财的态度 | 靠自己赚 | 从别人处抢 |
| 竞争方式 | 正面竞争 | 暗斗抢夺 |

## 三、取象类象

| 类象维度 | 比肩 | 劫财 |
| :--- | :--- | :--- |
| **人物** | 兄弟、同辈朋友、合伙人 | 姐妹、损友、竞争者 |
| **身体** | 手脚、体力 | 意志力、胆量 |
| **性格** | 独立、自我、主见 | 冲动、冒险、敢拼 |
| **职业** | 领导者、独立工作者 | 销售、竞争型职业、开拓者 |
| **财富** | 靠自己赚钱 | 合伙赚钱、抢市场 |

## 四、心性特征

### 比肩心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 有朋友帮扶、自己有主见、抗压能力增强 |
| 为忌时 | 竞争多、同行夺财、朋友兄弟拖累、自我太强 |
| 月令比肩 | 有领袖倾向 |

### 劫财心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 有人帮、敢拼、适合竞争/销售/团队开拓 |
| 为忌时 | 小人多、合伙破财、朋友夺财、男命妻缘受损 |
| 月令劫财 | 抗压能力强，适合销售类、竞争类场景 |

## 五、比劫能担财官

比劫的根本作用是增强日主自身力量。

- 身强时比劫能**担财官**：有了足够的力量去承担钱财（财）和权力责任（官）。
- 身弱时比劫可**帮身**：朋友、同辈力量助日主分担压力。
- 地支有比劫、印、库之根，都能够担得住财官。

## 六、男命劫财重的影响

男命以财星为妻，劫财克财。劫财重时：

| 影响领域 | 具体表现 |
| :--- | :--- |
| **婚姻** | 婚姻不稳、夫妻争执 |
| **妻子** | 妻子受压、健康或情绪受影响 |
| **朋友** | 外界朋友/兄弟/合伙影响婚姻 |
| **财产** | 钱财被竞争者夺走、合伙破财 |

## 七、比劫的正面与负面总结

| 十神 | 正面 | 负面 |
| :--- | :--- | :--- |
| **比肩** | 刚毅果断、独立自主、有领袖气质 | 自我太强、不听人言、固执己见 |
| **劫财** | 胆大敢拼、有行动力、适合开拓 | 争夺抢夺、容易得罪人、财来财去 |
