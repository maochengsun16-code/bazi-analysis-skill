# C4-官杀专篇

## 一、官杀定义

官杀是**克我**的力量，即克日主之五行。代表外界约束、压力、规矩、权力、危险。

- **正官**：阴阳异性相克，克得有情，属于规则性约束。
- **七杀**：阴阳同性相克，克得猛烈，属于暴力胁迫。

### 以甲木日主为例

| 天干 | 关系 | 十神 |
| :--- | :--- | :--- |
| 庚金（阳） | 庚克甲，庚阳甲阳 → 同性相克 | 七杀 |
| 辛金（阴） | 辛克甲，辛阴甲阳 → 异性相克 | 正官 |

## 二、正官 vs 七杀区别

| 对比维度 | 正官 | 七杀 |
| :--- | :--- | :--- |
| 克的方式 | 管理约束、定规则（如早九晚五） | 暴力胁迫、不听话就打击 |
| 力度 | 有情、温和 | 猛烈、无情 |
| 代表 | 纪律、法律、正当管理 | 暴力、压迫、危险 |
| 喜用时 | 稳定工作、职位名声 | 权力、魄力、执行力 |
| 忌用时 | 压力束缚、上级压制 | 伤灾、官非、疾病、意外 |

## 三、取象类象

| 类象维度 | 正官 | 七杀 |
| :--- | :--- | :--- |
| **人物** | 上司、领导、官方人物 | 强势领导、暴力倾向人物、竞争者 |
| **事业** | 工作、职位、正当权力 | 杀伐权力、管理岗、武职 |
| **规则** | 法律、纪律、制度 | 强制手段、惩罚机制 |
| **压力** | 合理规则压力 | 极端压迫、危险 |
| **身体** | 慢性病（长期规则压力） | 突发疾病、意外伤灾 |

## 四、心性特征

### 正官心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 有稳定工作、有职位名声、守规矩、能获得正当权力 |
| 为忌时 | 压力大、被规矩束缚、遇上级压制 |

### 七杀心性

| 状态 | 表现 |
| :--- | :--- |
| 无制时 | 压力沉重、易有危险/伤灾/官非/暴力/疾病、性格紧张或攻击性强 |
| 有制时 | 可成权力/魄力/执行力、适合管理/武职/竞争/风险行业 |

## 五、七杀的制化

七杀为凶时，须制化方能为我所用，制化顺序如下：

| 制化方式 | 手段 | 类象 |
| :--- | :--- | :--- |
| **1. 印化杀**（最优） | 用印星化七杀生身 | 把压力转换成学历/权力/资质/贵人/制度资源 |
| **2. 食伤制杀**（中策） | 用食伤去制七杀 | 用能力、技术、谋略对抗压力 |
| **3. 比劫扛杀**（下策） | 用比劫分担压力 | 用团队力量或自身意志硬扛 |

> 制化顺序：印化 > 食伤制 > 比劫扛

## 六、六亲对应

| 命主 | 正官 | 七杀 |
| :--- | :--- | :--- |
| **男命** | 女儿 | 儿子 |
| **女命** | 老公（正夫） | 情人（偏缘）/ 压力型对象 |

> 女命有正官时先以正官看丈夫，无正官则以七杀看丈夫（有顺位）。

## 七、官杀为忌时的注意事项

- 官杀为忌时，不宜让官杀被合会加强（如三合、三会局）。
- 求财行为若引动忌神官杀，容易因钱财问题带来官非。
- 合局、会局需看最终加强的是喜神还是忌神。
