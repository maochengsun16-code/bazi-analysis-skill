# C1-十神总纲

## 一、十神必须以日主为中心

十神不是单独存在的名称。十神一定是以日主为中心，看其他天干地支与日主之间的五行生克关系。日主代表命主自己。

## 二、十神五大类核心公式

| 关系 | 公式 | 十神（阴阳异） | 十神（阴阳同） |
| :--- | :--- | :---: | :---: |
| 生我者 | 生日主 | 正印 | 偏印 |
| 我生者 | 日主生 | 伤官 | 食神 |
| 克我者 | 克日主 | 正官 | 七杀 |
| 我克者 | 日主克 | 正财 | 偏财 |
| 同我者 | 同日主 | 劫财 | 比肩 |

## 三、五善神与五恶神

| 分类 | 十神 |
| :--- | :--- |
| **五善神** | 正印、正财、正官、比肩、食神 |
| **五恶神** | 偏印、偏财、七杀、劫财、伤官 |

> 善神偏于稳定、正统、有情；恶神偏于激烈、偏门、攻击性。但善恶非绝对，需结合命局喜忌具体判断。

## 四、阴阳决定正偏

同一类关系中，根据阴阳决定正与偏：

| 关系 | 阴阳不同（正） | 阴阳相同（偏） |
| :--- | :--- | :--- |
| 生我（印） | 正印 | 偏印 |
| 克我（官杀） | 正官 | 七杀 |
| 我克（财） | 正财 | 偏财 |
| 我生（食伤） | 伤官 | 食神 |
| 同我（比劫） | 劫财 | 比肩 |

一般原则：
- **阴阳异性**关系较有情、温和。
- **阴阳同性**关系较激烈、无情。

## 六、十神速记口诀

- **生我者印**：正印（阴阳异）、偏印（阴阳同）
- **我生者食伤**：伤官（阴阳异）、食神（阴阳同）
- **克我者官杀**：正官（阴阳异）、七杀（阴阳同）
- **我克者财**：正财（阴阳异）、偏财（阴阳同）
- **同我者比劫**：劫财（阴阳异）、比肩（阴阳同）
