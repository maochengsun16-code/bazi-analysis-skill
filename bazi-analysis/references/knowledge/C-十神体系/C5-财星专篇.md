# C5-财星专篇

## 一、财星定义

财星是**我克**的力量，即日主所克之五行。代表钱财、资源、现实利益，以及可被自己支配、管理、获得的东西。

- **正财**：阴阳异性相克，代表稳定、正当、固定的钱财。
- **偏财**：阴阳同性相克，代表非固定、非工资类的钱财。

### 以甲木日主为例

| 天干 | 关系 | 十神 |
| :--- | :--- | :--- |
| 己土（阴） | 甲克己，甲阳己阴 → 异性相克 | 正财 |
| 戊土（阳） | 甲克戊，甲阳戊阳 → 同性相克 | 偏财 |

## 二、正财 vs 偏财区别

| 对比维度 | 正财 | 偏财 |
| :--- | :--- | :--- |
| 性质 | 稳定、正当、固定 | 非固定、非工资 |
| 来源 | 工资、主业收入、固定资产、稳定客户 | 投资、副业、横财、项目财、风险财 |
| 态度 | 脚踏实地的积累 | 敢作敢当的投机 |
| 流动 | 来得稳走得也稳 | 来得快去得也快 |

## 三、取象类象

| 类象维度 | 正财 | 偏财 |
| :--- | :--- | :--- |
| **金钱** | 工资、主业收入、固定资产 | 投资回报、副业收入、横财 |
| **职业** | 稳定职业、实业积累、稳定经营 | 风险投资、商业项目、创业 |
| **男命婚姻** | 正妻（妻子） | 情人、偏缘 |
| **父亲** | 女命：父亲 | 男命：父亲 |
| **其他亲属** | — | 女命：婆婆 |

## 四、心性特征

### 正财心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 财源稳定、婚姻现实稳定、适合稳定经营/主业收入/实业积累 |
| 为忌时 | 为钱辛苦、被现实压力拖住、男命婚姻中因妻财产生压力、与父亲缘分薄或父亲相关压力重 |

### 偏财心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 有投资机会/副业机会、财路灵活、容易接触外部资源 |
| 为忌时 | 投机破财、欲望大、男命感情复杂、钱来快去也快 |

## 五、六亲对应

| 命主 | 正财 | 偏财 |
| :--- | :--- | :--- |
| **男命** | 妻子 | 父亲（无正财时偏财也可代表妻子） |
| **女命** | 父亲 | 婆婆 |

> 男命以财星为妻。正财为妻子，偏财为父亲。女命正偏财均与父亲有关。

## 六、财星虚实

| 概念 | 定义 | 体现 |
| :--- | :--- | :--- |
| **虚透** | 天干有财星，地支无根无库 | 财来财去、不稳定、虚有其表 |
| **天地一气** | 天干地支均为财星 | 财实力强、稳定有力 |
| **实根** | 天干财星在地支有根（通根/库根） | 财能落到实处，能真正拥有 |

## 七、财为喜与财为忌

| 状态 | 表现 |
| :--- | :--- |
| **财为喜** | 财能为命局服务，带来资源、收入、妻缘、父缘、现实成果 |
| **财为忌** | 财消耗日主或破坏用神，容易破财/债务/投资失败；对应的父亲/妻子/欲望也可能成为压力来源 |
