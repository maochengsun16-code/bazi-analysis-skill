## 二、心性特征对比表

### 2.1 正面心性

| 十神 | 正面心性关键词 |
| :--- | :--- |
| **正印** | 仁慈、稳重、有逻辑、有学识、有贵人、包容 |
| **偏印** | 聪明、悟性高、领悟力强、有特殊才能、善于研究 |
| **食神** | 温和、有福气、有口福、情商高、懂得享受生活 |
| **伤官** | 才华横溢、创新突破、表达力强、有艺术天赋 |
| **正官** | 守规矩、负责任、有担当、有领导力、正直 |
| **七杀** | 有魄力、执行力强、杀伐果断、适合管理/武职 |
| **正财** | 务实、节俭、稳定、有经营头脑、脚踏实地 |
| **偏财** | 慷慨、人脉广、财路活、敢于投资、会赚钱 |
| **比肩** | 刚毅果断、独立自主、有主见、有领袖气质 |
| **劫财** | 胆大敢拼、有行动力、善于开拓、抗压能力强 |

### 2.2 负面心性

| 十神 | 负面心性关键词 |
| :--- | :--- |
| **正印** | 依赖性强、思虑多行动少、被规矩/母亲/房子拖住 |
| **偏印** | 孤独、敏感、偏门、不合群、思维另类难被理解 |
| **食神** | 懒散、贪图享乐、泄身太过、缺乏执行力 |
| **伤官** | 叛逆、桀骜不驯、说话带刺、顶撞权威、不服管 |
| **正官** | 压力大、被规则束缚、过于保守、缺乏灵活性 |
| **七杀** | 攻击性强、紧张焦虑、易惹官非、暴力倾向 |
| **正财** | 吝啬、为钱辛苦、过于现实、缺乏冒险精神 |
| **偏财** | 大手大脚、投机心理、欲望过大、钱来快去也快 |
| **比肩** | 自我太强、固执己见、不听人言、竞争心过重 |
| **劫财** | 冲动好斗、容易得罪人、争夺心理、合伙易破财 |

## 六、十神旺衰表现表

### 6.1 十神最旺时

| 十神 | 旺时表现 |
| :--- | :--- |
| **正印** | 学历高、贵人广、平台大、保护多、逻辑强 |
| **偏印** | 特殊才能突出、悟性极高、玄学天赋、孤独学者 |
| **食神** | 多才多艺、口福好、人缘佳、技术精湛、安乐人生 |
| **伤官** | 才华爆发、极具创造力、名扬四方、能破旧立新 |
| **正官** | 职位高、名声好、管理能力强、受人尊敬 |
| **七杀** | 大权在握、权威极强、杀伐决断、能成大事（有制时） |
| **正财** | 财富积累、家大业大、实业稳固、妻贤子孝 |
| **偏财** | 财富暴发、人脉极广、投资神准、交际能力强 |
| **比肩** | 独立自主、龙头老大、核心竞争力强、无人可替 |
| **劫财** | 团队核心、敢为人先、市场掠夺能力强、客户资源广 |

### 6.2 十神最弱时

| 十神 | 弱时表现 |
| :--- | :--- |
| **正印** | 学历低、缺乏贵人、无后台支持、逻辑思维弱 |
| **偏印** | 思路偏激但不通透、学而不精、敏感过度 |
| **食神** | 才华难施展、财路不通、缺乏表达力、口福差 |
| **伤官** | 有想法没产出、怀才不遇、破罐破摔、顶撞惹祸 |
| **正官** | 工作不稳定、无管理权、缺乏自律、名声不佳 |
| **七杀** | 被压制、被欺凌、无还手之力、健康受损（无制时） |
| **正财** | 收入不稳、存不住钱、为钱所困、经济拮据 |
| **偏财** | 无偏财运、投资失利、人脉匮乏、赚不到外快 |
| **比肩** | 缺乏主见、无竞争力、被他人左右、自我意识弱 |
| **劫财** | 缺乏胆量、不敢竞争、团队中无存在感、被人夺财 |
