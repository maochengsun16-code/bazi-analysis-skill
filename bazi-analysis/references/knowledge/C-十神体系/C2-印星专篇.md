# C2-印星专篇

## 一、印星定义

印星是**生我**的力量，即生日主之五行。印星是给日主输送能量、保护日主的资源。

- **正印**：阴阳异性相生，有情、稳定、正统。
- **偏印**（亦称枭神）：阴阳同性相生，偏门、敏感、悟性高。

### 类比：正印如亲妈，偏印如后妈

| 特征 | 正印（亲妈） | 偏印（后妈） |
| :--- | :--- | :--- |
| 生你的方式 | 无私付出，不求回报 | 虽有生助，但要求回报 |
| 态度 | 你帮不帮我无所谓，我只需要爱你 | 我帮了你，你也要回报我，否则不帮 |
| 情感联结 | 天然稳定，无条件 | 有条件交换，时好时坏 |

## 二、取象类象

| 类象维度 | 具体对应 |
| :--- | :--- |
| **人物** | 母亲、长辈、贵人、老师、医生、学者 |
| **学业** | 文凭、学历、正规知识、逻辑能力、考试运 |
| **实物** | 房产、车、单位平台、固定庇护所 |
| **权力** | 保护性权力、制度资源、后台靠山 |
| **其他** | 玄学（偏印）、偏门知识（偏印）、特殊领悟（偏印） |

## 三、心性特征

### 正印心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 贵人相助、学业顺利、有稳定庇护、适合读书/考试/研究/体制/专业路线 |
| 为忌时 | 依赖性强、思虑多行动少、被母亲/房子/学历/单位/规矩拖住 |

### 偏印心性

| 状态 | 表现 |
| :--- | :--- |
| 正面 | 聪明、悟性强、有特殊领悟力、适合偏门学术/玄学/研究 |
| 负面 | 孤独、敏感、偏门、不合群、思维过于另类 |

## 四、身强身弱与印星喜忌

| 条件 | 印星作用 | 喜忌 |
| :--- | :--- | :--- |
| 日主身弱 | 印星生身 | 通常为喜 |
| 日主身强 | 印星再生身易过旺 | 通常为忌 |
| 身衰印弱 | 学习支持不足，考试运、学历运不占优势 | 再逢财星破印更不利 |

> 须结合调候、格局和原局病神具体判断。

## 五、六亲对应

| 命主 | 正印 | 偏印 |
| :--- | :--- | :--- |
| 男命 | 母亲 | 后母/干妈/老师 |
| 女命 | 母亲 | 后母/干妈/老师 |

## 六、补充：偏印与枭神夺食

- 偏印不一定就是坏。
- 当偏印去克食神，尤其食神为用时，称为**枭神夺食**。
- 具体应象轻重和救应条件，参见《C3-食伤专篇》中枭神夺食一节。
