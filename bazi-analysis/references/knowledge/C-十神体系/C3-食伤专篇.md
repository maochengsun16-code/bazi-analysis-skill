# C3-食伤专篇

## 一、食伤定义

食伤是**我生**的力量，即日主所生之五行。代表日主向外释放的能力、才华、输出。

- **食神**：阴阳同性相生，较平和、有福气。
- **伤官**：阴阳异性相生，较激烈、有攻击性。

### 以甲木日主为例

| 天干 | 关系 | 十神 |
| :--- | :--- | :--- |
| 丙火（阳） | 甲生丙，甲阳丙阳 → 同性相生 | 食神 |
| 丁火（阴） | 甲生丁，甲阳丁阴 → 异性相生 | 伤官 |

## 二、食神 vs 伤官区别

| 对比维度 | 食神 | 伤官 |
| :--- | :--- | :--- |
| 福气 | 福星，有口福，爱吃喝玩乐 | 技术狂，重才华输出 |
| 情商 | 高，会换位思考，说话招人喜欢 | 低，说话带刺，容易得罪人 |
| 性格 | 平和、温和、享受生活 | 叛逆、桀骜、不服管 |
| 才华 | 稳定技术输出 | 创新突破 |
| 规则感 | 守规矩 | 破规则 |

## 三、取象类象

| 类象维度 | 食神 | 伤官 |
| :--- | :--- | :--- |
| **口才表达** | 稳定表达、语言温和 | 犀利表达、批判性强 |
| **技术才华** | 稳定技术输出、手艺人 | 技术突破、创新创意 |
| **作品产出** | 常规作品、食禄 | 艺术创作、原创作品 |
| **子嗣** | 女命：女儿 | 女命：儿子 |
| **财路** | 稳定财路、技能吃饭 | 商业创意、风险项目 |

## 四、心性特征

### 食神心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 有稳定财路、才华输出顺畅、人温和、能靠技术/口才/作品/技能吃饭 |
| 为忌时 | 懒散贪享受、泄身太过、对身体和执行力不利 |

### 伤官心性

| 状态 | 表现 |
| :--- | :--- |
| 为喜时 | 才华横溢、能创新、适合靠表达/技术/作品/商业创意发展 |
| 为忌时 | 顶撞权威、不服管理、说话伤人、破坏官星所代表的工作/婚姻/规矩 |

## 五、六亲对应

| 命主 | 食神 | 伤官 |
| :--- | :--- | :--- |
| 男命 | 才艺、食禄（非子女） | 技术才华 |
| 女命 | 女儿 | 儿子 |

> 对于女命，食神伤官统称子女星。

## 六、伤官见官

伤官克正官，称为**伤官见官**。

### 正官受克的类象

- 顶撞上级
- 工作不稳
- 官司是非
- 婚姻不顺
- 不遵守规则
- 学业、事业受影响

### 判断条件

不能机械断凶，需看：
- 伤官是否旺
- 官星是否为用神
- 有无印星制伤
- 有无财星通关
- 具体应在事业、婚姻、身体还是人际

## 七、枭神夺食详解

### 7.1 结构定义

偏印克食神，尤其食神为用时，称为**枭神夺食**。

### 7.2 常见应象

| 领域 | 具体表现 |
| :--- | :--- |
| **健康** | 少时缺乳、少食、消化吸收问题、中年大病 |
| **财运** | 食禄被夺、财路受损、突发破产 |
| **事业** | 才华输出受阻、技术能力被压制 |
| **寿命** | 严重时可应短寿 |

### 7.3 严重程度判断

| 条件 | 影响 |
| :--- | :--- |
| 食神为用且被夺 | 问题严重 |
| 食神为忌 | 偏印制食神未必全坏 |
| 食神旺相 | 偏印难以真正夺尽，影响较轻 |

### 7.4 救应条件

枭神夺食是否严重，需看是否有以下救应：
- **财星制偏印**：财克印，解偏印之克
- **比劫生食伤**：同辈力量加强食伤
- **大运流年救食神**：行至食神旺运
- **合冲改变关系**：地支合冲可化解夺食结构
