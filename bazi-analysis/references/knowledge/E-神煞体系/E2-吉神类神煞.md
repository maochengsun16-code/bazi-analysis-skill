# E2 吉神类神煞

> 吉神类神煞以吉利属性为主，但同样受制于命局喜忌和冲合刑害的影响。

---

## 一、太极贵人

### 含义

太极贵人代表神秘学、悟性、好学、贵气、起点不低。

### 特点

- 对玄学、宗教、哲学、命理感兴趣
- 学东西有悟性
- 思维能钻深
- 容易接触高层次知识

### 使用注意

- 查法以排盘软件为准，如甲乙见子午等
- 若配合格局清贵、五行有情，且不被冲破，可主富贵福寿
- 若命局本身格局破败，只能表示兴趣、悟性或某种精神追求，不必夸大

---

## 二、天德贵人与月德贵人

### 天德贵人

| 项目 | 说明 |
|:--- |:--- |
| 查法 | 按月令查 |
| 含义 | 如保险一般，遇灾祸能减轻 |
| 类象 | 逢凶化吉、灾祸减轻、官非减轻、身体灾伤减轻、遇贵人救助 |

### 月德贵人

| 项目 | 说明 |
|:--- |:--- |
| 查法 | 按三合月查 |
| 含义 | 与天德类似，也是减灾、解厄、化凶的吉神 |

### 天月二德的作用

- 天德、月德有一个就有一定保护力
- 二者同见，减灾效果更明显
- 本应死亡可能免死，本应牢狱可能免牢或刑罚减轻，本应大赔可能少赔

### 使用注意

- 天月二德是**减灾**，不是让所有凶事完全消失
- 若原局凶神很重，大运流年又强烈引动，仍会有事，只是程度可能减轻

## 三、文昌贵人、学堂、词馆

### 文昌贵人

| 项目 | 说明 |
|:--- |:--- |
| 含义 | 学习、考试、文才、清高、上进 |
| 特点 | 学习能力较好、考试运较好、有文笔或表达能力、气质清高、对知识有追求 |
| 被冲 | 考试发挥不稳、学习计划被打断、文书资料出问题 |

### 学堂

- 利学习、考试、学历
- 若命局印星、文昌、食伤配合好，读书考试更有优势

### 词馆

- 偏向文字、表达、教学、研究
- 适合教育、学校、教研、写作、语言表达、文书策划方向

## 四、天医

### 含义

天医与健康、医学、治疗、心理、调理有关。

### 特点

- 对医学感兴趣
- 适合医疗健康行业
- 适合心理咨询、调理、康复等方向
- 容易接触药物、医生、治疗

### 生旺与死绝

| 状态 | 表现 |
|:--- |:--- |
| 天医生旺 | 有利健康和医学缘分；若与印星、食伤、贵人配合好，可从事医学健康类工作 |
| 天医死绝冲克 | 身体偏弱，或健康问题较多，不一定代表当医生，也可能代表自己常接触医疗 |

---

## 五、将星

### 含义

将星代表领导力、掌控力、组织能力。

### 特点

- 有号召力，像孩子王
- 喜欢管事
- 能成为团队中心

### 使用注意

将星要真正成权势，还需配合官杀、印星、格局、权力环境。单有将星不等于一定当大官。

---

## 六、禄神与建禄格

### 禄神的含义

禄是临官之地，代表：

| 含义 | 说明 |
|:--- |:--- |
| 俸禄 | 职位对应的收入 |
| 饭碗 | 谋生依靠 |
| 职位 | 工作、社会角色 |
| 身体根气 | 日主自身力量 |
| 稳定收入来源 | 稳定的经济基础 |

### 建禄格

禄在月令，称为建禄格。

### 禄被冲

| 情形 | 类象 |
|:--- |:--- |
| 禄被冲 | 饭碗被冲，可能丢官、罢职、失业、断粮、身体根基受冲 |
| 禄为喜用 | 用神被冲，尤其要注意 |

---

## 七、金舆

### 含义

金舆代表体面、车舆、身份、社会地位。

### 特点

- 有面子
- 社会地位不低
- 生活条件较体面
- 受人尊重的外在形象

---

## 八、吉神使用注意事项汇总

1. **吉神也怕冲破**：所有吉神若被冲克刑害，吉力打折
2. **吉神需得地**：吉神在生旺之位、不被冲破时作用最大
3. **吉神需配格局**：吉神需配合良好的格局和五行平衡才能充分发挥
4. **吉神为忌时作用有限**：若吉神所在十神为忌神，吉力减弱
