# E3 凶神类神煞

> 凶神类神煞虽偏向凶性，但并非绝对为凶。若被制化得当，凶煞也可转为胆量、权力、行动力等正面力量。

---

## 一、魁罡

### 查法

魁罡以日柱为主，常见魁罡日：庚戌、庚辰、壬辰、戊戌。

### 含义与特点

| 项目 | 说明 |
|:--- |:--- |
| 性质 | 刚强、霸道、果断、权力、刑罚、强势 |
| 性格 | 性格硬、不服软、有掌控欲、做事果断、气势强 |

### 吉凶判断

**正面体现**（成格有力）：
- 适合公职、权力、管理、执法类方向
- 果断、敢管、有权威、能处理强压力事务

**负面体现**（弱或行财官杀）：
- 要注意罚款、官司、牢狱等问题
- 强势、霸道、不容易妥协

**婚姻影响**：
- 魁罡对婚姻不友好，气太刚导致感情中缺柔和，配偶关系压力大

### 制化

魁罡若成格有力，对事业有利。魁罡弱或者运行财官杀引发不利时，需注意守法、控制性格。

## 二、驿马

### 查法

以年支或日支查三合局的对应驿马地支：
- 申子辰马在寅
- 巳酉丑马在亥
- 寅午戌马在申
- 亥卯未马在巳

### 含义与特点

| 性质 | 说明 |
|:--- |:--- |
| 核心含义 | 走动、变化、远方、迁移、交通、奔波 |
| 常见应事 | 出差、搬家、迁移、外地发展、旅游、交通工具、行业变化 |

### 吉凶判断

| 状态 | 表现 |
|:--- |:--- |
| 驿马生旺 | 人圆滑，适应能力强，适合走动求财，外地机会较多 |
| 驿马死绝 | 人容易古板，走动不顺，外出机会质量不佳 |
| 驿马无合（无缰之马） | 到处跑，不容易安定，外地定居，流动性强 |
| 驿马被冲（惊马） | 奔波劳碌，突发出行，交通惊险，搬迁变化大 |
| 马星带财 | 动中求财、远方求财、交通物流贸易求财；财为喜则富，财为忌则奔波破财 |

### 流年驿马

流年遇驿马，常主当年走动增加，具体好坏看驿马所带十神和喜忌。

## 三、阳刃与羊刃

### 查法

阳干见帝旺之位：
- 甲刃在卯，丙刃在午，戊刃在午，庚刃在酉，壬刃在子
- 阴干不严格成刃

### 含义与特点

羊刃是很强的同类力量，带刚烈、争斗、刀刃、血光、冲动之象。

### 吉凶判断

| 状态 | 表现 |
|:--- |:--- |
| 身强犯羊刃 | 容易过旺，常见风险：血光、手术、磕碰、争斗、破财、冲动决策 |
| 身弱遇羊刃 | 羊刃可扶身，可能表示：胆量、执行力、体力、抗压能力、朋友同类帮助 |

### 流年羊刃

流年遇羊刃且为忌时，要慎重投资、避免冲动和危险活动。可能应破财、血光、手术、伤灾。若已有必要手术，可用计划性手术消应。

## 四、桃花

### 含义与特点

| 项目 | 说明 |
|:--- |:--- |
| 含义 | 异性缘、审美、吸引力、情感、欲望、人缘 |
| 正面 | 长相较好、聪明、情商高、会表达魅力、容易被人喜欢 |

### 吉凶判断

| 状态 | 表现 |
|:--- |:--- |
| 桃花为喜 | 魅力、人缘、情感顺利 |
| 桃花为忌（桃花煞） | 感情纠纷、桃色事件、婚姻不稳、情欲过重、名声受损 |
| 桃花死绝 | 容易好色而质量不高，感情欲望与现实关系不协调 |

### 使用注意

桃花不等于一定出轨。判断婚姻仍需看配偶星、配偶宫、桃花是否为喜、是否冲克配偶宫、是否与财官混杂。

## 五、空亡

### 查法

六十甲子中，每一旬有十个天干配十二地支，每旬会剩下两个地支没有配到，这两个地支就是空亡。

### 含义

| 项目 | 说明 |
|:--- |:--- |
| 核心含义 | 暂时虚、空、落空、不实、难以发挥 |
| 注意 | 落空的东西不是永远没有，而是当下不实、不稳、不容易落实 |

### 应事

需看空亡落在哪个地支、哪个宫位、哪个十神、哪个神煞：

| 落空对象 | 可能表现 |
|:--- |:--- |
| 财星空 | 钱财或父妻类象容易虚 |
| 子星空 | 子女缘分或子女信息不实 |
| 贵人空 | 贵人力打折 |

### 出空与冲空

空亡并非永远无用。大运流年出现出空、冲空时，原本落空的东西可能被激活。

### 含义与特点

| 项目 | 说明 |
|:--- |:--- |
| 性质 | 清冷、孤高、艺术、宗教、玄学、理性、孤独 |

### 组合参考

| 组合 | 效果 |
|:--- |:--- |
| 华盖+桃花+贵人 | 可能形成明星气质（桃花带吸引力，华盖带独特气质，贵人提升层次） |
| 华盖+文昌 | 学习态度较好，适合研究、艺术、学术、玄学、宗教、设计等方向 |

### 查法

金神格相关日柱：**乙丑日、己巳日、癸酉日**。

### 含义与特点

| 项目 | 说明 |
|:--- |:--- |
| 喜 | 金神喜火炼，真金需经火炼才能成器，喜丙火 |

## 十二、凶神类神煞使用总结

### 凶煞的正反转化

| 条件 | 转向正面 | 维持负面 |
|:--- |:--- |:--- |
| 制化得当 | 转为胆量、权力、行动力、横财 | — |
| 制化不力或为忌 | — | 应凶：血光、破财、官司、孤独 |

### 判断要点

1. **先看喜忌**：凶神为喜用时，凶性减弱，可能反而得力
2. **再看制化**：凶神被制化，凶力减轻；无制化且为忌，凶性增强
3. **结合宫位**：凶神所在宫位，决定在哪个领域应事
4. **大运流年引动**：原局凶神需大运流年引动才容易应事
5. **冲合刑害**：凶神被冲被合，状态会发生变化
