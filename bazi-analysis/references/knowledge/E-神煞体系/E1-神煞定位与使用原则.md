# E1 神煞定位与使用原则

## 一、神煞是辅助体系

神煞不能代替五行、十神、旺衰、格局。正确使用顺序如下：

### 1.1 使用优先级

| 步骤 | 内容 | 说明 |
|:--- |:--- |:--- |
| 第一步 | 看旺衰 | 日主和全局五行的旺衰强弱 |
| 第二步 | 看喜忌 | 确定哪些十神为喜、哪些为忌 |
| 第三步 | 看格局 | 分析八字格局成败 |
| 第四步 | 看宫位 | 分析各宫位代表的六亲、阶段 |
| 第五步 | 看合冲刑害 | 地支之间的相互作用 |
| 第六步 | 神煞补象 | 用神煞补充象义，辅助验证 |

> 不能只因命局有某个神煞就直接断吉凶。

### 1.2 神煞的真假

问真排盘中显示的神煞，多数属于可参考的真神。但深入辨别真神假神较为复杂，入门阶段先掌握常见神煞的象义和使用边界。

---

## 二、神煞与宫位

同一个神煞落在不同柱，含义不同。

| 宫位 | 对应含义 |
|:--- |:--- |
| 年柱 | 早年、祖上、父母、外部大环境 |
| 月柱 | 青年、事业环境、兄弟朋友、家庭环境 |
| 日柱 | 自己、配偶、中年、亲密关系 |
| 时柱 | 子女、晚年、结果、下属晚辈 |

---

## 三、神煞的作用条件

### 3.1 神煞要看旺衰与冲合刑害

神煞不是孤立存在的，要看以下因素：
- 是否生旺
- 是否死绝
- 是否被冲
- 是否被合
- 是否被刑害穿
- 是否落空亡
- 所在地支藏干是什么十神
- 这个十神是喜是忌

| 条件 | 结果 |
|:--- |:--- |
| 吉神被冲破 | 吉力减弱、打折 |
| 凶煞被制化 | 凶力减轻、转为用 |

### 3.2 吉神也怕冲破

贵人、文昌、禄神、天医等吉神，如果被冲克刑害，作用会打折。

### 3.3 凶煞也可被制化

羊刃、魁罡、劫煞、灾煞等凶煞，如果被制化得当，也可转为：
- 胆量
- 权力
- 行动力
- 横财机会

### 3.4 神煞要结合大运流年应事

| 情形 | 应事参考 |
|:--- |:--- |
| 原局有神煞 | 是结构，代表潜在特质 |
| 大运流年引动神煞 | 容易应事 |
| 流年驿马 | 奔波走动 |
| 流年羊刃 | 注意血光破财 |
| 文昌被流年冲 | 考试发挥不稳 |
| 空亡出空 | 原本虚的东西被激活 |

---

## 四、使用边界与注意事项

1. **主次分明**：五行十神是主体，神煞是辅助，不可颠倒
2. **忌以偏概全**：单凭一个神煞断命，误差极大
3. **因地制宜**：同一神煞在不同命局中吉凶不同，取决于喜忌和制化情况
4. **动态看待**：神煞随大运流年变化，某段时期吉、某段时期凶，需要动态分析
5. **参考属性**：神煞之间也相互影响，多个吉神同见增强，凶煞叠加加剧
