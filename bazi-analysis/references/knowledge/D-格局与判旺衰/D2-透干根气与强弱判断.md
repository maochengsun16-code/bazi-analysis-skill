# D2 透干根气与强弱判断

## 一、透干的含义

地支中的藏干在四柱天干中出现，即为**透干**。透干不要求必须同柱，只要地支藏有某天干，而四柱天干中出现这个天干，即可称为透出。

### 1.1 本气、中气、余气均可透

| 藏干层次 | 说明 | 力量 |
|:--- |:--- |:--- |
| 本气透干 | 地支主气在天干出现 | 力量最明显 |
| 中气透干 | 地支次气在天干出现 | 有作用，力量中等 |
| 余气透干 | 地支余气在天干出现 | 有作用，力量偏弱 |

### 1.2 透干的意义

- 地支内部力量显露到天干层面（隐藏的事变得外显）
- 某个十神有表达渠道
- 格局取用有依据
- 原本藏在地支的人事浮到表面

---

## 二、根气的概念

判断日主旺衰时，**有根气只认地支层面的明现根**，不按藏干层级简单认根。

正式有根气只有三种情况：

1. 地支明现比劫。
2. 地支明现印。
3. 地支有日主五行对应的库。

**中气根和余气根不算有根气**，只能作为藏干线索或透干后的辅助信息，不能直接据此断“日主有根、能担财官”。

戊己土比较特殊：辰戌丑未是戊己土的比劫根；巳午火既是戊己土的印，也可以兼看成戊己土的比劫助力。

### 2.1 根气的精确定义

| 日主 | 明现比劫根 | 明现印根 | 对应库根 |
|:--- |:--- |:--- |:--- |
| 甲/乙木 | 寅/卯 | 亥/子 | 辰/未 |
| 丙/丁火 | 巳/午 | 寅/卯 | 戌/未 |
| 戊/己土 | 辰/戌/丑/未（巳午可兼看助力） | 巳/午 | 辰/戌/丑/未 |
| 庚/辛金 | 申/酉 | 辰/戌/丑/未 | 丑/戌 |
| 壬/癸水 | 亥/子 | 申/酉 | 辰/丑 |

### 2.2 中气、余气的处理

中气和余气可以说明地支内部藏有什么，但**不能单独作为日主有根气的依据**。

| 情况 | 是否算根气 | 用法 |
|:--- |:--- |:--- |
| 地支明现比劫 | 算 | 正式根气 |
| 地支明现印 | 算 | 正式根气 |
| 地支有对应库 | 算 | 正式根气 |
| 只有中气 | 不算 | 藏干线索，需另有明现根才可加强判断 |
| 只有余气 | 不算 | 藏干线索，不可直接断有根 |

### 2.3 按干支位置划分

日主坐支有正式根气，强于年支、月支、时支的根气。

| 位置 | 力度 | 示例 |
|:--- |:--- |:--- |
| 坐支明现比劫/印 | 极强 | 甲寅、乙亥、丙寅、庚辰等 |
| 他支明现比劫/印 | 强 | 日主在年/月/时支见明现比劫或印 |
| 坐支对应库 | 中 | 坐支为日主对应库 |
| 他支对应库 | 中偏弱 | 年/月/时支为日主对应库 |
| 只有中气/余气 | 不按根气论 | 只能作为藏干线索 |

## 三、库根详解

天干五行在地支有对应墓库，称为**库根**。库根按正式根气论，但要明确：这是因为该地支是对应库，不是因为中气或余气本身算根。

### 3.1 四墓库藏干与五行归属

| 地支 | 本气 | 中气 | 余气 | 主库 | 次库 |
|:--- |:--- |:--- |:--- |:--- |:--- |
| 辰 | 戊土 | 乙木 | 癸水 | 水库 | 木库 |
| 未 | 己土 | 丁火 | 乙木 | 木库 | 火库 |
| 戌 | 戊土 | 辛金 | 丁火 | 火库 | 金库 |
| 丑 | 己土 | 癸水 | 辛金 | 金库 | 水库 |

### 3.2 各五行对应的库根

| 五行 | 库根地支 | 说明 |
|:--- |:--- |:--- |
| 甲乙木 | 辰（乙木余气）、未（乙木余气） | 木的库根 |
| 丙丁火 | 未（丁火中气）、戌（丁火余气） | 火的库根 |
| 庚辛金 | 戌（辛金中气）、丑（辛金余气） | 金的库根 |
| 壬癸水 | 辰（癸水余气）、丑（癸水中气） | 水的库根 |
| 戊己土 | 辰戌丑未本气即为土 | 视为比劫根/土库；巳午火为印，也可兼看比劫助力 |

> 库根可算有根气；但中气、余气本身不算有根气。

---

## 四、通根与担财官

只要是地支有**明现比劫、明现印、对应库**的，都可以按有根气论，具备担财官的基础。

| 种根类型 | 是否可以担财官 |
|:--- |:--- |
| 比劫根（地支明现比劫） | 可以担财官 |
| 印根（地支明现印） | 可以担财官 |
| 库根（地支见对应库） | 可以担财官 |
| 仅中气或余气 | 不算有根气，不可直接断能担财官 |

## 五、综合判断日主强弱

判断日主强弱需要从以下三个维度综合分析：

### 5.1 得令

日主是否在月令当令，或在月令有生扶。

| 月令状态 | 对日主的影响 |
|:--- |:--- |
| 日主当令（月令为日主比劫） | 得令，力量强 |
| 月令生日主（月令为日主印） | 得生，力量较强 |
| 月令克日主（月令为官杀） | 失令，力量弱 |
| 月令耗日主（月令为财） | 失令，力量偏弱 |
| 月令泄日主（月令为食伤） | 失令，力量偏弱 |

### 5.2 得根

日主在地支是否有强根支持。

| 根气类型 | 力度 |
|:--- |:--- |
| 坐支明现比劫/印 | 最强 |
| 他支明现比劫/印 | 强 |
| 坐支库根 | 中 |
| 他支库根 | 中偏弱 |
| 仅中气/余气 | 不算根气 |
| 无根 | 极弱 |

### 5.3 得生

日主在天干是否有印星生扶或其他干支帮扶。

### 5.4 强弱综合判断

| 条件组合 | 日主状态 |
|:--- |:--- |
| 得令 + 有明现比劫/印 + 得生 | **极旺** |
| 得令 + 有明现比劫/印 | **偏旺** |
| 得令 + 有库根 | 中和偏旺 |
| 不得令 + 有明现比劫/印 + 得生 | 中和 |
| 不得令 + 有库根 | 中和偏弱 |
| 不得令 + 只有中气/余气 + 不得生 | **偏弱（按无根看）** |
| 不得令 + 无根 + 被克泄耗 | **极弱（从格）** |

### 5.5 根气强弱维度总结

```
根气强弱：坐支 > 他支；明现比劫/印 > 对应库。中气、余气不算根气。

坐支明现比劫/印	↓↓↓ 极强
坐支库根		↓↓ 中等
他支明现比劫/印	↓↓ 较强
他支库根		↓ 中偏弱
中气/余气		. 不算根气
无根			× 极弱
```
