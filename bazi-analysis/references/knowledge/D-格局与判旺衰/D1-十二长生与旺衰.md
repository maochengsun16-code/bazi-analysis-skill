# D1 十二长生与旺衰

## 一、十二长生的含义

十二长生描述五行从出生到旺盛、衰退、死亡、收藏的能量循环，共十二个阶段：

| 阶段 | 含义 | 能量状态 |
|:--- |:--- |:--- |
| 长生 | 出生、萌发、新起点 | 开始生长 |
| 沐浴 | 成长、成形、脆弱期 | 逐渐壮大 |
| 冠带 | 成年、独立、渐趋成熟 | 趋于完备 |
| 临官（禄） | 当官、有职位、巅峰 | 能量极强 |
| 帝旺（刃） | 鼎盛至极、顶峰 | 能量最旺 |
| 衰 | 衰退、下落 | 开始减弱 |
| 病 | 衰败、多病 | 继续衰退 |
| 死 | 消亡、结束 | 能量消散 |
| 墓 | 收藏、入库、蓄势 | 暂时收存 |
| 绝 | 灭绝、无气 | 能量最低 |
| 胎 | 受胎、孕育 | 重新酝酿 |
| 养 | 滋养、蓄力 | 准备再起 |

> 十二长生并非单独断吉凶的体系，而是用来观察某五行、某十神所处的状态阶段。

## 二、阳顺阴逆排法

十二长生的排列遵循**阳干顺行、阴干逆行**的原则：

- **阳干**：从长生位开始，顺时针方向排列（长生→沐浴→冠带→临官→...→胎→养）
- **阴干**：从长生位开始，逆时针方向排列（长生←沐浴←冠带←临官←...←胎←养）

## 四、重点阶段详解

### 4.1 临官（禄）

- 能量极强、处于当权阶段
- 称为"禄"，代表俸禄、职位、稳定收入
- 地支见临官即为有禄根

### 4.2 帝旺（刃）

- 能量最旺、达到顶峰的阶段
- 称为"刃"，严格来说阳干才有刃
- 帝旺过刚，容易冲动争斗

### 4.3 死

- 能量消散、功能终止
- 某个五行或十神到死地，力量极弱

### 4.4 墓

- 能量收藏入库，并非消失
- 时运到来时可重新激活（出墓）
- 与四墓库概念有密切关联

### 4.5 绝

- 能量最低、濒临灭绝
- 需要胎养阶段重新蓄力

## 五、六亲临死墓绝的判断应用

某个十神代表的六亲临死、墓、绝，说明该六亲状态不强或缘分偏弱：

| 十神 | 临死墓绝的可能影响 |
|:--- |:--- |
| 官星临死墓绝 | 女命丈夫缘、丈夫状态受影响 |
| 财星临死墓绝 | 男命妻缘、父亲、财源受影响 |
| 食伤临死墓绝 | 子女缘、输出能力受影响 |
| 印星临死墓绝 | 母亲缘、学业、文书受影响 |
| 比劫临死墓绝 | 兄弟姐妹缘、社交能力受影响 |

> 不能仅凭十二长生直断，必须结合喜忌和大运救应综合判断。

## 六、使用原则

1. **状态参考**：十二长生是能量阶段标识，本身不直接定吉凶
2. **结合喜忌**：同样的临官/帝旺状态，喜则吉、忌则凶
3. **大运救应**：死墓绝的状态可通过大运流年引动重新激活
4. **整体判断**：需要结合五行旺衰、十神组合、格局成败综合分析
