# D3 四墓库专题

## 一、四墓库的定义

四墓库指地支中的**辰、戌、丑、未**四个地支，每个库都包含两种五行属性。

| 地支 | 属性 | 藏干 | 主库 | 次库 |
|:--- |:--- |:--- |:--- |:--- |
| 辰 | 水库 | 戊（本气）、乙（中气）、癸（余气） | 水库 | 木库 |
| 戌 | 火库 | 戊（本气）、辛（中气）、丁（余气） | 火库 | 金库 |
| 丑 | 金库 | 己（本气）、癸（中气）、辛（余气） | 金库 | 水库 |
| 未 | 木库 | 己（本气）、丁（中气）、乙（余气） | 木库 | 火库 |

> 关键原则：每个库都包含两种五行属性，分析时必须两种都考虑。

## 二、开库的四种方式

一个墓库要发挥作用，需要先"打开"。满足任意一种即为开库。

### 2.1 天干透出

当库中的藏干在四柱天干任何一柱透出时，该库即打开。

| 库 | 天干透出条件 |
|:--- |:--- |
| 辰（水库+木库） | 天干透壬/癸水（水透出），或甲/乙木（木透出） |
| 戌（火库+金库） | 天干透丙/丁火（火透出），或庚/辛金（金透出） |
| 丑（金库+水库） | 天干透庚/辛金（金透出），或壬/癸水（水透出） |
| 未（木库+火库） | 天干透甲/乙木（木透出），或丙/丁火（火透出） |

### 2.2 地支出现

库中五行的同类在地支出现，等于透出，库即为开。

| 库 | 地支出现条件 |
|:--- |:--- |
| 辰 | 地支见子/亥水，或寅/卯木 |
| 戌 | 地支见巳/午火，或申/酉金 |
| 丑 | 地支见亥/子水，或申/酉金 |
| 未 | 地支见巳火（午火不行，午未合闭库），或寅/卯木 |

### 2.3 冲库（对冲开库）

辰戌相冲、丑未相冲，两库同时打开。

- **辰戌冲**：辰库和戌库同时打开
- **丑未冲**：丑库和未库同时打开

> 与天干透出不同（只开一个库），冲和刑会使两个库都打开。

### 2.4 刑库（相刑开库）

| 相刑类型 | 效果 |
|:--- |:--- |
| 丑戌相刑 | 开库 |
| 未戌相刑 | 开库 |
| 丑未戌三刑 | 开库 |
| 辰辰自刑 | **不开库**（唯一例外） |

### 2.5 拱开库（拱合开库）

当两个地支挨着（必须相邻，不能隔位）时，可以通过拱合的方式开库。

| 拱合 | 拱出之物 |
|:--- |:--- |
| 申+辰 | 拱子水 |
| 寅+戌 | 拱午火 |
| 巳+丑 | 拱酉金 |
| 亥+未 | 拱卯木 |

> 必要条件：两个地支必须在地支中相邻（紧挨着），中间有隔位则不论拱。

### 2.6 开库方式总结

| 方式 | 说明 | 范围 |
|:--- |:--- |:--- |
| 天干透出 | 库中五行在天干显现 | 打开该库 |
| 地支出现 | 库中五行在地支有同类出现 | 打开该库 |
| 冲库 | 辰戌冲 / 丑未冲 | 两库全开 |
| 刑库 | 丑戌刑 / 未戌刑 / 丑未戌三刑 | 两库全开 |
| 拱开 | 相邻地支拱合 | 拱出即为开 |

## 三、闭库与半开半闭

### 3.1 六合闭库

当出现以下六合时，对应的库即为闭。

| 六合 | 被闭的库 | 说明 |
|:--- |:--- |:--- |
| 辰+酉 | 闭辰库 | 辰酉六合 |
| 卯+戌 | 闭戌库 | 卯戌六合 |
| 子+丑 | 闭丑库 | 子丑六合 |
| 午+未 | 闭未库 | 午未六合 |
| 寅+亥 | 不影响开库 | 寅亥合火 |
| 巳+申 | 不影响开库 | 巳申合水 |

### 3.2 半开半闭状态

天干已透出库中五行（满足开库条件），同时地支又有六合，则属于没有彻底打开、但也没有彻底关闭的状态。此时依然按开库处理。

> 结论：天干透出 > 六合，此时库不算彻底闭，依然按开库处理。

### 3.3 六冲与六合同在时的判断

| 条件 | 库的状态 |
|:--- |:--- |
| 辰或戌在月令 | 冲的力量大，库是开着的 |
| 辰戌都不在月令，且天干未透出 | 库是闭的 |
| 天干已透出 | 无论谁在月令，库都闭不掉 |

## 四、库做功与发财/破财（核心应用）

### 4.1 内外之分

| 位置 | 含义 | 性质 |
|:--- |:--- |:--- |
| 年月 | 家外/社会上 | 公共的、别人的 |
| 日时 | 家里 | 私有的、自己的 |

### 4.2 发财原理

家里的比劫库制掉外面的财库则发财。

| 做功方式 | 发财条件 |
|:--- |:--- |
| 冲 | 家内比劫库（日时）冲开年月财库 |
| 刑 | 家内比劫库（日时）刑开年月财库 |
| 合 | 家内有午/卯/亥合住年月的财库（拉回家） |

> 合财时，财库必须是开着的（闭库则不论）。

### 4.3 破财原理

外面的比劫库制掉我家的财库则破财。

| 做功方式 | 破财条件 |
|:--- |:--- |
| 冲 | 年月比劫库冲开家内财库 |
| 刑 | 年月比劫库刑开家内财库 |

**示例**：年月有丑（比劫库），日时有未（财库），丑未冲，年月的比劫库冲开家里的财库，此为破财之象。

## 六、四墓库月取用方法

### 6.1 土月两重做功原则

辰戌丑未月（土月）的八字，有两重做功：

1. **土本身**的做功（一层）
2. **土里面透出来的字**的做功（另一层）

| 情况 | 取用方向 |
|:--- |:--- |
| 库中五行一点没透，只透出土 | 直接当土看 |
| 库中五行有透出 | 土和透出之物两重都要看 |

### 6.2 各月取用要点

**辰月（春末）**
- 春天木旺，身旺则喜财官（土、官杀）
- 需具体看日主强弱

**未月（夏末）**
- 火旺，如未中火透出，印旺则先制化印
- 制印用财，化印用比劫
- 食伤也可用（耗土、制官）

**戌月（秋末）**
- 秋天金旺，杀旺时先取印（官杀猖獗，以印化之）
- 食伤制不住太旺的七杀则不用

**丑月（冬末）**
- 冬天水旺
- 水旺为财，用比劫制财
- 水旺为官杀，用印化官杀

### 6.3 取用原则总结

| 最旺的五行 | 取用原则 | 具体方法 |
|:--- |:--- |:--- |
| 印旺 | 制化印 | 用财制 / 用比劫化 |
| 财旺 | 制化财 | 用比劫制 / 用印制 |
| 官杀旺 | 以印化之 | 食伤制不住则取印 |
| 比劫旺 | 取财官食伤 | 用官杀 / 食伤制 |
