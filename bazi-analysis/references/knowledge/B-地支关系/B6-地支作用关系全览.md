# 地支作用关系全览

---

## 一、所有地支关系一览总表

### 1.1 地支关系分类总纲

| 关系类型 | 数量 | 核心含义 | 主要应用 |
|----------|------|----------|----------|
| 六合 | 6组 | 牵绊、结合、合作、私情 | 婚恋、合作、闭库 |
| 三合局 | 4组 | 力量凝聚、化局 | 五行力量增强 |
| 三会局 | 4组 | 季节气势聚集 | 季节气场加强 |
| 拱合 | 4组 | 暗中趋势、潜在结构 | 暗财、暗贵人、潜在应事 |
| 六冲 | 6组 | 动、散、变、撞击、对立 | 变动、分离、冲突 |
| 破 | 6组 | 破坏、破财、关系不好 | 夫妻、六亲、合作、破财 |
| 害（穿） | 6组 | 快速伤害、内部损伤 | 健康、六亲、宫位受伤 |
| 三刑 | 2组+自刑 | 痛苦、纠缠、内耗 | 牢狱、健康、长期纠缠 |
| 绝 | 4组 | 死绝、关系难续 | 夫妻、六亲寿元、关系断绝 |

### 1.2 关系强弱的使用边界

地支关系没有一个可以跨所有专题机械套用的绝对排序。分析时先看专题目标，再看该专题下哪种关系最关键：

- 看财运：重点看制财官、库做功、破到财星是否破财。
- 看婚姻：重点看夫妻星与夫妻宫的星宫关系，破、绝、冲、穿、刑都要回到夫妻宫和夫妻星。
- 看六亲健康：重点看六亲星是否受伤，穿、刑、绝、破、冲都要看状态，不先按喜忌。
- 看合作关系：破、破合、冲、穿常比普通合更值得优先看。

在夫妻、六亲、合作等“关系伤害”语境下，可用以下顺序作粗略参考：

**合 < 冲/破 < 害(穿)/刑 < 绝**

其中“绝比破重”主要是破绝内部比较，不代表所有专题都要先看绝。

## 二、各关系详解

### 2.1 六合

| 组合 | 合化五行 | 分类 | 暗含伤害 |
|------|----------|------|----------|
| 子丑合 | 化土 | 闭库之合 | — |
| 寅亥合 | 化木 | 破合 | 寅中丙火被亥水灭掉 |
| 卯戌合 | 化火 | 闭库之合 | 卯木被戌中辛金克伤 |
| 辰酉合 | 化金 | 闭库之合 | 辰中乙木被酉金破坏 |
| 巳申合 | 化水 | 破合 | 巳中丙火被申中壬水灭掉 |
| 午未合 | 化火 | 闭库之合 | — |

**核心应用场景**：
- 感情合婚、合作关系
- 配偶宫被合 → 婚恋被外界牵动
- 暗合 → 偷情出轨、暗中求财

### 2.2 三合局

| 合局 | 组合 | 化神 | 季节要求 | 位置要求 | 透干要求 |
|------|------|------|----------|----------|----------|
| 申子辰合水 | 申+子+辰 | 水 | 冬>秋 | 三字挨着 | 透壬/癸水 |
| 寅午戌合火 | 寅+午+戌 | 火 | 夏>春 | 三字挨着 | 透丙/丁火 |
| 亥卯未合木 | 亥+卯+未 | 木 | 春>冬 | 三字挨着 | 透甲/乙木 |
| 巳酉丑合金 | 巳+酉+丑 | 金 | 秋、辰月、丑月 | 三字挨着 | 透庚/辛金 |

**核心应用场景**：
- 判断命局五行力量增强
- 合化为喜神→富贵，合化为忌神→麻烦
- 合绊与合化区别判断

### 2.3 三会局

| 三会局 | 组合 | 方位 | 季节 |
|--------|------|------|------|
| 寅卯辰三会木 | 寅+卯+辰 | 东方木 | 春季 |
| 巳午未三会火 | 巳+午+未 | 南方火 | 夏季 |
| 申酉戌三会金 | 申+酉+戌 | 西方金 | 秋季 |
| 亥子丑三会水 | 亥+子+丑 | 北方水 | 冬季 |

**核心应用场景**：
- 季节气场判断
- **力量大于三合**，成局后对应五行力量非常明显

### 2.4 拱合

| 拱合 | 拱出的中神 | 透干要求 |
|------|------------|----------|
| 生辰拱子水 | 子→癸水 | 透癸水 |
| 寅戌拱午火 | 午→丁火 | 透丁火 |
| 亥未拱卯木 | 卯→乙木 | 透乙木 |
| 巳丑拱酉金 | 酉→辛金 | 透辛金 |

**条件**：必须挨着 + 天干透阴干化神

**核心应用场景**：
- 暗中财运、暗贵人、小人
- 原局有拱，大运流年补齐应事

### 2.5 六冲

| 冲 | 属性 | 冲动季节 | 疾病取向 |
|----|------|----------|----------|
| 子午冲 | 水火交战 | 冬冲夏，夏不敢冲冬 | 心脏、近视 |
| 巳亥冲 | 水火交战 | 冬冲夏，夏不敢冲冬 | 心脑血管、三高 |
| 寅申冲 | 金木交战 | 秋申冲寅 | 四肢、头疼、肝胆 |
| 卯酉冲 | 金木交战 | 秋酉冲卯 | 四肢、肝胆 |
| 丑未冲 | 库冲（土旺） | 秋丑冲未 | 胃/脾 |
| 辰戌冲 | 库冲（土旺） | 冬辰冲戌 | 车祸 |

**核心应用场景**：
- 变动、搬家、分离、冲突、手术
- 宫位被冲应事方向判断
- 用神制忌神为福，忌神制用神为祸

### 2.6 破

| 破 | 条件 |
|----|------|
| 子酉破 | 子水当令 |
| 寅亥破 | 不论季节（六合带破） |
| 卯午破 | 卯木当令 |
| 辰丑破 | 不论季节，但原局需相邻 |
| 巳申破 | 不论季节（六合带破） |
| 未戌破 | 不论季节，但原局需相邻 |

原局论破要看上表条件；大运流年来破原局时，不再按原局月令和位置筛选，只看运年之字破到原局哪个十神或宫位。

**核心应用场景**：
- 夫妻关系、合作关系
- 破到夫妻宫→关系破损、争吵；原局配合差或运年引动时才上升为离婚风险
- 破到年月的印星→与母亲关系不好
- 破到财星→破财、损耗

### 2.7 害（穿）

| 穿 | 属性 | 疾病取向 |
|----|------|----------|
| 寅巳穿 | 火木相穿 | 头疼、四肢筋骨受伤 |
| 子未穿 | 水土相穿 | 子宫肌瘤/卵巢囊肿、近视、爱哭 |
| 卯辰穿 | 木土相穿 | 肝胆问题、皮肤问题 |
| 酉戌穿 | 金火相穿 | 牙齿不好、近视、心脏问题 |
| 申亥穿 | 金水相穿 | 头部或四肢暗疾 |
| 丑午穿 | 土火相穿 | 近视、牙齿问题、爱哭 |

**核心应用场景**：
- 六亲健康判断（穿是很直接的伤害）
- 宫位被穿→对应六亲出问题
- 原局带穿，大运流年再来→应期

### 2.8 三刑

| 刑 | 组合 | 特点 |
|----|------|------|
| 丑未戌三刑 | 丑+未+戌 | 恃势之刑，库墓相刑，土气纠缠 |
| 寅巳申三刑 | 寅+巳+申 | 无恩之刑，看月令决定谁受伤 |
| 自刑 | 亥亥、辰辰、酉酉、午午 | 自我纠结，同一气场内耗 |

**核心应用场景**：
- 牢狱判断（丑未戌三刑为官杀）
- 抑郁症判断（食伤受刑）
- 六亲健康、人际关系纠缠

### 2.9 绝

| 绝 | 属性 | 条件 |
|----|------|------|
| 子巳绝 | 水火绝 | 子水当令或冬天 |
| 亥午绝 | 水火绝 | 冬天亥水有力 |
| 卯申绝 | 金木绝 | 秋天申金有力 |
| 寅酉绝 | 金木绝 | 秋天酉金有力 |

**核心应用场景**：
- 六亲寿元判断（被绝的母亲/父亲走得早）
- 婚姻稳定性（绝到夫妻宫→多段婚姻）
- 在破绝同类取象中，绝通常比破更严重；但仍要看十神、宫位和力量条件

## 三、核心口诀体系

### 3.1 用神忌神交战（适用于所有作用关系）

```
用神制忌神 → 应吉（好事/多富贵）
忌神制用神 → 应凶（坏事/负债/勿冒进）
忌神与忌神交战 → 日主得利（应吉）
用神与用神交战 → 有凶
用神化忌神 → 应吉
忌神化用神 → 应凶
```

> 这是整个八字论命的核心，适用于刑、冲、穿等所有作用关系。

### 3.2 全关系判断口诀

| 关系 | 判断口诀 |
|------|----------|
| 冲 | 力量大的一方才有主动权去冲，看季节决定 |
| 穿 | 力量大的一方才有主动权去伤害，看月令决定 |
| 三刑 | 看月令决定谁受伤 |
| 绝 | 我的力量比你强，我才有主动权去伤害你 |
| 三合 | 季节+位置+透干三条件同时满足才叫合化，缺一为合绊 |
| 拱合 | 必须挨着+天干透阴干化神 |
| 三刑 | 刑必原局有，运年补齐不算真 |

### 3.3 同一原则：看月令决定力量大小

所有地支关系的共同核心原则——**看月令**：

| 关系 | 月令如何起作用 |
|------|----------------|
| 冲 | 月令上的地支力量大，能冲对方；月令上的地支弱则被对方冲 |
| 穿 | 谁在月令上谁力量大，就能穿得动对方 |
| 三刑（寅巳申） | 看月令上是谁，决定到底谁把谁刑坏 |
| 破 | 部分破需当令（子酉破需子当令，卯午破需卯当令） |
| 绝 | 要看季节力量，月令决定是否有力量绝对方 |
| 三合 | 季节（月令）是合化三条件之一 |
| 六合 | 化神是否有力与月令相关 |

## 四、实践应用速查表

### 4.1 六亲健康速查

| 六亲 | 看哪个地支/藏干 | 易受伤的关系 |
|------|-----------------|--------------|
| 父亲 | 偏财、财星地支 | 穿、破、绝、刑 |
| 母亲 | 印星地支 | 穿、破、绝、刑 |
| 配偶（男） | 财星优先，官杀备用；夫妻宫 | 穿、冲、破、绝 |
| 配偶（女） | 官杀优先，财星备用；夫妻宫 | 穿、冲、破、绝 |
| 子女 | 食伤、时柱 | 穿、刑、冲 |
| 自己 | 日主禄、日支 | 破、绝、穿 |

### 4.2 婚姻感情速查

| 关系 | 出现在夫妻宫 | 表现 |
|------|--------------|------|
| 六合（暗合） | 子丑暗合、巳申暗合等 | 容易出轨/偷情 |
| 闭库之合 | 子丑合、午未合等 | 关系糟糕，互相拖累 |
| 破合 | 寅亥合、巳申合 | 表面和谐，内里算计 |
| 冲 | 日支被冲 | 感情不稳，易变动分离 |
| 破 | 破到夫妻宫 | 关系破损、争吵；星宫同破或运年引动时离婚风险高 |
| 绝 | 绝到夫妻宫 | 一生婚姻不稳定，多段婚姻 |
| 穿 | 穿配偶宫 | 婚恋关系容易受伤 |
| 刑 | 刑到夫妻宫 | 关系痛苦纠缠 |

### 4.3 牢狱风险速查

- **丑未戌三刑为官杀** + 流年不佳 → 有牢狱风险
- 三刑断牢狱比自刑严重

### 4.4 抑郁症速查

- **食伤受刑**（无论三刑还是自刑）→ 极易抑郁
- 食伤代表表达、宣泄、自由精神，受刑则表达受阻、内心压抑

### 4.5 疾病判断速查

| 关系 | 常见疾病 |
|------|----------|
| 子午冲 | 心脏问题、近视 |
| 巳亥冲 | 心脑血管疾病、三高 |
| 寅巳穿 | 头疼、四肢筋骨受伤 |
| 子未穿 | 子宫肌瘤/卵巢囊肿、近视、爱哭 |
| 卯辰穿 | 肝胆问题、皮肤问题 |
| 酉戌穿 | 牙齿不好、近视、心脏问题 |
| 丑午穿 | 近视、牙齿问题 |
| 辰戌冲 | 车祸（应验率高） |

### 4.6 财运判断速查

| 关系 | 为福条件 | 为祸条件 |
|------|----------|----------|
| 六冲 | 用神冲忌神 → 发财 | 忌神冲用神 → 破财/负债 |
| 三合 | 合化为喜神 → 富贵 | 合化为忌神 → 麻烦 |
| 拱合（财星） | 暗财、莫名其妙发财 | 暗中破财不知不觉 |
| 穿财星宫位 | — | 财路、父亲、妻子有损 |
| 破财星 | — | 与父亲关系不好，容易破财 |

### 4.7 各关系宫位应用

| 宫位 | 冲 | 穿 | 刑 | 破 | 绝 |
|------|----|----|----|----|----|
| 年柱 | 祖上、父母、早年平台动 | 父母、祖上受伤 | 早年纠缠 | 母亲关系不好 | 母亲/父亲走得早 |
| 月柱 | 家庭、兄弟、工作环境动 | 兄弟、家庭受伤 | 青年期痛苦 | 父亲/财路 | — |
| 日支 | 配偶、婚姻动 | 配偶受伤 | 婚姻纠缠 | 闹离婚 | 多段婚姻 |
| 时柱 | 子女、晚年动 | 子女受伤 | 晚年纠缠 | — | 子女问题 |
