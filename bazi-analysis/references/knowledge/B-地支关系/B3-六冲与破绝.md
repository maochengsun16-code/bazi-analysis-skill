# 六冲与破绝

---

## 一、地支六冲

### 1.1 六冲总论

六组对冲：

| 对冲组合 | 属性 | 类型 |
|----------|------|------|
| 子午冲 | 水火交战 | 五行冲 |
| 巳亥冲 | 水火交战 | 五行冲 |
| 寅申冲 | 金木交战 | 五行冲 |
| 卯酉冲 | 金木交战 | 五行冲 |
| 丑未冲 | 库冲（土冲土旺） | 库冲 |
| 辰戌冲 | 库冲（土冲土旺） | 库冲 |

#### 冲的基本含义

冲表示动、散、变、撞击、对立。常见类象：
- 搬家、变动、分离、冲突、事故、手术、工作变化、感情不稳、宫位被动摇

#### 冲的核心原则

**力量大的一方才有主动权去冲**：看季节来决定谁力量大。为祸为福看喜忌：
- 用神制忌神 → 冲福（好事）
- 忌神制用神 → 冲祸（坏事）

### 1.2 冲与宫位

| 宫位 | 应事方向 |
|------|----------|
| 年柱被冲 | 祖上、父母、早年、远方环境、单位平台等动 |
| 月柱被冲 | 家庭环境、兄弟朋友、工作环境、青年阶段动 |
| 日支被冲 | 配偶宫、婚姻、亲密关系、中年阶段动 |
| 时柱被冲 | 子女、晚年、事业结果、下属晚辈动 |

### 1.3 子午冲（水火交战）

| 情况 | 说明 |
|------|------|
| 冲动的季节 | 冬天子水冲午火；夏天子水不敢冲午火 |
| 为福 | 子水（用神）冲午火（忌神）→ 发财 |
| 为祸 | 子水（忌神）冲午火（用神）→ 负债/破财 |
| 疾病 | 心脏问题、近视（午为心、目） |

### 1.4 巳亥冲（水火交战）

| 情况 | 说明 |
|------|------|
| 冲动的季节 | 冬天亥水冲巳火；夏天亥水不敢冲巳火 |
| 为福 | 亥水（用神）冲巳火（忌神）→ 好事 |
| 为祸 | 亥水（忌神）冲巳火（用神）→ 破财 |
| 疾病 | 心脑血管疾病、三高（巳为心血管） |

### 1.5 寅申冲（金木交战）

| 情况 | 说明 |
|------|------|
| 冲动的季节 | 只有秋天申金才能冲寅木；其他季节（春、夏、冬）申金都冲不了寅木 |
| 为福 | 申金（用神）冲寅木（忌神）→ 应富贵 |
| 为祸 | 申金（忌神）冲寅木（用神）→ 破财 |
| 疾病 | 四肢易受伤、头疼、肝胆问题（寅为胆、头、四肢） |

### 1.6 卯酉冲（金木交战）

| 情况 | 说明 |
|------|------|
| 冲动的季节 | 只有秋天卯木受伤（金旺木死） |
| 为福 | 酉金（用神）制卯木（忌神）→ 好事 |
| 为祸 | 酉金（忌神）制卯木（用神）→ 财运不佳、破财 |
| 疾病 | 四肢手脚受伤、肝胆问题 |

### 1.7 丑未冲（库冲）

共性特点：库冲 → 土被冲旺（把库中藏干冲坏，土变旺），以土为喜忌判断福祸。

| 情况 | 说明 |
|------|------|
| 本质 | 金库（丑）vs 木库（未） |
| 冲动季节 | 秋天金旺时丑可冲毁未 |
| 疾病 | 未病（胃/脾） |

### 1.8 辰戌冲（库冲）

| 情况 | 说明 |
|------|------|
| 本质 | 水库（辰）vs 火库（戌） |
| 冲动季节 | 冬天水旺时水库灭火库，戌受重伤 |
| 疾病 | 辰戌冲流年车祸应验率高 |

### 1.9 六冲核心口诀

```
用神制忌神 → 应吉（多富贵）
忌神制用神 → 应凶（勿冒进，易负债）
用神与用神交战 → 应凶
忌神与忌神交战 → 应吉（日主得利）
用神化忌神 → 应吉
忌神化用神 → 应凶
```

> 八字六冲乃至整个论命的核心。忌神制用神时，劝命主不要冒进扩张，好好工作尚不至于凄惨；用神制忌神者多非普通人，常有富贵。

### 1.10 六冲与喜忌总表

| 冲的组合 | 冲动的季节 | 疾病取向 | 为福条件 | 为祸条件 |
|----------|------------|----------|----------|----------|
| 子午冲 | 冬>夏 | 心脏问题、近视 | 子水(用)冲午火(忌) | 子水(忌)冲午火(用) |
| 巳亥冲 | 冬>夏 | 心脑血管、三高 | 亥水(用)冲巳火(忌) | 亥水(忌)冲巳火(用) |
| 寅申冲 | 秋 | 四肢受伤、头疼、肝胆 | 申金(用)冲寅木(忌) | 申金(忌)冲寅木(用) |
| 卯酉冲 | 秋 | 四肢手脚受伤、肝胆 | 酉金(用)制卯木(忌) | 酉金(忌)制卯木(用) |
| 丑未冲 | 秋 | 胃/脾问题 | 以土为喜忌 | 以土为喜忌 |
| 辰戌冲 | 冬 | 车祸 | 以土为喜忌 | 以土为喜忌 |

## 二、破

### 2.1 破的种类

破主要应用于夫妻关系、六亲关系、合作关系，也可用于财运中的破财判断。破不是所有专题里的统一最高优先级，必须落到十神和宫位上取象。

| 破的种类 | 说明 | 条件 |
|----------|------|------|
| 子酉破（子破酉） | 子水当令时才能破 | 需当令 |
| 寅亥破（寅亥合中带破） | 合中带破 | 不论季节（因是六合） |
| 卯午破 | 卯木当令时才能破 | 需当令 |
| 辰丑破（辰丑土破） | 相邻才论破 | 不论季节，但原局需挨着 |
| 巳申破（巳申合中带破） | 合中带破 | 不论季节（因是六合） |
| 未戌破（未戌土破） | 相邻才论破 | 不论季节，但原局需挨着 |

破的意象：破坏、破财、关系不好。

**原局论破**：子酉破、卯午破要看当令；寅亥破、巳申破为合中带破，见到即论；辰丑破、未戌破不看季节，但原局要相邻。

**流年大运论破**：流年或大运来破原局，不再按原局月令和位置筛选，只要见到就论破；也不强调谁破谁，主要看运年之字破了原局哪个十神或宫位。

### 2.2 破的应用

| 位置 | 表现 |
|------|------|
| 破到年月的印星 | 与母亲关系不好，争执多 |
| 破到年月的财星 | 与父亲关系不好；同时容易破财 |
| 破到财星 | 财运上主破财、花费、损耗 |
| 破到夫妻宫 | 夫妻关系不好，争吵明显；是否离婚要看原局配合 |

婚姻中不能见破就直断离婚：
- 原局没有破，只是大运流年来破夫妻宫，通常先断夫妻矛盾、争吵；原局组合差时才上升为离婚风险。
- 原局已有破、绝、冲、刑、害等伤婚结构，大运流年再引动，离婚风险明显增高。
- 夫妻星来破夫妻宫，或原局形成“星宫同破”，对婚姻影响最大，常主多段婚姻。

## 三、绝

### 3.1 绝的种类

在破绝这一组关系里，绝通常比破更严重。"绝"字面意思就是死绝、绝后，一点希望都没有了。这个比较主要用于夫妻、六亲、合作等关系伤害取象，不能机械扩展成所有地支关系、所有专题里的全局优先级。

| 绝 | 属性 | 条件 |
|----|------|------|
| 子巳绝 | 水火绝 | 子水当令或冬天时才叫子巳绝；夏天子水无力不叫绝 |
| 亥午绝 | 水火绝 | 冬天时亥水才有力量绝午火 |
| 卯申绝 | 金木绝 | 秋天时申金才敢绝卯木 |
| 寅酉绝 | 金木绝 | 秋天时酉金才能绝寅木 |

> 原则：我的力量比你强，我才有主动权去伤害你。绝要看季节力量。

### 3.2 绝的应用（按十神取象）

| 被绝的十神 | 表现 |
|------------|------|
| 印星（巳火为印）被绝 | 母亲身体不好，容易走得早 |
| 偏财被绝 | 父亲身体不好，容易走得早 |
| 女命食伤被绝 | 掉孩子，或没有该性别的小孩 |
| 绝到夫妻宫 | 一生婚姻不稳定，不是离婚就是在再婚的路上，多段婚姻 |

### 3.3 破绝与婚姻

- 破到夫妻宫：婚后矛盾多，容易争吵；原局配合差或运年引动时容易闹离婚。
- 绝到夫妻宫：婚姻稳定性差，容易离婚或多段婚姻。
- 夫妻宫与夫妻星相破或相绝，尤其“星宫同破”或“星宫同绝”，才是婚姻判断中的重结构。
