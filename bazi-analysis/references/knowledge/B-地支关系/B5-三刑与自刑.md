# 三刑与自刑

---

## 一、刑的总义

刑包括**丑未戌三刑**、**寅巳申三刑**和**自刑**（亥亥、辰辰、酉酉、午午）。

刑的本质：
1. **痛苦**——所有刑都代表痛苦，三刑与自刑皆是
2. **纠缠**——刑也代表纠缠不清的状态

> 刑的十神不同，所表达的意向和人物也不同，需结合十神取象。

---

## 二、丑未戌三刑

### 2.1 成立条件

**丑未戌三刑一定是原局就有的三个字同时存在，才算真正的三刑。**

如果原局只有未戌相刑，大运或流年来了丑，补齐三字——不叫真正意义上的三刑。寅巳申三刑同理。

> 口诀：刑必原局有，运年补齐不算真。

### 2.2 按十神取象

| 丑未戌所临十神 | 对应六亲/方面 | 表现 |
|----------------|---------------|------|
| 财星 | 父、妻（男命） | 克父克妻；易被父亲/妻子拖累；父亲认知不够做出错误抉择连累命主 |
| 印星 | 母 | 母亲身体不好；易被母亲拖累（无心之失） |
| 比劫 | 兄弟姊妹、朋友、手脚 | 无真心朋友，易被朋友坑；严重者肢体残疾 |
| 官杀 | 配偶（女命）、子女（男命）、事业、领导 | 女：克夫/被另一半坑；男：克子/子女身体差；事业一团糟；与领导关系不好 |
| 食伤 | 子女（女命）、下属（男命） | 女：掉孩子、克子；男：下属不服管束 |

### 2.3 丑未戌刑库做功

当丑未戌三刑发生时，其本质是**未戌（燥土/火木库）制丑（湿土/金库）**，意在打开丑中的金库。

- 丑中藏什么十神？看丑中对应的天干（辛金）在命局中为何十神
- 如果丑中的金是忌神，被刑制掉则好（财运不错）
- 如果丑中的金是用神，被刑制掉则不好（破财等）
- 但无论喜忌，对应的六亲都会受到刑克（如父、妻身体不好）

> **注意**：男命丑未戌为官杀时，特别容易有牢狱之灾，在不好的流年稍不留神就容易入狱。这是断牢狱的方法之一。

### 2.4 六亲健康：只看状态，不看喜忌

看六亲身体健康时，不看喜忌，只看状态——即这个字在八字中是否被泄耗太过、被克太过、被刑太过：

- 如果被刑克太过，直接断该六亲身体不好
- 看财运时才需要看喜忌

## 三、寅巳申三刑

### 3.1 总原则：看月令决定谁受伤

寅巳申三刑比丑未戌更复杂。判断要点：**看月令上是谁**，以此决定到底是谁把谁刑坏了。

### 3.2 巳火或寅木当令 → 申金受伤

| 月令 | 原理 |
|------|------|
| 巳火在月令（夏天） | 火旺 → 寅木生巳火 → 火克申金 → 申金受伤 |
| 寅木在月令（春天） | 木旺火相 → 木火一起来制申金 → 申金受伤 |

无论寅、巳在年或日时，效果相同：都是为了制申金。

### 3.3 申金当令 → 寅木受伤

| 月令 | 原理 |
|------|------|
| 申金在月令（秋天） | 金旺木死 → 申金制寅木 → 巳火来穿寅木 → 寅木受伤最严重 |

### 3.4 十神对应（以受伤的地支定）

| 受伤的地支 | 对应十神 | 男命表现 | 女命表现 |
|------------|----------|----------|----------|
| 财星 | 父、妻 | 父妻身体不好 | — |
| 印星 | 母 | 母亲身体差 | 母亲身体差 |
| 禄（比劫） | 自己 | 自身层次低、身体差 | 自身层次低、身体差 |
| 官杀 | 事业、领导、配偶 | 易坐牢、受领导排挤 | 克夫 |

### 3.5 季节影响总结

| 月令 | 寅巳申结果 | 备注 |
|------|------------|------|
| 巳火月（夏） | 申金受伤 | 寅生巳，巳克申 |
| 寅木月（春） | 申金受伤 | 木旺火相，木火制金 |
| 申金月（秋） | 寅木受伤 | 金旺木死 |

> 注意：即使在时柱，只要三刑成立，影响一致。

## 四、自刑

### 4.1 四组自刑

| 自刑 | 含义 |
|------|------|
| 亥亥自刑 | 自我纠结，水气内耗 |
| 辰辰自刑 | 自我纠缠，土气反复 |
| 酉酉自刑 | 自我较劲，金气内斗 |
| 午午自刑 | 自我折磨，火气内耗 |

自刑表示自己与自己较劲，同一气场反复内耗。

常见类象：纠结、反复、自我折磨、同类问题重复出现、某宫位类象内部不稳定。

### 4.2 自刑的用法

自刑的用法和三刑大同小异：看自刑的十神是什么，再对应相应的六亲。

| 自刑十神 | 表现 |
|----------|------|
| 财星自刑 | 男命克妻克父 |
| 印星自刑 | 克母辈 |
| 比劫自刑 | 无真心朋友，易被朋友骗 |
| 官杀自刑 | 与领导关系不好，不受待见；女命克夫 |
| 食伤自刑 | 易抑郁——三刑或自刑中，食伤受刑都是抑郁症高发人群 |

### 4.3 自刑的特殊含义

自刑多一层意思：**自我纠结、自己跟自己过不去，容易有自残倾向。**

> 食伤受刑（无论三刑还是自刑）→ 极易抑郁，此为断抑郁症的简便方法。

### 4.4 三刑与自刑的轻重比较

| 方面 | 三刑 | 自刑 |
|------|------|------|
| 牢狱 | 丑未戌三刑为官杀时，流年不好就有牢狱风险 | 自刑即使是官杀自刑，一般也不太容易有牢狱 |
| 其他事情 | 视具体问题和所刑十神而定 | 视具体问题和所刑十神而定 |

---

## 五、刑与牢狱判断

- **丑未戌三刑为官杀** → 流年不佳 → 有牢狱风险（重要断法）
- 三刑断牢狱比自刑严重

## 六、刑与抑郁症

- **食伤受刑**（无论三刑还是自刑）→ 极易抑郁
- 这是断抑郁症的简便方法
- 食伤代表表达、宣泄、自由精神，受刑则表达受阻、内心压抑

## 七、刑的核心口诀

### 用神忌神交战（适用于刑、冲等所有作用关系）

```
用神制忌神 → 应吉（好事）
忌神制用神 → 应凶（坏事）
忌神与忌神交战 → 日主得利（应吉）
用神与用神交战 → 有凶
用神化忌神 → 应吉
忌神化用神 → 应凶
```

> 这是整个八字论命的核心，不仅适用于刑，也适用于冲等其他作用关系。
