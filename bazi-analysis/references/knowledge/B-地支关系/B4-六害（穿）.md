# 六害（穿）

---

## 一、六害总论

### 1.1 六组害（穿）

六害 = 穿，是一种快速的伤害。

| 六害组合 | 属性 |
|----------|------|
| 寅巳穿 | 火木相穿 |
| 子未穿 | 水土相穿 |
| 卯辰穿 | 木土相穿 |
| 酉戌穿 | 金火相穿 |
| 申亥穿 | 金水相穿 |
| 丑午穿 | 土火相穿 |

### 1.2 害（穿）的性质

- **六害 = 穿**，二者为同一概念
- 穿的意思是伤害宫位、穿破关系、暗中损害
- **穿比冲更隐蔽**：冲是明显撞击，穿更像内部损伤、暗中破坏、关系不舒服
- **在看六亲健康时，穿的伤害是最严重的**，比刑、冲更直接、更快速

### 1.3 穿的应期

**原局带穿，逢流年大运再来穿一下，应期极明显。**

### 1.4 穿的力量判断

穿要看季节力量：
- 谁在月令上谁力量大，就能穿得动对方
- 如果都不在月令，要看在什么季节
- 原则：力量大的一方才有主动权去伤害对方

### 1.5 穿伤宫位

| 被穿宫位 | 表现 |
|----------|------|
| 穿配偶宫 | 婚恋关系容易受伤 |
| 穿财星宫位 | 财路、父亲、妻子、钱财类象容易有损 |
| 穿官星宫位 | 工作、职位、丈夫、规矩类象容易出问题 |
| 穿印星宫位 | 学业、母亲、房产、庇护类象容易不稳 |

### 1.6 专题应用分流

六害（穿）是基础原理，不可直接推出某个具体事项。必须先确认所属模块，再按专题主规则使用。

| 模块 | 使用重点 | 适用条件 | 误用提醒 |
|------|----------|----------|----------|
| 婚姻感情 | 是否穿夫妻宫、夫妻星所在宫、夫妻星与比劫同宫 | 日支、夫妻星或夫妻星所在地支参与穿害 | 不要见穿就断离婚，必须回到星宫关系 |
| 事业财运 | 是否伤到财星、财库、财路、做功用神 | 财星、财库、财官做功链或大运财星参与穿害 | 不要把健康/婚姻中的穿直接套成破财 |
| 父母子女 | 六亲星或六亲宫是否受伤 | 父母星、子女星、父母宫、子女宫参与穿害 | 六亲健康看状态，不直接看喜忌 |
| 身体健康 | 被穿五行、藏干、宫位对应身体部位 | 受伤五行明确，且大运流年引动 | 健康应象不能反推财运吉凶 |

## 二、各组穿详解

### 2.1 寅巳穿

| 项目 | 内容 |
|------|------|
| 原理 | 巳火中的庚金想制寅木 → 寅木受伤 |
| 用法 | 看寅木是什么十神，对应相应六亲 |
| 疾病取向 | 头疼病；四肢手脚筋骨容易受伤 |

### 2.2 子未穿

| 项目 | 内容 |
|------|------|
| 力量判断 | 子水在月令则未土被穿散；未土在月令则子水受伤严重 |
| 疾病取向（女性） | 1. 子宫肌瘤或卵巢囊肿（子为子宫/卵巢生殖系统）<br>2. 未病（未代表相关部位）<br>3. 近视（未中丁火为眼睛）<br>4. 容易哭鼻子、淌眼泪（癸水为眼泪，女性更明显） |

### 2.3 卯辰穿

| 项目 | 内容 |
|------|------|
| 原理 | 卯木克辰土，一般是辰土受伤 |
| 季节影响 | 春天时辰土一定受伤严重 |
| 疾病取向 | 1. 卯为肝胆 → 肝胆有问题<br>2. 卯为四肢 → 四肢筋骨易受伤<br>3. 辰为皮肤 → 皮肤问题 |

### 2.4 酉戌穿

| 项目 | 内容 |
|------|------|
| 力量判断 | 酉金在月令（秋天）时戌穿不动酉；夏天时酉金受伤严重（火旺金死） |
| 疾病取向 | 1. 牙齿不好（酉为牙齿）<br>2. 近视（戌中丁火为眼睛）<br>3. 心脏方面问题（戌为心包） |

### 2.5 申亥穿

| 项目 | 内容 |
|------|------|
| 原理 | 申金去制亥中的甲木；但因申金先生了一下壬水（化解了部分力量），相对没那么严重 |
| 疾病取向 | 头或四肢有暗疾，容易受伤（甲木为头、四肢） |

### 2.6 丑午穿

| 项目 | 内容 |
|------|------|
| 疾病取向 | 1. 近视（午中丁火为眼睛）<br>2. 牙齿不好、蛀牙/拔牙（丑中辛金为牙齿）<br>3. 女性容易哭鼻子、淌眼泪（丑中癸水为眼泪） |

| 丑午穿 | 近视、牙齿不好/蛀牙、爱哭淌眼泪 |

| 穿组合 | 月令当令方 | 受伤方 | 备注 |
|--------|------------|--------|------|
| 寅巳穿 | 寅或巳月令 | 寅木/甲木 | 巳中庚金制寅中甲木，形成金木交战 |
| 子未穿 | 子水月令 | 未土受伤 | 水胜土 |

| 酉戌穿 | 酉金月令（秋） | 戌穿不动酉 | 金旺 |
| 酉戌穿 | 夏天 | 酉金受伤 | 火旺金死 |
| 申亥穿 | — | 相对温和 | 申先生水，化解部分力量 |

## 五、寅巳穿跨模块边界

寅巳穿是基础作用，不直接等同某个断事。必须先看寅、巳在本命中的十神、宫位和用忌，再进入对应专题。

| 模块 | 可用断法 | 必须满足 | 不可误用 |
|------|----------|----------|----------|
| 财运 | 巳中庚金为财，或财星/财用神被冲合穿害时，可断赚钱机会受损、投资破财 | 财星为喜用，或大运财星是当前做功关键 | 不要只因寅巳穿就断破财 |
| 婚姻 | 巳为夫妻星所在宫，或巳穿日支夫妻宫，主婚姻不稳、聚少离多、同处易争执 | 夫妻星定位成立，且星宫发生穿害 | 不要把健康中的筋骨伤直接套成离婚 |
| 健康 | 巳中庚金伤寅中甲木，金木交战，可取头部、肢体、筋骨、胯骨、筋膜等伤病象 | 寅木/甲木为身体取象，运年再引动时应期增强 | 不要替代医疗判断 |
| 父母 | 若寅中甲木或相关印星代表母亲，被巳穿坏，可看母亲身体长期不佳或逢寅巳年应事 | 已定位印星为母亲星，且印星受穿 | 不因印为喜用就忽略受伤状态 |
