# 六合、三合局、三会局与拱合

---

## 一、地支六合

### 1.1 六组六合

| 六合 | 合化五行 | 属性分类 |
|------|----------|----------|
| 子丑合 | 化土 | 闭库之合 |
| 寅亥合 | 化木 | 破合（合中带破） |
| 卯戌合 | 化火 | 闭库之合 |
| 辰酉合 | 化金 | 闭库之合 |
| 巳申合 | 化水 | 破合（合中带破） |
| 午未合 | 化火 | 闭库之合 |

### 1.2 六合的含义

六合首先是"合"，有牵绊、结合、合作、私情、羁绊的含义。

是否合化，要看化神是否有力。

### 1.3 闭库之合与破合

#### 闭库之合（四组）

闭库的本质是把库闭掉，使库中藏干不出来：

| 闭库 | 效果 |
|------|------|
| 子丑合 | 闭丑库（金库）→ 丑中辛金、癸水出不来 |
| 午未合 | 闭未库（木库）→ 未中乙木、丁火出不来 |
| 辰酉合 | 闭辰库（水库）→ 辰中癸水出不来；辰中乙木被酉金克坏 |
| 卯戌合 | 闭戌库（火库）→ 戌中丁火出不来；卯木被戌中辛金克伤 |

> 夫妻宫逢闭库之合（如子丑合），关系糟糕，互相拖累。

#### 破合（两组）

寅亥合、巳申合属于破合——表面合作，内里算计。

| 领域 | 表现 |
|------|------|
| 夫妻关系 | 表面和谐，外人看着挺好，实际两人勾心斗角 |
| 合作关系 | 表面合伙做事，实际互相算计，各打小算盘 |

### 1.4 六合不论合化

六合与三合局不同：六合一般不论合化，没有"合化成功"的概念。六合更多体现为合作关系，按十神不同有不同解读。

但六合中暗含的伤害需留意：

| 六合 | 暗含伤害 |
|------|----------|
| 辰酉合 | 辰中乙木被酉金破坏 |
| 卯戌合 | 卯木被戌中辛金克伤 |
| 寅亥合 | 寅中丙火被亥水灭掉 |
| 巳申合 | 巳中丙火被申中壬水灭掉 |

### 1.5 六合的课堂类象

| 六合 | 类象 |
|------|------|
| 子丑合 | "心不甘情不愿""心里不痛快"；丑为玄武，有欺诈、隐秘类象；因小失大之象 |
| 寅亥合 | 救苦救难、帮扶、慈悲、修复 |
| 卯戌合 | 飞蛾扑火、被吸引、感情投入 |
| 辰酉合 | 养育、滋养、孕育、收成 |
| 巳申合 | 管教、再造、重新塑造 |
| 午未合 | 按火意理解，热烈、合聚、情绪性 |

### 1.6 六合判断方法

判断六合时需同时考虑：
- 合到哪个宫位
- 合住哪个十神
- 合神是喜是忌
- 是否化成功
- 是否被冲破
- 大运流年是否引动

例如配偶宫被外部地支合走，常与婚恋关系被外界牵动有关。

### 1.7 暗合的应用

#### 暗合与感情出轨

暗合多用于看偷情、出轨。

关键判断：**暗合出现在夫妻宫（日支）**
- 子水是夫妻宫，月上有丑或时上有丑，形成子丑暗合
- 巳火是夫妻宫，月上有申或时上有申，形成巳申暗合

> 可断：要么自己不安分，要么另一半不安分，容易出轨/偷情。
> 寅丑暗合的出轨应验率非常高。实际批命中谨慎使用。

#### 暗合与暗中求财

暗合也代表暗中合作、暗中求财：
- 用坐下的字把月上的财星"悄咪咪"搞到家里来
- 暗中投资、悄悄与别人合作
- 除了主业还偷偷搞副业挣钱，别人不知道

> 暗合的核心意象：暗地里的、别人不知道的。具体应何事要看暗合的十神是什么。

## 二、地支三合局

### 2.1 四组三合局

| 合局 | 地支组合 | 长生位 | 帝旺位（中神） | 墓库位 | 化神 |
|------|----------|--------|----------------|--------|------|
| 申子辰合水 | 申+子+辰 | 申 | 子 | 辰 | 水 |
| 寅午戌合火 | 寅+午+戌 | 寅 | 午 | 戌 | 火 |
| 亥卯未合木 | 亥+卯+未 | 亥 | 卯 | 未 | 木 |
| 巳酉丑合金 | 巳+酉+丑 | 巳 | 酉 | 丑 | 金 |

三合局由长生位、帝旺位（中神）、墓库位组成，其中中神（帝旺位）最为关键。

### 2.2 合化条件

三合局需区分**合化**（真正合成了）与**合绊**（只绊住、未合成）。

三个条件必须同时满足才叫合化成功，缺一则为合绊：

#### 申子辰合水局

| 条件 | 要求 |
|------|------|
| 季节 | 冬天（水旺）或秋天 |
| 位置 | 申、子、辰在地支必须挨着（年月、月日、日时连在一起），不可隔位 |
| 透干 | 天干透壬水或癸水（化神透干） |

#### 寅午戌合火局

| 条件 | 要求 |
|------|------|
| 季节 | 夏天（火旺）或春天（木旺火相） |
| 位置 | 寅、午、戌在地支必须挨着 |
| 透干 | 天干透丙火或丁火 |

#### 亥卯未合木局

| 条件 | 要求 |
|------|------|
| 季节 | 春天（木旺）或冬天（水旺木相） |
| 位置 | 亥、卯、未在地支必须挨着 |
| 透干 | 天干透甲木或乙木 |

#### 巳酉丑合金局

| 条件 | 要求 |
|------|------|
| 季节 | 秋天（金旺），或在辰月、丑月也可 |
| 位置 | 巳、酉、丑在地支必须挨着 |
| 透干 | 天干透庚金或辛金 |

### 2.3 合化条件总表

| 合局 | 化神 | 季节要求（按优先级） | 位置要求 | 透干要求 |
|------|------|----------------------|----------|----------|
| 申子辰合水 | 水 | 冬 > 秋 | 三字挨着 | 透壬/癸水 |
| 寅午戌合火 | 火 | 夏 > 春 | 三字挨着 | 透丙/丁火 |
| 亥卯未合木 | 木 | 春 > 冬 | 三字挨着 | 透甲/乙木 |
| 巳酉丑合金 | 金 | 秋、辰月、丑月 | 三字挨着 | 透庚/辛金 |

> 三个条件必须同时满足才叫合化；缺一则为合绊。

### 2.4 合化之后的喜忌判断

合化成功后，看合化出来的五行是喜神还是忌神：

| 合化结果 | 含义 |
|----------|------|
| 为喜神（用神） | 力量大增，有富贵之象 |
| 为忌神 | 力量大增，带来麻烦 |

### 2.5 生地半合与墓地半合

三合局缺一个字时，可能形成半合。

| 类型 | 组合 | 举例 |
|------|------|------|
| 生地半合 | 长生位+帝旺位 | 申子半合水 |
| 墓地半合 | 帝旺位+墓库位 | 子辰半合水 |

半合力量不如三合全局，但仍会增强对应五行。

## 三、地支三会局

### 3.1 四组三会局

| 三会局 | 地支组合 | 方位 | 季节 |
|--------|----------|------|------|
| 寅卯辰三会木 | 寅+卯+辰 | 东方木 | 春季 |
| 巳午未三会火 | 巳+午+未 | 南方火 | 夏季 |
| 申酉戌三会金 | 申+酉+戌 | 西方金 | 秋季 |
| 亥子丑三会水 | 亥+子+丑 | 北方水 | 冬季 |

### 3.2 三会局的含义

三会局是一个季节的气势全部聚齐，比三合更偏向季节气场、方向气场。例如寅卯辰会东方木，表示春季木气成片。

### 3.3 三会局力量

三会局成局后，对应五行的力量非常明显。**三会力量大于三合**。

判断要点：
- 会成的五行是否为命局所喜
- 是否冲克原局重要用神
- 是否让某个忌神过旺
- 是否在大运流年补齐后应事

### 3.4 拱会

原局有三会局中的两个字，缺中间或关键一字，大运流年补齐时，可能形成拱会。拱会不是原局已经成局，而是有引动条件时才显现。

## 四、拱合与拱局

### 4.1 四组拱合

| 拱合 | 拱出的中神 | 透干要求 |
|------|------------|----------|
| 生辰拱子水 | 子 → 癸水 | 天干透癸水 |
| 寅戌拱午火 | 午 → 丁火 | 天干透丁火 |
| 亥未拱卯木 | 卯 → 乙木 | 天干透乙木 |
| 巳丑拱酉金 | 酉 → 辛金 | 天干透辛金 |

### 4.2 拱合条件

1. **必须挨着**：两字在地支必须相邻（年月、月日、日时皆可），隔位则不论拱
2. **天干透阴干化神**：
   - 生辰拱子水 → 透癸水（而非壬水）
   - 寅戌拱午火 → 透丁火（而非丙火）
   - 亥未拱卯木 → 透乙木（而非甲木）
   - 巳丑拱酉金 → 透辛金（而非庚金）

> 拱合透干要求的是**阴干**（癸、丁、乙、辛），与三合局透阳干或阴干皆可不同。

### 4.3 拱合成功后的用法

| 拱出十神 | 为喜神（用） | 为忌神 |
|----------|--------------|--------|
| 财星 | 有暗财，莫名其妙的发财 | 暗中破财，不知不觉钱就没了 |
| 官星 | 暗中有领导/贵人帮忙 | 小人作祟 |
| 印星 | 贵人相助 | 小人 |
| 比劫 | 有朋友暗中帮忙 | 容易被朋友坑 |

### 4.4 拱局注意事项

1. 拱局是原局两个字形成潜在结构，不等于已经成完整三合局
2. 拱出来的字不是原局明摆着存在，表示一种潜在趋势
3. 必须等大运流年补齐、引动，才更容易显现
4. 不宜说"大运拱局"——大运是应期和引动条件，原局才是结构基础
5. 拱出的五行如果是喜神，被补齐时多有好事；是忌神则容易应凶

判断拱局需综合看：
- 缺的字是否被大运流年补上
- 原局是否有冲破
- 化神是否有气
- 拱出的五行是喜是忌
