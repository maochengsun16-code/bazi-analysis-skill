## 第0步：程序预处理

> ⚠️ **排盘环节由外部程序完成**，输入性别和出生时间，直接得出四柱八字和大运。
> 程序还应先生成十神体系和基础标记。命理分析只在这些固定结果完成后开始。

### 0.1 程序应输出的固定结果

外部程序应输出以下信息：

```
性别：男 / 女
八字：年柱 月柱 日柱 时柱
      (天干地支) × 4
日主：XX（第3个字的天干）
月令：XX（第2个字的地支）
月令十神：XX（月支本气相对日主的十神）
十神矩阵：四天干、四地支藏干全部对应十神
根气摘要：明现比劫 / 明现印 / 对应库 / 无根
寒燥标记：冬令寒重 / 夏令燥重 / 平
七杀标记：有无七杀，七杀是否在月令或多处出现
四墓库标记：辰戌丑未的位置、藏干、空间（年月/日时）
大运：从X岁起运，每十年一大运
      XY运 → XY运 → XY运 ...
```

### 0.2 确认四柱结构

检查八字是否为标准的四柱八格式：

| 位置 | 天干 | 地支 | 五行 | 含义 |
|---|---|---|---|---|
| 年柱 | × | × | × | 祖辈、家世 |
| 月柱 | × | × | × | 月令、环境 |
| 日柱 | ×(日主) | ×(夫妻宫) | × | 自己、配偶 |
| 时柱 | × | × | × | 子女、晚年 |

## 第2步：定月令十神，判环境定调

### 2.1 月令是什么

月令 = 月柱的地支。它是一个八字中**能量最强、权柄最大**的字。

月令的作用不是简单判日主的"得令/失令"，而是：
1. **定调全局环境** → 月令决定了八字中**哪个十神/五行的能量最强**
2. **取用神的起点** → 制化月令这个最旺十神/五行为普通取用的核心思路
3. **地支关系的裁判** → 看刑冲合害破绝时，谁在月令谁的力量大，谁有主动权

程序必须先给出“月令十神”，也就是月支本气相对日主的十神。后续定格局时，不只说“子月水旺”，还要说“子水对日主是正印/正财/七杀等”，因为月令是原局最强十神。

### 2.2 月令五行与四季

| 季度 | 地支月 | 月令最旺五行 |
|---|---|---|
| 春季 | 寅、卯、辰 | 木旺 |
| 夏季 | 巳、午、未 | 火旺 |
| 秋季 | 申、酉、戌 | 金旺 |
| 冬季 | 亥、子、丑 | 水旺 |
| 四季末 | 辰、未、戌、丑（土月） | 土旺（但有两层做功） |

### 2.3 月令定调 → 为主的事

月令定下全局基调后，整个八字的走向由它主导：

| 月令五行旺 | 全局基调 | 取用神方向 |
|---|---|---|
| 印旺（木日主生亥子丑月等） | 印星为全局最强能量 | 制化印星——用财制 / 比劫化 |
| 食伤旺（火日主生巳午未月等） | 食伤为全局最强能量 | 制化食伤——用印制 / 官杀生印 |
| 财旺（土日主生亥子丑月等） | 财星为全局最强能量 | 制化财星——用比劫制 / 印生身 |
| 官杀旺（金日主生巳午未月等） | 官杀为全局最强能量 | 制化官杀——用印化 / 食伤制 / 比劫扛 |
| 比劫旺（水日主生亥子丑月等） | 比劫为全局最强能量 | 制化比劫——用官杀 / 财星 / 食伤 |
| 四墓库月（辰戌丑未） | 两层做功 | 土本身做功 + 透出之物做功 |

### 2.4 月令与日主强弱无关

> ⚠️ **实战派的核心认知**：月令旺什么，不代表日主就"得令强"或"失令弱"。
>
> 日主能不能扛事，不取决于是否得令，而取决于**在地支有没有根气**。
> 根气只看三类：地支明现比劫、地支明现印、地支有对应库。中气根、余气根不算有根气，只作藏干线索。
> 戊己土特殊：辰戌丑未为土之比劫根；巳午火是戊己土的印，也可兼看成戊己土的比劫助力。
> 只要满足以上三类根气之一，
> 日主就能**担财官、用食伤**，就有作为的能力。
>
> 月令的意义在于告诉你看盘的人：**这个局里哪个五行气势最强，你要怎么应对它**。
> 而不是纠结于"日主得令所以身强、失令所以身弱"这个过时的二分法。
>
> 具体日主能否担财官、用食伤，交由 **第5步（判旺衰，看根气）** 来判断。

## 第3步：特殊优先级扫描

> 在按月令十神定普通格局之前，先检查三类高优先级特殊分支：库做功、强七杀、寒燥调候。

### 3.1 四墓库做功优先

如果原局中有辰戌丑未，并且形成开库、闭库、冲库、刑库、拱开或家内外制化做功，则先分析库做功。库做功在财富层次、格局层次上优先级高于普通月令格局。

处理顺序：
1. 标记辰戌丑未位置：年月为家外，日时为家里。
2. 判断开闭库。
3. 判断家里制外面，还是外面制家里。
4. 再回到月令和普通格局。

### 3.2 有杀先论杀

如果七杀在月令、透干有力，或原局多处出现，则先处理七杀。先看印化杀，再看食伤制杀，再看比劫扛杀。七杀处理完之后，再看普通月令格局。

### 3.3 寒燥调候

寒燥属于原局偏枯问题。原则：
- 月令在亥、子、丑，先按寒重看，火为第一调候用神。
- 月令在巳、午、未，先按燥重看，水为第一调候用神。
- 调候用神不能只看“需要”，还要看原局有没有、是否透出、是否有根、是否被冲合刑害破坏。

## 第4步：定格局

> 十神矩阵已由程序生成，本步不再手工排十神，而是基于十神矩阵判断格局。

### 4.1 格局判断顺序

1. 先看高优先级特殊分支是否已经覆盖普通格局：库做功、强七杀、寒燥调候。
2. 再看月令十神：财、官杀、印、食伤、比劫禄刃、四墓库月。
3. 再看组合成格：杀印相生、食伤制杀、伤官配印、财格用印比、印旺用财、禄刃用官杀等。
4. 最后看破格条件：用神被坏、忌神无制、用神虚透无根、用神与用神交战等。

### 4.2 十神基础公式

以日主为中心，四柱中所有天干和地支藏干都与日主发生关系：

| 关系 | 正神 | 偏神 |
|---|---|---|
| 生我者 | 正印（异性） | 偏印（同性） |
| 我生者 | 食神（同性） | 伤官（异性） |
| 克我者 | 正官（异性） | 七杀（同性） |
| 我克者 | 正财（异性） | 偏财（同性） |
| 同我者 | 比肩（同性） | 劫财（异性） |

### 4.3 速查表（以甲木为例）

| 天干 | 十神 | 天干 | 十神 | 地支藏干 | 十神 |
|---|---|---|---|---|---|
| 甲 | 比肩 | 乙 | 劫财 | 寅中甲 | 比肩 |
| 丙 | 食神 | 丁 | 伤官 | 巳中丙 | 食神 |
| 戊 | 偏财 | 己 | 正财 | 未中己 | 正财 |
| 庚 | 七杀 | 辛 | 正官 | 申中庚 | 七杀 |
| 壬 | 偏印 | 癸 | 正印 | 亥中壬 | 偏印 |

> 将四柱八个字（天干）和地支藏干（本气/中气/余气）逐一转换成十神，构成完整的十神网络。

### 4.4 五善神与五恶神

| 善神 | 恶神 |
|---|---|
| 正印 → 护身、安稳 | 偏印 → 另类、自私 |
| 正财 → 稳定财源 | 偏财 → 横财、风险 |
| 正官 → 规则、约束 | 七杀 → 暴力、压力 |
| 比肩 → 帮身、同辈 | 劫财 → 争夺、损耗 |
| 食神 → 福气、口福 | 伤官 → 叛逆、技术 |

> 善神多不一定好，恶神多不一定坏——关键看**喜忌**。
> 恶神如果被制住（为用神），常常成就大事业。
> 善神如果成为忌神且无制，也会带来麻烦。

### 4.5 观十神整体格局

列出四柱的全部十神后，先做一个粗略评估：
- 哪种十神最多？（印多？官杀多？比劫多？）
- 哪种十神最少或没有？
- 有没有特殊组合？（官杀混杂、枭神夺食、伤官见官等）

## 第5步：判旺衰，看根气

> 判断日主能否担财官、用食伤，核心看**根气**。
> 传统三要素中的"得令"仅供参考，不是决定性因素。
> **只要地支有根气，不分得令失令，日主就能发挥作用。**

### 5.1 日主担事能力评估

实战派的核心判断维度：

| 判断维度 | 内容 | 权重 |
|---|---|---|
| 根气（绝对核心） | 地支有无明现比劫、明现印、对应库 | **最高 — 有根就能用** |
| 得生 | 天干或地支有无印星生扶 | 辅助 |
| 得令 | 日主是否生于当旺之月（传统参考） | 最低 — 供参考即可 |

### 5.2 根气判断

**正式有根气只认三类**：

| 日主 | 明现比劫根 | 明现印根 | 对应库根 |
|---|---|
| 甲/乙木 | 寅/卯 | 亥/子 | 辰/未 |
| 丙/丁火 | 巳/午 | 寅/卯 | 戌/未 |
| 戊/己土 | 辰/戌/丑/未（巳午可兼看助力） | 巳/午 | 辰/戌/丑/未 |
| 庚/辛金 | 申/酉 | 辰/戌/丑/未 | 丑/戌 |
| 壬/癸水 | 亥/子 | 申/酉 | 辰/丑 |

> **中气根、余气根不算有根气**。比如某五行只藏在中气或余气里，不可直接断“日主有根能担财官”。只有该地支本身属于明现比劫、明现印或对应库时，才按根气论。
> 戊己土以辰戌丑未为比劫根；巳午火既是戊己土的印，也可兼看成戊己土的比劫助力。

### 5.3 十二长生辅助参考

日主对应的十二长生状态可以辅助判断：

| 地支状态 | 对应旺衰 |
|---|---|
| 临官（禄）、帝旺（刃） | 最强 |
| 长生、冠带 | 强 |
| 衰、病 | 弱 |
| 死、墓、绝 | 最弱 |

> 注意：十二长生只是状态参考，不单独断吉凶。

### 5.4 综合判断结论

| 根气情况 | 结论 | 能否担财官/用食伤 |
|---|---|---|
| 地支明现比劫 | 有根气，有做事能力 | ✅ 能担能用的标志 |
| 地支明现印星生扶 | 有根气，有助力 | ✅ 可以 |
| 地支有库根（辰戌丑未中的对应库） | 有库根，有储备力量 | ✅ 可以（力量介于中弱之间） |
| 只有中气或余气 | 不算有根气，只作藏干线索 | ❌ 不可直接断能担财官 |
| 地支完全没有根气（无明现比劫、无印、无库） | 无根，力量偏弱 | ❌ 需谨慎，需大运补根 |
| 无根 + 无生扶 + 失令 | 从格倾向（从旺或从弱） | 按从格论，不按常规担财官 |

> **关键认知**：日主有根气（不在乎是否得令、身强身弱），就能担财官、用食伤。
> 身强身弱不是吉凶标准，全局的配合才是。
> 比如身弱但食伤制杀有力，同样可以成就大事业。

> 🔗 **详见**：[D2-透干根气与强弱判断](D-格局与判旺衰/D2-透干根气与强弱判断.md)

## 第6步：取用神，定喜忌 ⭐ 核心

> 一步定生死。取对了用神，后续所有判断才有依据。
> **总原则**：制化八字中最旺五行的那个字即为用神。

### 6.1 按最旺五行分类取用

#### 印旺（月令为印星）
| 策略 | 方法 | 说明 |
|---|---|---|
| 用财制印 | 财星克印星 | 效率最高 |
| 用比劫化印 | 比劫泄印生食伤 | 间接化解 |

- 土月（辰戌丑未）：注意土有两层做功
- 夏天：未戌帮火不可用，辰丑可用
- 冬天：辰丑帮水不可用，未戌可用

#### 食伤旺（月令为食伤）
| 策略 | 方法 | 说明 |
|---|---|---|
| 用印制食伤 | 印星直接制食伤 | 首选 |
| 官杀生印 | 官杀生印来制食伤 | 间接路线 |
| 财星❌ | 食伤生财更泄身 | 不可用 |

#### 财旺（月令为财星）
| 策略 | 方法 | 说明 |
|---|---|---|
| 用比劫制财 | 比劫帮身抗财 | 首选 |
| 用印生身 | 印星生扶日主 | 可辅助 |
| 官杀❌ | 财生官杀更耗身 | 不可用 |

#### 官杀旺（月令为官杀）
| 层次 | 策略 | 方法 |
|---|---|---|
| 最高 | 逢官看印，用印化杀 | 印星化官杀生身 |
| 中等 | 食伤制官杀 | 食伤克制官杀 |
| 最低 | 比劫扛官杀 | 比劫拼死抵抗 |

> 原则：**有杀先论杀**。见到原局七杀旺，优先处理七杀。
> 印化杀效率最高，食伤制杀次之，比劫扛杀最次。

#### 比劫旺（禄刃月）
| 策略 | 方法 |
|---|---|
| 用官杀制比劫 | 官杀克制比劫 |
| 用财星耗比劫 | 财星消耗比劫 |
| 用食伤泄比劫 | 食伤泄比劫生财 |

#### 四墓库月（辰戌丑未）
- **两层做功**：土本身做功 + 土中透出的五行做功
- 库中五行一点没透 → 直接当土看
- 库中五行有透出 → 两层都要看

### 6.2 特殊取用原则

| 原则 | 内容 |
|---|---|
| 有杀先论杀 | 原局七杀旺，先处理七杀再考虑其他 |
| 逢官看印 | 正官当令时，先看有无印星化官 |
| 真神得用 | 原局就有该专题用神，大运再次出现→对应专题高峰 |
| 假神 | 原局没有，大运来了也有用，但效果有限 |

### 6.3 喜忌结论

确定用神后，明确以下清单：

```
✅ 用神（喜神）：【XX、XX】→ 这些就是命主的"好"五行
❌ 忌神：【XX、XX】→ 这些就是命主的"坏"五行
📌 调候：过于偏枯的八字需要调候（如冬天火少则喜火）
```

> 注意：用神制忌神 → 应吉；忌神制用神 → 应凶。
> 这条口诀贯穿整个八字论命过程。

## 第7步：看地支关系 —— 刑冲合害破绝 ⭐

> 这是实战派体系最核心的部分。地支关系决定了命主一生的吉凶起伏。
> 所有地支关系遵循同一个原则：**看力量**——谁在月令谁力量大，强者才有主动权。

### 7.1 地支关系概述

| 作用 | 性质 | 伤害度 | 速查 |
|---|---|---|---|
| 六合 | 合作、暗合 | 安全 | 子丑/寅亥/卯戌/辰酉/巳申/午未 |
| 三合 | 力量加强 | 中性 | 申子辰/亥卯未/寅午戌/巳酉丑 |
| 六冲 | 对撞、交战 | 强 | 子午/丑未/寅申/卯酉/辰戌/巳亥 |
| 破 | 破坏、破财 | 中 | 子酉/寅亥/卯午/辰丑/巳申/未戌 |
| 害（穿） | 快速伤害 | 很强 | 寅巳/子未/卯辰/酉戌/申亥/丑午 |
| 三刑 | 痛苦、纠缠 | 很强 | 丑未戌/寅巳申/自刑 |
| 绝 | 死绝、无望 | 最强 | 子巳/亥午/卯申/寅酉 |

### 7.2 六合分析

**闭库之合**（闭掉对应库）：
- 子丑合 → 闭丑中金库
- 午未合 → 闭未中木库
- 辰酉合 → 闭辰中水库（同时辰中乙木被酉金坏）
- 卯戌合 → 闭戌中火库（同时卯木被戌中辛金坏）

**破合**（表面合作，内里算计）：
- 寅亥合 → 合中带破
- 巳申合 → 合中带破

**暗合**（偷情、暗中求财）：
- 夫妻宫逢暗合（尤其寅丑暗合）→ 易出轨
- 比劫暗合夫妻宫 → 另一半出轨

### 7.3 六冲分析

**火力原则**：谁在月令谁力量大，冲得动对方。

| 冲 | 旺方 | 应事 |
|---|---|---|
| 子午冲 | 冬→子旺；夏→午旺 | 心脏、眼睛（午）、神志（子） |
| 巳亥冲 | 冬→亥旺；夏→巳旺 | 心脑血管 |
| 寅申冲 | 秋→申旺 | 四肢、肝胆（寅） |
| 卯酉冲 | 秋→酉旺 | 四肢、肝胆（卯） |
| 丑未冲/辰戌冲 | 看季节 | 库冲土旺，车祸（辰戌） |

### 7.4 六害（穿）分析 —— 健康六亲中伤害直接

看六亲健康时，穿是很直接的伤害，比普通冲更隐蔽也更容易落到身体部位；不要把它机械套成所有专题里的最高优先级。

| 穿 | 主要疾病/应象 |
|---|---|
| 寅巳穿 | 头疼、手脚筋骨损伤 |
| 子未穿 | 女命子宫肌瘤/卵巢囊肿、近视、易哭 |
| 卯辰穿 | 肝胆问题、四肢受伤、皮肤问题 |
| 酉戌穿 | 牙齿不好、近视、心脏问题 |
| 申亥穿 | 头或四肢暗疾 |
| 丑午穿 | 近视、牙齿不好、女命易哭 |

### 7.5 三刑分析 —— 痛苦与纠缠

**丑未戌三刑**（最凶）：
- 原局自有才算（大运流年补上不算真）
- 财星三刑 → 克父克妻
- **官杀三刑** → 牢狱风险极高
- 食伤三刑 → 抑郁症高发

**寅巳申三刑**：
- 夏天/春天→申金受伤
- 秋天→寅木受伤

**自刑**（亥亥/辰辰/酉酉/午午）：
- 自我纠结、自己跟自己过不去
- 自残倾向
- 食伤自刑 → 抑郁

### 7.6 破绝分析

- **破**：主要用于夫妻、六亲、合作，也可用于财运破财。破到财星主破财；破到夫妻宫主争吵、关系破损，是否离婚要看原局配合和运年引动。
- **绝**：主要用于夫妻、六亲寿元、关系断绝。在破绝同类取象中通常比破重；绝入夫妻宫主婚姻稳定性差，夫妻星绝到夫妻宫时影响更重。

> 破绝不能脱离专题机械比较强弱。看财运先看是否破财，看婚姻先看星宫关系，看六亲先看六亲星状态。

### 7.7 核心口诀体系

```
用神制忌神 → 应吉（多富贵）
忌神制用神 → 应凶（易负债、招灾）
用神与用神交战 → 应凶（都是好的一面打架）
忌神与忌神交战 → 应吉（日主得利）
用神化忌神 → 应吉
忌神化用神 → 应凶
```

> 这六句口诀是整个八字论命的核心，不仅适用于地支关系，也适用于十神组合和大运流年判断。

## 第8步：看四墓库开闭与做功

> 如果八字中没有辰戌丑未，本步可跳过。
> 四墓库是八字财富层次判断的**最高优先级做功方式**。

### 8.1 识别四墓库

八字地支中的辰、戌、丑、未即为四墓库。

每个库有双重属性：

| 地支 | 本气 | 藏干 | 主库 | 次库 |
|---|---|---|---|---|
| 辰 | 戊土 | 乙木、癸水 | 水库 | 木库 |
| 戌 | 戊土 | 辛金、丁火 | 火库 | 金库 |
| 丑 | 己土 | 癸水、辛金 | 金库 | 水库 |
| 未 | 己土 | 丁火、乙木 | 木库 | 火库 |

### 8.2 判断开库还是闭库

**开库方式**（满足任一即为开）：
1. **天干透出**：库中藏干在天干出现
2. **地支出现**：库中五行的同类在地支出现
3. **冲库**：辰戌冲、丑未冲（两库全开）
4. **刑库**：丑戌刑、未戌刑、丑未戌三刑（两库全开）
5. **拱开**：相邻地支拱合（申辰拱子、寅戌拱午等）

**闭库方式**：
| 六合 | 闭的库 |
|---|---|
| 辰酉合 | 闭辰库（水库） |
| 卯戌合 | 闭戌库（火库） |
| 子丑合 | 闭丑库（金库） |
| 午未合 | 闭未库（木库） |

**半开半闭**：天干透出 + 地支有六合 → 门开了条缝，依然按开库论。

### 8.3 库做功 —— 发财/破财判断（核心应用）

**空间定位**：
- 年月 = 家外/社会（别人的、公共的）
- 日时 = 家里（自己的、私有的）

**发财原理**：**家里的比劫库制年月的财库** → 发财
- 如：日时有未（比劫库），年月有丑（财库），丑未冲 → 家里的比劫库冲开外面的财库 → 发财

**破财原理**：**年月的比劫库制家里的财库** → 破财
- 如：年月有丑（比劫库），日时有未（财库），丑未冲 → 外面的比劫库冲开家里的财库 → 破财

### 8.4 各日主看四库的十神

快速定位（以甲乙木日主为例）：

| 地支 | 甲乙木日主 | 丙丁火日主 |
|---|---|---|
| 辰 | 印库 + 比劫库 | 官杀库 + 印库 |
| 戌 | 官杀库 + 食伤库 | 比劫库 + 财库 |
| 丑 | 官杀库 + 印库 | 财库 + 官杀库 |
| 未 | 比劫库 + 食伤库 | 印库 + 比劫库 |

> 注意：每个库包含两种五行，对同一天干对应两种十神。
> 四墓库月的八字有两层做功（土本身 + 透出之物），相对复杂。

> 🔗 **详见**：[D3-四墓库专题](D-格局与判旺衰/D3-四墓库专题.md)

## 第9步：看神煞辅助

> ⚠️ **神煞不能单独定吉凶**。使用顺序：**旺衰 > 喜忌 > 格局 > 宫位 > 合冲刑害 > 神煞**。

### 9.1 常用神煞速查

**查神煞前先确认**：该神煞对应的十神是喜神还是忌神。神煞在同一组（如桃花）中，若是喜神则锦上添花，若是忌神则雪上加霜。

| 文昌贵人 | 学习、考试、文才 | 学习能力强 |
| 天医 | 医学、健康 | 对养生医药有兴趣 |
| 将星 | 领导力、掌控力 | 有号召力和组织能力 |
| 禄神 | 俸禄、体面 | 饭碗稳定、收入体面 |
| 金舆 | 车舆、地位 | 面子、身份象征 |

**凶神/中性的神煞**：

| 神煞 | 象征 | 喜用则 | 忌则 |
|---|---|---|---|
| 魁罡 | 刚强、霸道 | 有魄力、能成事 | 易惹是非 |
| 驿马 | 走动、变化 | 行动力强 | 奔波劳碌 |
| 阳刃 | 刚烈、争斗 | 勇武有魄力 | 血光之灾 |
| 桃花 | 异性缘 | 有魅力、人缘好 | 烂桃花、感情乱 |
| 空亡 | 虚无、落空 | 看破放下 | 所得落空 |
| 华盖 | 清冷、孤高 | 有艺术玄学天赋 | 孤独、不合群 |
| 劫煞 | 聪明、有想法 | 机智敏锐 | 易遭劫难 |
| 灾煞 | — | 警示作用 | 易有意外灾害 |
| 孤辰寡宿 | 孤独 | 独立、能独处 | 婚缘淡、孤独终老 |

### 9.2 神煞应用要点

1. **不看多而看准**：挑2-3个最明显的就够，不必全部罗列
2. **结合宫位**：桃花在夫妻宫→感情丰富；空亡在财宫→财运易落空
3. **结合喜忌**：桃花是喜神→异性缘好且受益；桃花是忌神→烂桃花缠身

### 10.1 婚姻感情判断

#### 夫妻宫与夫妻星

| 概念 | 定义 |
|---|---|
| 夫妻宫 | 日柱的地支（日支） |
| 夫妻星 | 与日柱发生作用的财星或官杀星；男命财星优先，女命官杀优先，但财官都可代表配偶 |

**判断逻辑**：先按性别优先级找财/官杀，再检查是否与日柱发生作用。天干看与日干合冲；地支看与日支夫妻宫合、六合、半合、冲、害、会、半会、破、生。若财官完全不与日柱发生作用，不定夫妻星，只按桃花、短缘或露水情缘看。财官都不成立时，才降级看其他与日柱发生作用的十神，尤其六合到夫妻宫者。

#### 婚姻稳定性

| 夫妻星与夫妻宫的关系 | 婚姻结果 |
|---|---|
| 合进来（天合地合） | ✅ 最稳定，一婚到头 |
| 天干五合（地支无冲克） | ✅ 比较稳定 |
| 半合/半会 | 可成关系，但不稳定 |
| 冲（星宫交战） | ❌ 二婚命 |
| 害（穿） | ❌ 也会离婚 |
| 刑进夫妻宫 | ❌ 痛苦、不稳定 |
| 破到夫妻宫 | ❌ 关系破损，易闹离婚 |
| 生入夫妻宫 | ❌ 不稳定 |
| 绝入夫妻宫 | ❌ 一生婚姻极不稳定 |

#### 配偶出轨判断

- 夫妻星和比劫同宫同柱 → 另一半会出轨（必断）
- 夫妻星被比劫合到 → 出轨
- 比劫合到夫妻宫 → 百分百出轨
- 暗合出现在夫妻宫 → 不安分

#### 结婚/离婚时间

| 事项 | 判断方法 |
|---|---|
| 结婚时间 | 适婚年龄段，夫妻星到位，或合夫妻宫的地支出现的年份 |
| 离婚时间 | 男怕劫财，女怕伤官。伤官同时坏掉夫妻宫和夫妻星的年份 = 离婚年 |

### 10.2 财运与财富层次

#### 财富层次判断

| 层次 | 条件 |
|---|---|
| 过亿级 | 年上的财官被我用神制到家里 |
| 千万级 | 月上的财官被我用神制到家里 |
| 百万级 | 食伤生财、或日时的财被制到 |

**做功方式排序**：
1. 库做功（最高优先级） → 有大格局
2. 印来化（生过来） → 别人给钱（父母、贵人）
3. 食伤制官杀 → 自己干（最累）

#### 大运发财时间

- 原局有制财官的字（即使这个字是忌神），走到该字的大运 → 发财
- 真神得用（原局用神在大运再次出现）→ 对应专题高峰
- 化忌为用（忌神生用神且用神得力）→ 大富

#### 财星虚实

- 地支的财 = 实财（实实在在的钱）
- 天干的财 = 虚财（要有天地一气才做实）
- 年上财官天地一气为喜用 → 亿级（比劫除外）

### 10.6 大运流年判断

#### 大运总原则

| 原则 | 内容 |
|---|---|
| 重地支 | 整个十年都看地支，不分前五年后五年 |
| 月令为大 | 大运来了先和月令作用，决定当前专题的大体吉凶 |
| 四句核心口诀 | 贯穿始终 |

#### 大运流年分专题核心口诀

```
忌神生用神 → 当前专题应吉
用神生忌神 → 当前专题应凶
用神交战 → 应凶
忌神交战 → 应吉
```

#### 大运作用于命局的三种路径

| 路径 | 表现 |
|---|---|
| 生助忌神 | 大运生旺了当前专题忌神 → 该专题不利 |
| 合出忌神 | 大运与原局合化，化出的是当前专题忌神 → 该专题不利 |
| 用神生忌神 | 大运用神却生了原局忌神 → 好事变坏事 |

#### 流年判断

| 场景 | 规律 |
|---|---|
| 当前专题好大运中 | 该专题差年份只是减量或波动，不直接翻车 |
| 当前专题坏大运中 | 该专题好年份只是相对好一点，坏年份可能爆雷 |
| 流年顺序 | 流年先和大运发生作用，再作用于原局 |

#### 墓库运特殊看法

- 走辰戌丑未运时，看天干透出的字决定该库属性
- 透出实还是虚透，定吉凶

#### 重要概念

| 概念 | 含义 |
|---|---|
| 真神得用 | 原局有此用神，大运再次出现→对应专题高峰；若用于财运，才看事业财富层面的阶层跨越 |
| 化忌为用 | 忌神生了用神→表面忌神但结果好 |
| 忌神虚透 | 天干透忌神但地支无根→危害不大反而应吉 |
| 用神虚透 | 天干透用神但地支无根→力量不够反而应凶 |

## 附：分析优先级速查

```
第一优先级：取用神 → 用神制忌神则为吉，忌神制用神则为凶

第五优先级：大运流年 → 好命还要看对应专题是否走到好大运
第六优先级：神煞 → 锦上添花用的，不要反客为主
```
