# 八字命理知识库 · 实战派体系

> 本知识库基于实战派老师八字讲义整理，以实战为导向，系统梳理了从入门到断命的完整知识体系。
> 原始资料存放在 `/基础知识整理/实战派/` 目录，本库为清洗、去重、整合后的结构化版本。

## 📚 知识结构总览

```
bazi-reader/
│
├── A-入门基础/           ← 阴阳五行、天干地支、宫位结构、天干关系
├── B-地支关系/           ← 六合/三合/冲/刑/害/破/绝/墓库（核心中的核心）
├── C-十神体系/           ← 印/食伤/官杀/财/比劫及组合关系
├── D-格局与判旺衰/      ← 十二长生、根气强弱、四墓库、取用神
├── E-神煞体系/           ← 各类神煞查法及使用原则
├── F-实战专题/           ← 大运流年、婚姻、财运、子女父母、灾祸、职业
├── rules/                ← 既有经验规则
└── G-八字全流程.md       ← 📍 看八字完整流程（将散落知识点串联为操作步骤）
```
