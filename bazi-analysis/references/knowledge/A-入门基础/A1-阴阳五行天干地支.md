# 阴阳五行天干地支

## 一、五行基础

### 五行的基本内容

五行包括：**木、火、土、金、水**。天干与地支都是从五行系统中演化出来的基础符号。

### 五行相生

五行相生是指一个五行能量可以滋生、助长另一个五行能量，依次循环：

```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

### 五行相克

五行相克是指一个五行能量可以制约、克制另一个五行能量，依次循环：

```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

### 五行与五常

| 五行 | 五常 | 对应品性 |
| --- | --- | --- |
| 木 | 仁 | 仁义、仁爱 |
| 火 | 礼 | 礼仪、礼貌 |
| 土 | 信 | 诚信、信用 |
| 金 | 义 | 义气、道义 |
| 水 | 智 | 智慧、谋略 |

### 地支的复杂性

地支比天干更为复杂，因为地支不仅具有自身五行属性，还涉及：

- **季节**：代表出生月份和五行能量旺衰
- **温度**：巳午未为热，亥子丑为冷
- **方位**：东西南北各有所属
- **藏干**：地支内部藏有其他天干能量
- **墓库**：辰戌丑未为四墓库，具有双重属性
- **旺衰**：不同季节能量强弱不同

## 四、天干地支的阴阳关系

### 五行的阴阳分法

除了天干地支各自有阴阳分类外，五行本身也有阴阳划分：

- **木火为阳**：主动、向上、向外、温热
- **金水为阴**：主静、向下、向内、寒凉
- **土居中**：调和阴阳，承载四方

此分法从能量特性出发：木生火，火性温热升腾，为阳；金生水，水性寒凉沉降，为阴。

### 天干阴阳汇总

| --- | --- |
| 阳干 | 甲木、丙火、戊土、庚金、壬水 |
| 阴干 | 乙木、丁火、己土、辛金、癸水 |

## 五、日主概念简介

### 什么是日主

日主，又称**日元**，是四柱八字中日柱上面的天干。

日柱由上方的天干和下方的地支组成，其中**日干就是日主**，代表命主自身，也代表命主的个人五行属性。

### 日主与五行属性

| 日主 | 五行属性 | 命主类型 |
| --- | --- | --- |
| 甲 | 阳木 | 甲木之人 |
| 乙 | 阴木 | 乙木之人 |
| 丙 | 阳火 | 丙火之人 |
| 丁 | 阴火 | 丁火之人 |
| 戊 | 阳土 | 戊土之人 |
| 己 | 阴土 | 己土之人 |
| 庚 | 阳金 | 庚金之人 |
| 辛 | 阴金 | 辛金之人 |
| 壬 | 阳水 | 壬水之人 |
| 癸 | 阴水 | 癸水之人 |

例：日主为甲，即为甲木之人；日主为丙，即为丙火之人，以此类推。

### 日主的意义

日主是八字判断的核心，所有十神关系、五行生克、旺衰强弱等分析，均以日主为中心展开。日主确定了命主的五行属性后，才能进一步分析与其相关的生克制化、六亲十神等关系。
