# 天干关系合集

## 一、天干生克基础

### 五行生克法则

天干之间的生克关系以五行生克为基础：

**五行相生：**
```
木生火 → 火生土 → 土生金 → 金生水 → 水生木
```

**五行相克：**
```
木克土 → 土克水 → 水克火 → 火克金 → 金克木
```

### 天干相克：同性相克与异性相克

天干相克时，需要区分阴阳同性和阴阳异性，不同的阴阳属性导致克的力量和性质不同。

**阴阳划分：**
- 两个天干均为阳，则为**同性**
- 两个天干一阴一阳，则为**异性**

#### 同性相克

| 特点 | 说明 |
| --- | --- |
| 力量 | 力量较大，克得更重 |
| 性质 | 激烈、无情、对抗性强 |
| 十神对应 | 多表现为七杀式压力 |
| 举例 | 甲木克戊土、庚金克甲木 |

#### 异性相克

| 特点 | 说明 |
| --- | --- |
| 力量 | 力量相对温和 |
| 性质 | 更像正当约束、管理、规矩 |
| 十神对应 | 多表现为正官式管制 |
| 举例 | 辛金克甲木、乙木克戊土 |

### 天干相生：同性相生与异性相生

天干相生时，同样需要区分阴阳：

#### 异性相生

| 特点 | 说明 |
| --- | --- |
| 力量 | 生力较大，生得更有情 |
| 效果 | 接受方更容易吸收滋养 |
| 举例 | 癸水生甲木（阴生阳） |

#### 同性相生

| 特点 | 说明 |
| --- | --- |
| 力量 | 生力相对较小 |
| 效果 | 帮助存在，但不如异性相生有情 |
| 举例 | 壬水生甲木（阳生阳） |

### 综合判断原则

天干生克不能只看单个关系，需要综合考量以下因素：

1. **五行关系**：生我、我生、克我、我克
2. **阴阳属性**：同性还是异性
3. **位置远近**：相邻相克力量大，隔位力量减弱
4. **是否被合住**：被合的天干作用力减弱
5. **牵制关系**：是否被其他天干牵制
6. **喜忌判断**：该天干在命局中是喜神、用神、忌神还是闲神

单看"某干克某干"不能直接下结论，必须结合整体旺衰和喜忌，综合分析所有的相互关系。

## 二、天干五合

### 五合组合

天干五合是十天干之间两两相合的特定组合，共有五组：

| 相合组合 | 化神（合化后的五行） |
| --- | --- |
| **甲己合** | 化土 |
| **乙庚合** | 化金 |
| **丙辛合** | 化水 |
| **丁壬合** | 化木 |
| **戊癸合** | 化火 |

### 合与化的区别

**合不等于化**，这是核心要点。

| 概念 | 含义 |
| --- | --- |
| **合** | 两个天干相互吸引、纠缠、合作、被拴住，注意力被拉走 |
| **化** | 合化成功，两个天干实际转变成另一种五行力量 |

### 合化的条件

天干合化通常需要满足以下条件：

1. **位置邻近**：两个相合天干最好相邻，距离越近合化力量越强
2. **化神有根**：化神对应的五行在地支有根基支持
3. **月令支持**：化神在月令中处于旺相状态
4. **地支配合**：地支整体气势不破坏化神，最好能生扶化神

**举例说明：**
- 甲己合土：若地支土旺，有辰戌丑未等土根，合化土的可能性较强
- 若环境不支持（如月令木旺克土），则只是合住，不一定化

### 合的实际判断意义

天干被合住后，其原本的作用会减弱或改变：

| 情况 | 影响 |
| --- | --- |
| 忌神被合 | 减轻忌神的坏作用 |
| 用神被合 | 用神不自由，力量打折 |
| 财星被合 | 感情、钱财、父亲、妻子等财星类象被牵动 |
| 官星被合 | 工作、丈夫、规矩、职位等官星类象被牵动 |

判断合的影响，**必须看被合的是哪一个十神**，以及它在命局中的喜忌状态。

### 五合的应用场景

- **婚姻感情**：男命财星被合、女命官星被合，婚恋关系易有变化
- **事业发展**：官星被合或印星被合，工作局面可能变动
- **合作竞争**：合代表合作、纠缠，看合的是喜神或忌神来判断吉凶
- **流年应事**：流年天干与原局天干相合，容易引发对应十神的应事

## 三、天干四冲

### 四组天干冲

天干冲主要有四组，均为阴阳同性之间的直接对立：

| 冲的组合 | 性质 |
| --- | --- |
| **甲庚冲** | 阳金冲阳木 |
| **乙辛冲** | 阴金冲阴木 |
| **丙壬冲** | 阳水冲阳火 |
| **丁癸冲** | 阴水冲阴火 |

### 土天干不参与冲

**戊土、己土不参与天干冲**，原因是土居中位，不像木火金水那样形成明确的对冲方向。

### 天干冲的含义

天干冲表示两种外显力量直接对立、碰撞。可对应以下情况：

| 冲的类象 | 具体表现 |
| --- | --- |
| 思路冲突 | 观点对立，难以协调 |
| 行为方向冲突 | 行动路径不同，相互拉扯 |
| 人事关系冲突 | 人际关系紧张、竞争对立 |
| 外部压力冲击 | 突发变故、意外冲击 |
| 十神动荡 | 该十神代表的人事物发生动荡或变动 |

### 冲的应用场景

**例：日主甲木遇庚金冲**
- 庚金若为七杀，则外界压力、规矩、竞争、疾病、危险等七杀类象容易冲到自己
- 若庚金为正官，则工作中的权威压力增加

**冲的应事判断要点：**
1. 确定被冲的天干是哪个十神
2. 判断该十神在命局中的喜忌
3. 喜神被冲则凶，忌神被冲反吉
4. 结合冲的力度（阴阳同类、位置远近）判断影响程度

## 四、干支上下组合

在一个柱中，天干与地支之间存在五种基本关系：

### 1. 盖头：天干克地支

| 项目 | 内容 |
| --- | --- |
| 定义 | 天干克地支，称盖头 |
| 象意 | 上层压制下层，外在压制内在 |
| 举例 | 甲辰（甲木克辰土）、庚寅（庚金克寅木） |
| 类象 | 表面力量压住根基、宫位承载受压、外部条件制约内部发展 |

### 2. 截脚：地支克天干

| 项目 | 内容 |
| --- | --- |
| 定义 | 地支克天干，称截脚 |
| 象意 | 根基反克外显力量，根基不稳 |
| 举例 | 甲申（申金克甲木）、丙子（子水克丙火） |
| 类象 | 表面有但底下受损、想做事但底层条件不支持、天干代表的人事被地支暗伤 |

### 3. 干生支：天干生地支

| 项目 | 内容 |
| --- | --- |
| 定义 | 天干生地支 |
| 象意 | 外在资源投入到内在根基 |
| 类象 | 表层力量服务宫位、天干十神去生地支藏干十神 |

### 4. 支生干：地支生天干

| 项目 | 内容 |
| --- | --- |
| 定义 | 地支生天干 |
| 象意 | 根基托举外显力量 |
| 类象 | 地支有根气支持天干、天干透出后有源头、力量有根本支撑 |

### 5. 比和：同类五行

| 项目 | 内容 |
| --- | --- |
| 定义 | 天干地支同类五行 |
| 象意 | 上下同气，内外一致 |
| 类象 | 力量相互呼应、某一五行根气更稳、天干地支同心协力 |

### 干支组合关系总表

| 关系类型 | 定义 | 力量方向 | 吉凶倾向 |
| --- | --- | --- | --- |
| 盖头 | 天干克地支 | 上→下压制 | 根基受压，发展受限 |
| 截脚 | 地支克天干 | 下→上反克 | 力不从心，根基不稳 |
| 干生支 | 天干生地支 | 上→下滋养 | 投入根基，积蓄力量 |
| 支生干 | 地支生天干 | 下→上托举 | 根基扎实，有源有根 |
| 比和 | 干支同类 | 上下呼应 | 同气连枝，力量稳固 |

## 五、各关系的应用场景对比

### 五种关系在命局中的应用

| 关系类型 | 应用场景 | 判断要点 |
| --- | --- | --- |
| **天干生克** | 判断十神关系、力量对比 | 阴阳同性/异性、位置远近、喜忌 |
| **天干五合** | 婚姻感情、合作竞争、流年应事 | 合是否成化、被合十神的喜忌 |
| **天干四冲** | 冲突变动、压力突破、事业变动 | 冲的是喜神还是忌神、冲的力度 |
| **盖头截脚** | 判断柱内力量协调性、根基是否稳固 | 干支五行力量对比、宫位类象 |
| **干支生比** | 判断天干是否有根、有无源头支撑 | 地支是否为天干的禄旺库根 |

### 综合应用原则

在实际断命中，以上关系往往同时作用，需要综合判断：

1. **先看整体**：先看命局整体五行旺衰、月令状态
2. **再看局部**：分析具体天干之间的关系（生、克、合、冲）
3. **再看柱内**：分析每个柱内天干与地支的组合关系（盖头、截脚等）
4. **后看喜忌**：以上所有关系最终要结合十神喜忌来判断吉凶

单看任何一种关系都无法得出准确结论，必须放在整个命局中综合考量。
